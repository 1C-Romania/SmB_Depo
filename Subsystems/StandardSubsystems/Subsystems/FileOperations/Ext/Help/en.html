The system provides an opportunity to Save the files of arbitrary format. IN contrast to the storage of files on the disks, the files Saved directly in the system can be linked to  infobase data. For example, in the file attached to the catalogue "Classification" it is possible to Save a detailed goods description.

The administration of files is performed in the form of the system parameters setting of the section Setting and administration. Personal user settings for the work with the files are located in the form My settings of the same section.
