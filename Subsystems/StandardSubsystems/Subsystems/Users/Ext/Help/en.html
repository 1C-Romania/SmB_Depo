The system allows us to administrate the users  and external users by synchronizing them with a list of information database users ( which can be seen in the Configurator ) . It makes achievable the ability to work with an extended information about the users and opportunity to refer to users from the other IB objects..

 

Important: it is recommended to maintain a list of IB users as well as to set their access rights only in the mode of "1C:Enterprise". IN the "Configurator" mode it is allowed only to Creating users with full permissions - system administrators. At the entrance of such the users in the IB in "1C:Enterprise" mode in the catalogue " Users " the corresponding elements will be Creatingd automatically.
