With group change of objects the user can in one click change value of attributes, additional attributes or additional information(hereinafter: attributes) of the selected objects of a certain type. To work with a group change of objects user does not require additional roles. If the user is able to edit the object, then he can perform group change of object. To go to the assistant of the group change, you can use the command Change selected command bar or the list context menu, previously selected a set of objects using multiple selection in the list.
