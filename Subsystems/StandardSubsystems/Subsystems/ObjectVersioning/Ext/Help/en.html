Configuration allows you to keep history of changes of infobase objects. This feature allows you to answer questions: "Who modified object?", "When changes occurred?" and "What has changed in the object?". Objects, that can be versioned are catalogs and documents. Configuration allows you to view any version of the object or compare any version of object.

Before using the versioning capabilities, you must set a flag Use object versioning, in the form of system settings(section Configuring and administration). After saving the settings, command Versioned objects, which opens the form of versioning configuration, will appear on the navigation bar.
