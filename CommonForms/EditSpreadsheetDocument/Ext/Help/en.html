Designed to edit a table document.

To edit a table document or a layout in a print form or a table document, run the Edit layout command in the More menu.

Important: If you edit area names or layout parameters in print form layouts, it may result in a print form failure. To restore the standard print form, use the Print form layouts list.

Editing

  • To edit a table document, use standard editing buttons: change  font size and type, alignment. You can find other formatting commands (e.g. background color and text color, borders, etc.) in the More menu.
  • You can Merge cells with the relevant command of the More menu or split cells with one more execution of the same command. 
  • You can Insert note with the relevant command of the More menu.
  • After you finish editing, click Save and close.

Switching editing and view modes

  • To switch the modes, click Editing. IN the view mode, the layout editing buttons get inactive.

Printing out layout

  • To print out a layout, run the Print command of the More menu.
  • To Preview a layout, run the relevant command in the More menu.
  • To edit the print settings, click More - Page parameters.
