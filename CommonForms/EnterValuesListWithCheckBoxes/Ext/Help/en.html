It is intended to enter several values in the list for filter.

To open the form, go to the Filters tab in the Report settings window, and then double-click the Value column for the In the list, Not in the list, Belongs to group from the list, Not in group from the list operations.

Selecting values

  • Click the Select button to add several values in the list. Select some values from the corresponding list.
  • Run the Add command on the More menu to select values one-by-one.
  • To include the elements from the list to the list of values for filter, select the corresponding check boxes (all check boxes are selected by default).

Selecting values from clipboard

  • Click thebutton, select values, and then click the Insert to the list button.
