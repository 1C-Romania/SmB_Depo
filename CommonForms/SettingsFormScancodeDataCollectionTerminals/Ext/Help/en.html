Data collection terminals driver handler setup form

Form is designed to set the parameters of the data collection terminals. The form allows you to configure the following parameters of the device:

  • Model
    Model of the connected device.
  • Port
    Port to which the data collection terminal is connected.
  • Speed
    Speed of data exchange with terminal.
  • IR - stand
    In case, If data collection terminal is connected to the computer using the IR-holders, need to set this flag.
  • Base separator
    A character or character code, used to separate fields in goods database of the terminal.
  • Document separator
    A character or character code, used to separate the fields in the document of data collection terminal.
  • Base number
    Number of the goods base of the terminal, into which exporting is performed.
  • Document No.
    Document No. of data collection terminal, from which the load is performed.
  • Select source of data load
    If Statusd that you must select the data source, than at booting from DCT choice form will appear. In the form will need to specify, where data should be loaded from: database or a document.
  • Clear document after loading from DCT
    If Statusd that document should be cleared when exporting from DCT, then after loading the original document of the terminal will be cleared. When you boot from the terminal's base - the base is not cleared.

After change of parameter values, press the "OK" button to save made changes, or button "Cancel" to refuse their keeping.
