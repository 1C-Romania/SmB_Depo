Address book

The address book enables you to select contacts and contact information:

There is a list of counterparties at the left part of the address book. Quick filters bar (1) is arranged as in standard list of counterparties. You can select a counterparty using several ways:

  • select single or several counterparties and click (4)
  • double-click
  • drag and drop to the right part of the address book - "Selected addressees"
  • select the whole group at once. In this case all counterparties of the group will be dragged to Selected addressees after confirmation

At that, the application will display the necessary contact information of dragged counterparties:

  • cell phones (email addresses) – all entered cell phones including added kinds of contact information (Settings – General settings – Kinds of contact information)
  • contact persons – counterparty contact persons
  • cell phones (email addresses) of contact persons – all entered cell phones including added kinds of contact information

You can select the contact information used to send SMS (Email) using the check boxes. For example if company phone numbers/addresses are specified in the counterparty card, then you can send the message to phone numbers/addresses specified for the contact persons in order to make your message personal. To do this select Check on the top Contact persons submenu (3) and Unmark in the Counterparties submenu. Click Done to move only the contact information of contact persons. When importing a counterparty to Selected addressees, the check boxes are selected only for the the counterparty and contact information by default. For convenience you can change the contact tree using the Hide all, Expand all icons (4).
