Intended to select digital signatures that should be saved with data.

Opens when the command is executed. Digital signatures and encryption - To save with digital signature, for example, from the files list (attached files of object), attached files list or file card (attached file card) if personal configuration On save with digital signature is set to "Ask". You can find the command in navigation panel or in menu More, as well as in the context menu with right click.

The list displays:

  • To save - check box.
  • Signature author - presentation of the person to whom the certificate was issued.
  • Signature date.
  • Comment - detailed information about the signature.

Select the check boxes for signatures which should be saved with the file and then click Choose.

To hide this window and always save all signatures, enable the check box Do not ask again.
