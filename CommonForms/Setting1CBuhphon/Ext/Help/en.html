Intended for setup of application start button on the home page of 1C-Buhphone. Settings are stored for each user and associated with the work place.

You can open it by running command Setup 1C:Buhphone in the Administration - General settings section.

Starting the same base under the web-client and thin client will require two setup procedures.

First step is to export an executable file from company website. By clicking the link Export the application you will start exporting the application in the browser installed by default. You can unpack the resulting archive in the convenient place.

Also it is required to register on the website 1C-Buhphone. For this use the link Get 1C-Buhfon account.

At second step configure the settings:

  • Select the Display 1C-Buhphone start button on the home page check box - start button will be displayed on the home page.
  • Specify the Path to the executable file 1CBuhphone. If on launch the file is not found in the specified path, the application will prompt you to specify it again.

At third step configure the settings:

  • Select the Insert login and password on application launch check box - entered login and password will be inserted automatically on application launch. IN this case the authorization window will be skipped.
  • Enter Login/password - user data for authorization at website 1C: Buhfon. If information is not filled or filled incorrectly, the application prompts you to enter login and password again. Login and password can be entered only if it is permitted to insert them on application launch.

Click Save to apply the changes. After this 1CBuhphone tab will be available on the home page.

For your convenience links to the company website are displayed for getting assistance. To see the requirements to the computer and the list of required settings, click link Technical requirements.
