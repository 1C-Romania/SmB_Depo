It is intended to select application objects.

You can open it from other objects of the application by running the Select command or according to the corresponding link.

The application objects list represents a tree grouped by application objects types.

Selecting the application objects

Select the corresponding check boxes for necessary objects.

  • Select a check box related to objects group to select all objects of one type, for example, select the Catalogs check box to select all catalogs.
  • Select a check box corresponding to a root to select all application objects listed in the list.

Click the Select button.
