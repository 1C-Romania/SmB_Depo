ATOL: Fiscal registers driver handler setup form

Form is designed to set the parameters of fiscal registers, supported by driver "ATOL: Banking register driver". The form allows you to configure the following parameters of the device:

  • Model
    Model of the connected device.
  • Port
    Number of serial port, to which the fiscal register is connected.
  • Speed
    Speed of data exchange with fiscal register.
  • Timeout
    MAX waiting time of fiscal register.
  • Password of user
    Password of FR user. It is used to print receipts on the FR.
  • Device password
    Password to access the device, needed to work with Banking register when transferring all commands from the PC to the cash register. Attention! Password for the device is not related to the passwords of Banking register operators.
  • Print tax in receipt

    If this property is enabled tax rates will be printed on the receipt.

After changing parameter values, press the button "OK" to Save made changes, or the button "Cancel" to refuse their keeping.
