The report of Subordination structure is designed to display the parent and subordinate documents. Document for which a report is generated is displayed in bold. Tree of parental documents builds up and to the right of the document for which the report is generated . Tree of the daughterly documents is constructed down-right of the document for which the report is generated .

By double clicking on any document displyed in the report it is possible to open the document for viewing.

Using the button of Display for current it is possible to generate the report Subordination structure for the document on which we are currently positioned .

To refresh the report use the Update button.
