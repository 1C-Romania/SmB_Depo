ATOL: Customer displays driver handler setup form

Form is designed to set the parameters of fiscal registers, supported by driver "ATOL: Customer displays driver". The form allows you to configure the following parameters of the device:

  • Model
    Model of the connected device.
  • Port
    Number of serial port, to which the fiscal register is connected.
  • Speed
    Speed of data exchange with fiscal register.
  • Parity
    Parameter of a port, to which device is connected; method of parity check..
  • Data bits
    Parameter of a port, to which device is connected; number of data bits in a transmitted byte.
  • Stop bits
    Parameter of a port, to which device is connected; number of stop bits.


     

After changing parameter values, press the button "OK" to Save made changes, or the button "Cancel" to refuse their keeping.
