SHTRIH-M: Fiscal banking registers driver handler setup form

Form is designed to set the parameters of fiscal registers, supported by driver "Skrikh-M: FR Driver". The form allows you to configure the following parameters of the device:

  • Model
    Model of the connected device.
  • Port
    Number of serial port, to which the fiscal register is connected.
  • Speed
    Speed of data exchange with fiscal register.
  • Timeout
    MAX waiting time of response from fiscal register.
  • Password of user
    Password of FR user. It is used to print receipts on the FR.
  • Administrator password
    Password of the FR administrator. Applied at performing operations with FR, requiring administrative rights.
  • Payments1
    Text for printing description of non-cash payment #1. For example, "PAYMENT CARD"
  • Payments4
    Text for printing descriptionof non-cash payment #4. For example, "CREDIT".
  • Keep a log of driver operations
    This property applies when you have problems when operating with the driver and FR. If this property is enabled a file of transactions will be Creatingd during the operation, that will contain all the transactions performed with driver, and results of of transactions.

After changing parameter values, press the button "OK" to Save made changes, or the button "Cancel" to refuse their keeping.
