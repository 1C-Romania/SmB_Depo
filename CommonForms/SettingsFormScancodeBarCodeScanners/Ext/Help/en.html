SCANCODE: Barcode scanner driver handler setup form

Form is designed to set the parameters of the barcode scanner, pluggable with an external component "1C:Barcode scanner". The form allows you to configure the following parameters of the device:

  • Port
    Serial port number (physical or emulated), to which barcode scanner is connected.
  • Speed
    Parameter of a port, to which scanner is connected; data transfer rate.
  • Data bit
    Parameter of a port, to which scanner is connected; number of data bits in a transmitted byte.
  • Stop bit
    Parameter of a port, to which scanner is connected; number of stop bits.
  • Prefix
    Prefix, added by barcode scanner to the value of the read bar code. Usually is a sign of the beginning of bar code reading.
  • Suffix
    Suffix, added by barcode scanner to the value of the read bar code. Is a sign of completion of barcode reading.

After change parameter values, press the "OK" button to save made changes, or button "Cancel" to refuse their keeping.
