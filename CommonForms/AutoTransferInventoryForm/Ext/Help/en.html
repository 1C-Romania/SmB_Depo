Auto transfer inventory parameters

An additional dialog "Auto transer inventory parameters" will open by clicking on hyperlink "View and edit parameters of inventory auto transferring" to fill and modify data of setting auto transferring parameters in the item form of the catalog of company.

When you open the dialog, specified on the current time values of the parameters will displayed. Quantity and character of control elements in this window depend on the type of (Structural unit, Structural unit, warehouse).

This can be such elements as:

  • Transferring, assembling, production. Belongs to group of attributes:
    • Record off from - selected from catalog "Structural unit". Indicates a and the location, from which stocks will be WrittenOff. The default value is substituted into the document "Inventory transfer" for the type of operation "Transfer" into the fields "Structural unit, warehouse" и "Location" when selecting values in the fields "Recipient" and/or "Location (recipient)", to the document "tools" on the tab "Inventory" into the fields "Record off from" and "Location" when selecting values in the fields "Structural unit" and/or "Location" (divisions) and to the document "Order for assembling, production" in the field "Structural unit, warehouse of reserve" at a choice of value in the field "Structural unit".
    • Recipient - selected from catalog "Structural unit". Indicates a and the location, where stocks will be accounted. The default value is substituted into the document "Inventory transfer" for the type of operation "Transfer" into the fields "Recipient" and "Location (recipient)" when selecting values in the fields "Structural unit, warehouse" and/or "Location" and to the document "tools" into the fields "Recipient" (recipient of finished goods) and "Location (recipient)" when selecting values in the fields "Structural unit" and/or "Location" (divisions).
    • Recipient of disposals - selected from catalog "Structural unit". Indicates a and the location, where returned disposals will be accounted. The default value is substituted into the document "tools" on the tab "Disposals" into the fields "Recipient" and "Location (recipient)" when selecting values in the fields "Structural unit" and/or "Location" (divisions).
  • Deduction of expenses. Belongs to group of attributes:
    • Record off from - selected from catalog "Structural unit". Indicates a and the location, from which stocks will be WrittenOff to the expenses. The default value is substituted into the document "Inventory transfer" for the type of operation "Deduction of expenses" into the fields "Structural unit, warehouse" and "Location" when selecting values in the fields "Recipient" and/or "Location (recipient)".
    • Recipient - selected from catalog "Structural unit". Indicates a and the location, on which charges will be accounted. The default value is substituted into the document "Inventory transfer" for the type of operation "Deduction of expenses" into the fields "Recipient" and "Location (recipient)" when selecting values in the fields "Structural unit, warehouse" and/or "Location".

  • Transfer to Exploitation. Belongs to group of attributes:

    • Record off from - selected from catalog "Structural unit". Indicates a and the location, from which stocks will be WrittenOff for transferring to operation. The default value is substituted into the document "Inventory transfer" for the type of operation "Transfer to Exploitation" into the fields "Structural unit, warehouse" and "Location" when selecting values in the fields "Recipient" and/or "Location (recipient)".

    • Recipient - selected from catalog "Structural unit". Indicates a and the location, where stocks in operation will be accounted. The default value is substituted into the document "Inventory transfer" for the type of operation "Transfer to Exploitation" into the fields "Recipient" and "Location (recipient)" when selecting values in the fields "Structural unit, warehouse" and/or "Location".

  • Return from operation. Belongs to group of attributes:

    • Record off from - selected from catalog "Structural unit". Indicates a and the location, from which stocks from operation will be returned. The default value is substituted into the document "Inventory transfer" for the type of operation "Return from operation" into the fields "Structural unit, warehouse" and "Location" when selecting values in the fields "Recipient" and/or "Location (recipient)".
