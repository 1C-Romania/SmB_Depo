It enables you to view and enter the details. You can enter details in a separate window, unlike additional attributes.

You can open it from the application object windows by clicking Details. You can open it for the objects with configured details.

The list displays:

  • Name - name of the details field.
  • Value - you can view and enter the details.

Entering the values

  • Double-click the field you need o switch to the entry and edit mode.
  • The value entry method depends on the details type (for more information on additional attributes type, see here).
  • Details may have only two values, for example, Yes and No. IN this case, you can not enter new values.
  • If you need to see a value list, click Show all (press F4). Then select the value you need from the list. Depending on the settings, the value list can be displayed with hierarchy or without it. If necessary, you can add values to the list.
  • Depending on the settings, you can also select values from the application catalogs.
  • Depending on the details settings, you may need to Select the data type.

Changing the details

  • If you have sufficient user rights, you can configure a new detail or change some settings of the existing ones. To do that, run the More - Change details command.

Deleted details

  • The details to delete are displayed with icon . To permanently delete the details, you might need to Clear values. Double-click the value to switch to the edit mode.
