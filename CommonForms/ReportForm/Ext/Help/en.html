Configure the preset parameters (if they are provided), and then click Generate  to view the report. You can generate the report in a background mode in the client/server version of the application.

Generation mode

Select the Generate immediately check box on the More menu to generate the report automatically right after it is opened, without clicking the Generate button.

Report versions

  • You can use several report versions performing similar analysis in the application. Use the Report versions button to preview the reports.
    • Select the report version you need from the drop-down list. Current report version check box is selected.
    • Use the Other reports command to view the list of other report versions.

Settings

  • The Settings button allows switching to report parameters setup.
  • Click More - Other - Standard settings to return to default settings.
  • Click More - Other - All settings to go to all available report parameters setup.
  • Click More - Other - Save settings to write current report settings.
  • Click More - Other - Select settings to use previously saved report settings.

Other reports

This window shows the list of similar reports.

  • The list displayes the names and descriptions of the reports.
  • The report list is grouped. You can use groups to view other similar reports of the same application section.
  • Click the header with left mouse button to go to another report version.
  • You can select the Close window after opening the report check box.

Printing the report

  • You can print the generated report by clicking thePrint button.
  • To check the report and change page settings, click Preview .
  • Run the More - Page settings command to set up report printing.

Sending the report by email

Using the Send report by email button you can:

  • Send now, to do this:
    • select one or several formats for the report to be sent selecting the check boxes.
    • select the Add to archive (.zip) check box if necessary.
    • Click Continue. The report will be attached to email.
  • Create a report mailing.
  • Include into existing report mailing. To do this, select the one you need from the mailing list.

Selecting report version

  • Run the More - Select report version command to load previously saved report version.
    • To switch to the report version you need, click Select button in the report versions list.

Changing the report version

  • Click More - Other - Change report version. There is an option to add fields, change parameters and report settings.
  • Revisions can be saved into another report version.

Saving the report version

  • Click Report versions - Save report version, specify the Name and other parameters.
  • You can also run the Save report version command on the More menu.

Reset the settings

  • Click More - Other - Standard settings to reset user settings.

Other available actions

  • Run the Search command for report searching.
  • Using theHide all groups or Expand all groups buttons you can hide or expand the report grouping.
  • To save the report result into a file, click , specify attachment file name and save path.
  • Click , to calculate the sum of numbers in the selected cells.
