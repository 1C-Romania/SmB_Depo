Click the Recall button to recover the master node prior to continue the start. It is required to do this when the master node is cancelled for importing into the subordinate node of the application master node during the distributed infobase recovery process (it is prohibited to import the earlier version of the application).

Click Disable to disable connection with the master node irrevocably prior to continue the start. It is required to do this when the master node is cancelled in cases as follows:
- when the master node is created from the copy of the subordinate node;
- when the independent infobase is created from the subordinate node (or its copy).
