It is intended to select the nodes of exchange plan (infobases).

You can run the form using the Pick or Add command from other application's objects, for example, when setting the Importing prohibition dates.

The list includes:

  • Application - the list of applications with which the data synchronization is enabled. The infobases in each application are grouped by data synchronization type - for example, without data exchange rules.
  • Inforbase - the list of infobases with which the data synchronization is enabled. Besides, for each application and synchronization type the <All infobases> value is provided - there is an option to select all infobases of the version you need and synchronization type by this value.

Selecting the node of exchange plan

  • Select necessary value, and then click Select.

Picking the nodes of exchange plan

  • Select several values, and then click Select.
