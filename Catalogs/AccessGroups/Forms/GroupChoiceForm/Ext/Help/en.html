You can use the list to select the Access groups folder.

Since there can be a large number of the access groups and groups with similar settings for rights restrictions, there is an option to create folders for easy search of necessary Access group and for better understanding of the access rights restrictions in general.

The Access groups can be grouped according to different attributes.

Selecting access groups folder

  • Select a folder, click Select. You can double-click also.

Entering access groups folder

  • Click Create group, enter folder name.
