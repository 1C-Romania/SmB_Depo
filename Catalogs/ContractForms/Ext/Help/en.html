Contract forms catalog

The Contract templates catalog is intended to save templates which are used for generation of printed versions of contracts with counterparties.

You can modify the templates supplied with the application, or use them as the basis for new ones.

You can generate a printed form of a contract for every counterparty using selected template.

When creating or modifying a contract template, you can use predefined parameters which will be automatically filled by application data in the print form. You can also use manually filled fields which are to be edited in the contract text directly prior to printing out.
