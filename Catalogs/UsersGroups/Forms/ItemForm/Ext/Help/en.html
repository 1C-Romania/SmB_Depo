Users groups catalog

Viewing and editing properties of a group of users.

You can divide users into groups by various criteria, e.g. by job positions or according to the enterprise structure. It speeds up the access rights setup procedure - you can configure them simultaneously for different users having alike functions.

Adding user group

  • Enter Name of the group.
  • Enter Group members:
  • When adding and deleting members, remember that they belong to parent groups, too.
  • To add more users, you can use the Select command. In this case, you can select several users in the list at once and add them to the group.
  • With the More - Add command, you can add only one user.
  • You can substitute the new group to one of the existing groups: select a necessary group from the list in the Belong to group field.
  • On the Comment tab, you can write a detailed description of the group.

Access rights of user group

  • Click Access rights in the group navigation panel. Click Add to group and select the access group from the list.
  • If you assign roles directly, you can do it for a user and a user group.
