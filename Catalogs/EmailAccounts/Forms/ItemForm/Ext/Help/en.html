Email account properties.

When you enter an account, most parameters are defined by default according to the email server settings for the email account.

For the system accounts, it is recommended to remember the password for the application to send out messages from such an account without involving the user. Still, you should decide whether to store passwords in the infobase individually for each case.

Basic data

  • Enter the Name of the account. Formulate the name to briefly identify the account, its purpose or owner, e.g. "Customer Support". This name will appear in various lists.
  • Enter the Email address of the account in the User@Server.net format.
  • Specify the Sender name - a name of the account owner that will be stated in correspondence. It may differ from the user name, e.g. "Administrator" user may be named as "John Smith".
  • Specify how to Use account:
  • Select the Send messages checkbox.
  • Select the Receive messages check box.

Sending messages (customizing outgoing email server)

  • Specify the User name of the account in the User@Server.net format. It may differ from the main one. IN this case, it will be used only for sending messages.
  • If necessary, enter the Password of the account (if it is not entered, you are prompted to specify it when sending messages).
  • If necessary, select the Secure password check check box.
  • In the Server field, specify the address of the SMTP server, e.g. "smtp.Server.net".
  • Specify the computer Port to send mail, e.g. "465".
  • Select the Use secure connection (SSL) check box.
  • If necessary, select the Connect to server before sending check box.

Receiving messages (customizing incoming email server)

  • Specify the User name of the account in the User@Server.net format. It may differ from the main one. IN this case, it will be used only for receiving messages.
  • If necessary, enter the Password of the account (if it is not entered, you are prompted to specify it when sending messages).
  • If necessary, select the Secure password check check box.
  • In the Server field, specify the address of the POP3 server, e.g. "imap.Server.net". 
  • Select the Protocol of the incoming email server: POP or IMAP.
  • Specify the computer Port to receive mail, e.g. "993".
  • Select the Use secure connection (SSL) check box.
  • Select the Leave message copies on server check box to prevent deleting the messages from the "Inbox" folder. By default, the check box is cleared, so all letters will be deleted from the mail server when received.
  • Use the Delete messages from server field to define the mail retention period, e.g. in "10 days".

Other settings

  • Define the Server timeouts, e.g. "30 sec".

Checking settings

  • Click Check settings to check the selected parameters of the account.

  • A sample message will be sent to check the connection with the incoming email server (if the account is used to receive mail). You can also use the relevant command in the More menu.
