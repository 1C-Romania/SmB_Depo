Which settings should be specified in the input assistant of new agreement on exchange using EDF operator?

You can use the assistant for registration of a new agreement to define cryptography global configuration in the system, import and customize EDS certificate, and register the unique ID of the exchange participant. At every transition to the next step the application automatically verifies completed settings and will not let you move forward if any errors in setup are detected.

The result of the assistant work is an open form new agreement with tested settings. It is recommended to use this method to input new agreements.

It is required to perform the following steps in the assistant:

  • Check common cryptography settings (the application will automatically prefill them).
  • Specify company and company certificate which will be used for authorization on operator EDF service.
  • Set company identifier which was received from EDF operator during registration of participant in electronic document flow.
