The application allows exporting Russian banks classifier from the website of the RusBusinessConsulting agency www.rbc.ru). At that the application takes into account closed banks (in the card of such a bank the relevant check box is selected).

Importing bank classifier

To export bank classifier run the Export classifier command from the bank classifier.

  • Select the Import option using the radio button:

    • from the site of RusBusinessConsulting agency (when operating in web client there is an option to export only from RBC site);

    • from Information Technological Support disk - data will be exported from the 1C:ITS "Russian Banks Classifier" (RBC) disk.

      • Specify the Path to ITS.

  • Click Export.

  • When successfully exported the application displays the message on the number of exported banks. Click Close.

Reloading

  • When reloading only the updated classifier data is ovewritten.
  • At that if bank BIC changed, a new bank record will be created.
  • If other fields were changed, then during reloading, they will be overwritten (except for the Address and Phones fields, when the classifier is reloaded from RBC).

Banks classifier exporting features in web client

  • The relevance of banks classifier is checked in web client also and message is displayed suggesting the update of the classifier.
  • If the bansk classifier was exported prior to application publication in the web client, then you need only to update the classifier when working in web client.
  • Exporting from the drive is not available in web client.
