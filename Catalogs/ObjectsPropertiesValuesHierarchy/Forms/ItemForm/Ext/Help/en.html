The form is intended to enter an additional attribute (data) value.

The application provides the use of several types of additional attributes (data). The data type is selected when at setting an additional attributes (data). The application suggests to enter values when selecting the Additional value (hierarchy) data type. At that, a separate values list is used for each additional attribute or data of this type.

Entering values

  • Specify the Name of an additional attribute (data) value.
  • Using the Included into group field a new element can be subordinated to another one from the list. To enter a new higher item, click Create (F8 key). To select an item from the list of available items, click Show all (F4 key), select the item you need from the list.
  • In case the Values are characterized also by weight factor check box is selected for an additional attribute (data), enter numerical value in the Weight factor field.
    • This field can be used further for the analysis in reports. The logic of using this field is defined by the user. For example, for the Reliability attribute the weight factor is set as follows: weight factor is set to zero for values from one to three, the factor is set to one for values of four and five.You can filter reliable clients (with weight factor more than zero) quickly in the report.
