The list contains values of additional attributes and data.  

You can use additional attributes (data) of different types. Select your type when configuring additional attributes (data). When selecting the Additional value (hierarchy) data type, you are invited to enter values into this list. Each additional attribute or data of this type uses its individual list of values.

The Additional value (hierarchy) data type is used frequently. Enter information by selecting values from the list to make the process quick and error-free. Use this data type to describe various properties of the application objects.

A list can use values hierarchy. IN such a list, the items have similar properties, but they can have subordinate items. The hierarchy may be rather complex.

You can select any hierarchy item unlike the grouped list of additional values where you can not select group names as values.

Selecting values

  • Select a necessary item.
  • Click Select.
  • You can also double-click the item.

Entering values

  • To enter values, click Create. Fill in the required fields.
