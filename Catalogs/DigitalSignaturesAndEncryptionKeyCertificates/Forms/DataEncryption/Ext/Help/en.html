It is intended to select certificates for data encryption.

Encrypting data

  • You can view the object to be signed by the link on the top of the window. The link displays the name of the object.
  • Personal certificate - is mandatory in order to enable subsequent details;
    • select personal certificate from the list;
  • Certificates owners allowed to decrypt data, except you - the list of other certificates, that can be used for details.
    • click Pick (select from the list one or several certificates) or Delete to form the list for selection.
  • You may use several certificates, for example if you need to encrypt data for several users.
  • Click Encrypt to confirm the selection.
