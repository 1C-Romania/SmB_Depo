Digital signature certificate info.

The application automatically displays certificate data at the top:

  • Issued to
  • Issued by
  • Valid before
  • Function

Certificate owner info:

  • Signature owner info (filled in automatically):
    • Full name
    • Company
    • Position
  • Presentation - certificate full name (filled in automatically)
    • Company - from the list select the company that received the certicate
    • User - select the user - certificate owner from the list
    • Added by - select the user who added the certificate to the list

Private key settings

    • Application - application data for digital signature and encryption.
    • Improved security of the private key - select the check box if provided.

Checking the certificate

  • Click Check to check the certificate.

Viewing the certificate data

  • Clickto view the certificate data.

Viewing the application

  • Press to view the application used to receive the certificate.

Exporting to file

  • Click Save to file, specify the path to the folder to save into the file.

Autofilled attributes

  • To Show autofilled attributes select corresponding check box on the More menu. IN this case signature owner data filled in automatically are displayed.

Revoked certificates

  • Select the More - Certificate is revoked check box. Action is irreversible.
