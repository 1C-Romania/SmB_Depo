It is used to apply for a new qualified certificate issue to 1C Certification Authority.

Agreement

  • Read an Agreement of an Accession to Certification Authority (CA) Rules on the page. Using hyperlink it is possible to view information of Certification Authority.
  • To Accept the Agreement enable the checkbox.
  • Click Then to start application preparation.

Company

  • It is necessary to entry company data on the page. Select the company from the list . If company data is filled in, then the fields are filled in by the application automatically.
  • The application checks the correctness of details entry.
  • Click Then to entry certificate owner data.

Application Save

  • The application is saved automatically at preparation stage and store in the certificates list . You can find it any time and continue working.

Certificate owner

  • It is necessary to enter data of the certificate owner on the page. If data of an individual person connected with the company is entered to the application, they will be automatically filled in.
  • The application checks the correctness of Personal insurance policy number and email address fields filling.
  • Click Then to start digital signature key generation and preparation of a request for certificate.

Generation of digital signature key

  • Data of digital signature owner is checked on the page. Click links to receive additional information.
  • In Using the application field select the application - cryptographic service provider from the list of applications Seted on the computer.
  • Click Then to generate digital signature key and application for certificate.
  • Application status changes to Prepared.

Send request

  • At this page you shall enter the Top Manager info to print the appliation.
  • You shall also prepare the following documents:
    • Application for a certificate (click Print to print the application);
    • Copy of Tax Authority Registartion Certificate (tax identification number);
    • Copy of Certificate of State Registration of Individual Entrepreneur;
    • Copy of the document certifying the certificate owner;
    • Copy of Personal Insurance Policy Number of the certificate owner.
  • Enter Service Company Data (1C partner).
  • Click Then to send application for processing.

Waiting for request processing

  • After application is sent Waiting for request processing page is opened. To check the status click Update status.

Received certificate installation

  • After the certificate is received, activate it on your computer using Activate certificate button.
  • To import the Certificate use the relevant link.
