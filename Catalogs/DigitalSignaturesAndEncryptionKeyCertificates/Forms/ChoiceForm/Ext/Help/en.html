It is intended to select a certificate of an electronic print full name.

The list includes:

  • Presentation - full name of the certificate;
  • Company - a company for which the certificate is issued;
  • User - the user who is an owner of the print full name.

Selecting certificate

  • Select necessary certificate, click Select. You may double-click also.

Adding certificate

  • Click Add, select the source:
    • from installed on computer - select from the list;
    • from file - specify a path to a file. Fill necessary fields in the opened window.

Viewing certificate data

  • There is an option to open certificate data by the Change command on a context menu by right-clicking or on the More menu.
