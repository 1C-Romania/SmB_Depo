The form is used to select the certificate of a digital signature.

You can open the form by clicking Select on the Data encryption window.

The list contains:

  • Presentation - full title of a certificate;
  • Firm - company which the Certificate is issued to;
  • User - user who is the owner of the signature;
  • Issued by - company that issued the certificate.

Filters

  • If there are a lot of certificates, there is an option to filter certificates you need:
    • by the Firm field - type the name of the company or copy it from the list;
    • by the User group field - select the group from the list.
  • Select the check box to select certificates by the value you need.

Selecting certificate

  • Select the certificate you need, and then click Select. You can also double-click it.

Adding certificate

  • Press Add and select the source:
    • from installed on computer - select from the list;
    • from file - specify the path to the file. IN the opened window fill in the necessary fields.

Viewing certificate data

  • You can open certificate data by right-clicking the Change command of the context menu or on the More menu.
