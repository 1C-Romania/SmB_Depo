Discount cards templates catalog

The list of configured templates of magnetic discount cards.

 

It is recommended to use the Check code by template button to develop a template:

  • Fill necessary fields of a template.

  • Read a discount card code by a magnetic cards reader.

  • Click Check code by template.

  • Check the received code which will be shown to the right of the button. If the received code is incorrect, repeat all steps once again except the second step.

    For magnetic discount cards it is supposed that data is written on the second track of the magnetic card. It is the most widespread option of data writing on the magnetic discount cards.
