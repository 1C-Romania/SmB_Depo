Configuring electronic document flow parameters

All parameters of exchange between regular exchange participants are defined in the Electronic document usage agreement.
The agreement specifies the exchange method, exchange participant ID, electronic document formats, etc.

When you export/import data from/to a single-deal file, all parameters are defined during the operation itself and require no preliminary definition.

For repeating deals, you can use the following exchange variants configured in the relevant bookmarks of the list form:

  • Exchange via electronic document flow operator (EDF operator)
  • Direct exchange between counterparties
  • Exchange between organizations within one company
  • Direct exchange with banks

Important. For direct exchange with counterparties, enter and configure an Electronic document usage agreement  individually for each counterparty.
For exchange via EDF operator, one copy of the agreement is required for each EDF operator to store settings for all the counterparties.

Important. Direct exchange agreement is of higher priority than agreement via EDF operator , i.e. if the application has two valid agreements for one counterparty (a direct exchange agreement and an agreement via EDF operator), an electronic document will be forwarded according to the "Exchange method" specified in the direct exchange agreement (if the Outgoing documents bookmark of the currently used  Electronic document kind has a selected Send check box).

Adding new agreement for electronic document exchange via EDF operator

In the common system settings, enable digital signature; in the list of electronic document usage agreement, in the Via EDF operators bookmark, add your new agreement:

  • Either with the Create command (using the New agreement registration assistant)
  • Or with the Create (manually) (filling in the form of the new item manually).

Important. Electronic document exchange is available only if a relevant agreement is Valid and not marked for deletion.

Adding new agreement for direct electronic document exchange between counterparties

You can add a new item from the list of electronic document usage agreements in the Direct exchange with counterparties bookmark.

Important. Electronic document exchange is available only if a relevant agreement is Valid and not marked for deletion.

Adding new agreement for direct electronic document exchange with banks

In the common system settings, enable:
- Digital signature
- Direct electronic document exchange with banks
Add a new item of the list of electronic document usage agreements in the Direct exchange with banks bookmark.

Important. Electronic document exchange is available only if a relevant agreement is Valid and not marked for deletion.

Automatic export of exchange settings for bank

Get a settings file in the bank account. Then use the Create (export from settings file) command and select the settings file. Then you will see an EDF settings form with completed data.
