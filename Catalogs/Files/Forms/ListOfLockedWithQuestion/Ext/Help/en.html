When you finish your work, this list shows the files you locked for editing.

To cancel this list, clear the Show edited files when finishing work checkbox. You can also use the Working with files tab in the personal settings.

The list shows:

  • File format as a sign
  • File name
  • Description - detailed file data (from the file card)

Available actions

  • View  - opens the file for editing with an external application. You can also easily open the file by double-clicking it or by clicking the Enter button.
  • Finish editing - saves a file version and makes the file free for other users.
  • To open and view the file details use the More - Open card command (press F2).
