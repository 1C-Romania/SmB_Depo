Designed to export files from the computer drive.

Open it from the file list by running the Export files command of the context menu or the More menu after selecting a file (files) in the opened Windows file explorer. In the file explorer, select one or more files, and then click Open.

A list of files to be exported is displayed. You can replace the entire list with the Select button (you can not enter new items to the list; when you select new files, the old files are deleted) - open Windows file explorer and get other files.

Enter the data required to export files:

  • The Folder field contains the folder automatically selected from the file list to export files from the computer drive. If necessary, you can select another folder.
  • To save drive space, select the Delete files after adding (by default, it is cleared.)
  • Create Description of created files.
  • Select the check box to Store versions (recommended; by default, the check box is selected).
  • For text files, you can Select encoding from the list.
  • To confirm, click Export file.
You can also drag files to export them from the hard or network drives: just use your mouse to drag them from Windows file explorer to a necessary folder of the application file list.
