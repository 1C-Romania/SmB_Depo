The application allows storing the files in the infobase without linking them to a specific object. Such files are managed through the list.

You can open the list by the Files command in the Organizer section.

Working with the list is arranged similarly to Windows Explorer.

The left part of the list contains the tree of folders. The list has predefined folders which can not be deleted, such as the Files templates folder.

In the right part of the list there is a list of files in the selected folder. Using corresponding commands, you can add files and folders to the list and copy, change or remove them.

To work with file details (author, description, information, etc.), use the card (More - Open card), and to work with a file (View or Edit file), use the corresponding application set on the computer.

When you edit a file, its previous version is saved automatically in the infobase (this is the default setting and it is not recommended to change it, since if the file has been damaged during editing, the setting makes it possible to get back to one of its previous versions).

Files being edited by you are displayed in green, while files used by other application users are displayed in grey.

The following items are shown in the list:

  • - existence of a digital signature or encryption
  • Description - the full name of the file. The sign next to the description identifies the type of the file. IN the same column there is the file information - the detailed information on the file
  • Author - the user who added the file
  • Date of change - the date and time when the file version was added into the base

With the help of the More - Additionally - Change form command you can introduce additional fields, for example:

  • Edited - the user who added the file version
  • Current version - the code of the current version, etc.

Adding Files

To add a new file into the application,

  • Select the needed folder
  • Click Create (press Insert)
  • Select the method of creation:
    • From file on hard drive
      • To attach to an object or put in a folder a file which already exists on the computer, specify the file path on the drive and click Open
      • Thereafter the application will save the file and will automatically create the File card and open it 
    • From template
      • Select a file template in the selection list, and then click Select
      • The application will automatically create the File Card and open it 
    • From scanner
      • The option to create a file From scanner becomes available if a scanner is connected to the computer (more details can be found in the scanner documents) and the scanner work component is activated on the computer. You also need to adjust the scanning process 
      • If you meet all the conditions and make all the adjustments, you can obtain the file from the scanner
      • To do that, follow the instructions of Scanning Assistant
      • As soon as you scan the document, run the Accept command, and after that a code will be automatically assigned to the file and the card will be generated and displayed on the screen. The code is also inserted automatically in the Description field. Edit needed fields, click Save and close to save the file card

File templates

  • The application is supplied with a File templates predefined folder. IN this folder you can place files which will be used as templates for creating new files. For example, a template of a contract document can be placed in this folder to be used to create contracts with counterparties
  • To place a new template into the folder, open the File templates folder, click Create, and using the radio button select From file on hard drive. Then select the needed file
  • The application automatically creates and displays the file card. Fill the template information, the Folder field is filled by default as "File templates"

Placing files to folders

  • In the list you can Create the required number of folders for storing files which are not linked to any specific application objects
    • You can Create a folder also with the help of the corresponding command of the More menu or of the right-click menu
    • Complete the required fields
  • You can drag files into various list folders. Besides, for selected files you can use the More - Move to folder command
  • You can also drag folders to subordinate one to another

Folders access rights

  • Various access rights can be assigned to folders:
    • File folders access rights are adjusted by the administrator of the application or by a user having the right to manage access rights for the particular folder
    • To adjust access rights, use the corresponding command of the folder navigation bar

Importing files

  • With the help of the Import files command of the More menu or the right-click menu, you can place files from the hard drive or the network drive into one of the folders of the list
  • In Windows Explorer that will be opened, specify the file path and click Open
  • Fill the data required for importing

Importing folders

  • To place a folder from the hard drive or the network drive into one of the folders of the Files list, choose the folder, run the Import folder command of the More menu or the right-click menu
  • In Windows Explorer that will be opened, specify the folder path, click Choose folder
  • Fill the data required for importing

Importing files and folders by dragging

  • If you need to import one or several files (folders), you may use dragging - just drag the file (folder) or several selected files (folders) from Windows Explorer to the desired folder in the application files list
  • In case of dragging a single file, the file is immediately saved in the list of files and the file card is generated automatically
  • In case of dragging several files, a folder or several folders, complete the required data

Exporting folders

  • The Save folder command (in the folder shortcut menu bar) allows copying the selected folder with files to the hard drive. Whereas only active versions of files are copied

Managing files list display

  • With the help of the More - Additionally - Use tree command you can change the presentation of the list and view all the files from different folders in one list
