The report is displayed after operations with files, for example, files importing or folders importing into the files list (attached files list) in case of errors.

The files list is displayed in the report. This list includes:

  • Full path to files, that were not imported.
  • Error description, for example, "File size <"Name">(2 Mb) exceeds the maximum allowable file size (1 Mb)".
