It is intended to pick up the files.

The list consists of two parts: folders list on the left, files list in the selected folder - on the right. The list consists of columns as follows:

  • Name - attachment file name, also has a description - file details;
  • Author - the user who created the file in the application;
  • Creation date - the date and time of a file created in the application.

Select several files in the list, and then click Select.
