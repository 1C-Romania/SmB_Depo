The form is used to manage files attached to the application object.

You can open it by running the Attached files command of the application object navigation bar. You can add new files to the list, copy, change or delete the files using corresponding commands.

Files that you open for editing are presented in green color, the grey color is used for the files opened by other users of the application.

The list displays:

  • - availability of the digital signature or encryption
  • Name - full name of the file. The icon next to the description specifies a type of the file
  • Author - the user added the file version
  • Change date - date of file version data changing (local time)
  • Description - detailed description of the file

You can add data using the More - Change form command:

  • Extension - file format, the icon next to it also specifies the format
  • Edited by - the user who opened the file for editing, etc.
  • Size (KB) - data size of the file version

Importing files

  • You can add files from the hard disk or network drive to the list using the More - Import files command.
    • Specify the file path in the opened Windows Explorer, and then click Open
    • Fill in the data necessary for importing
  • You can also drag one or several files from Windows Explorer. If you add a file, the file card is created and opened automatically. If you add several files, file cards are created automatically, but not opened.
