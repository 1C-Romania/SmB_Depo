It is used to import files and folders to the file list by dragging.

Importing files and folders by dragging

  • If it is necessary to import one or several files (folders), you can drag-and-drop - simply drag the file (folder) or several files (folders) from the Windows Explorer to the required folder of the application file list using the mouse.
  • One file is immediately moved to the file list and the file card is created automatically.
  • Dragging several files, folder or several folders you shall fill in the data as follows:
    • Files dragged for importing are displayed as a list. Folders dragged for importing are displayed as a tree. At that, you can open each folder and view files of the folder
      • Using check boxes you can specify folders and files to be imported to the application
    • The Folder field is automatically fillied with the selected folder in the file list where files from the computer disk will be imported. If necessary, you can select the other folder from the list
    • To save space of the disk, select the Delete files after adding check box (cleared by default)
    • Enter Description for the created files
    • Select the check box to Save versions (recommended, by default the check box is selected)
    • For text files you can Select encoding from the list
  • To confirm, click Import files
