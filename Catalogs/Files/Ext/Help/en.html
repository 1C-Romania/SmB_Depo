Files

A file card (Open Card) is used for operations with the file attributes (author, name, long description, etc.), when corresponding application is used for operations with the file itself. The previous version of the file is automatically Saved in the infobase when file is edited.

Create Files

To Creating a new file

  • Open a file for viewing list
  • Select required folder
  • Press the Insert button or use the Create command
  • Select Creating method - From Template, From File on Disk or From Scanner

The option of file Creating using scanner becomes available if a scanner component is connected.

File Operations

The commands, located on the command bar or in the All Actions submenu in a file list or a file card, can be used to work with a file:

Command Abbreviation
Edit (F4) Marks file as locked for editing and opens it with the external application.
Finish Edit Updates file in storage and removes the mark indicating that file is locked for editing.
Save Changes Saves version of the file being edited to a storage, but leaves file locked by the current user.
Lock Markes file to a storage as locked by the current user for editing, but does not open the external application. The file editing can be started later.
Cancel Edit Removes the mark indicating that file is locked.
Open Opens file with external application for view and edit, depending if the file is marked as locked for edit by the current user.
Save As Saves file to the specified directory on hard drive or network drive.

Note: Full-text search will not work for the encrypted file.
