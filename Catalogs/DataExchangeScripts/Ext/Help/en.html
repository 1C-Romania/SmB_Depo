It is used to customize and store data synchronization scripts.

You can open the list from the Data synchronization section by the relevant command. You can also open the script list from the Data Synchronization list using corresponding command on the More menu.

The list displays:

  • - importing (data import)
  • - exporting (data export)
  • Name - full name of the script

In the list the bold type is used for the scripts run automatically according to schedule.

Using the More - Change form command you can add extra fields like Data version, Comments, etc.

Script settings

  • Click Add, and then enter necessary fields. Setting of data synchronization script includes the following steps:

    • Adjust the schedule for running the data synchronization script
    • Select infobases for which the data synchronization script will be run
    • Select the connection type and action (import or export) for each infobase
  • After data synchronization script is set, the icon next to data synchronization setting is displayed in the list. Data synchronization is scheduled using routine task.

Enabling data importing and exporting

  • Using Enable/Disable drop-down menu you can manage data importing and exporting.
  • If importing or exporting is enabled, the check box is displayed in the corresponding column.
  • It is not recommended only to import data or export data, although the you can use the script for this. It is necessary to synchronize data completely in one script.

Enabling schedule

  • Clickto Enable / Disable data synchronization schedule. If the schedule is enabled, the script is displayed in the list in bold.
