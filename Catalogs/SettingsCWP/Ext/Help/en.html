Cashier workplace settings catalog

The catalog stores the CWP settings. One item matches one item of the Workplaces catalog. This allows for storing separate settings for different cashiers and their working computers.

The catalog stores the list of quick goods and the list of hotkeys and menu commands.

You can use the list of quick goods for setting the quick goods hotkeys on the CWP. For each hotkey, you can set a header and display order (sorting field), as well as the keys combination for input from keyboard.

You can fill the quick goods list with some goods group items or you can paste them from another item of the Cashier workplace settings catalog.

You can use the list of hotkeys and menu commands to set the headers and keys combinations for the main CWP hotkeys and menu commands. The list of hotkeys is limited.
