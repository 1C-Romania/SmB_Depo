The form is intended to view the set of additional attributes and data.

You can see the Name of the set and specify the catalog group related to this set.
The catalog includes the lists of additional attributes and data included in the set. You can only view the lists and can not make any changes in them. Use the Additional attributes or Additional data commands in the Administration - General settings sections to make changes.
