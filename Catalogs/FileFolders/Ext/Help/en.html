All files are stored in a convenient folder structure, for example:

  • According to organizational structure of your company
  • By file type (e.g. "Contracts", "Orders", "Internal notes")
  • By access level (e.g. "Common", "Confidential")

You can assign different access rights (the Access rights command in the folder card).

The list contains predefined folders that you can not delete, e.g. File templates.

The folder list has a hierarchical structure and is designed for selection. You can Edit the selected folder (if you have rights) with the relevant command in the More menu. You can add a new folder to the Files list with the Create command from the context menu.

Selecting folder

  • Select the folder, and then click Select or double-click the folder.
