Is it possible to see in a clear for the user way the content of an electronic document?

The viewing form of an electronic document displays the corresponding printed form completed according to the electronic document data. This representation allows visually to evaluate the content of an electronic document.
The form also displays the Seted for electronic document the digital print full name, passed and planned statuses of the electronic document.

Important. IN the viewing form of electronic documents, the format of which contains the requisite "Information field" that is used for the transmission of Information for the document (usually the service information), will be available the option "Disable information field output" that allows not to show an additional information of the electronic document.
The value of this adjustment the system memorizes automatically when you close the form for each user.

Which actions can be performed in the viewing form of an electronic document ?

  • Depending on the settings of the agreement (apply or do not spply the EDS) and on the status (passed stages) in the form it is possible to execute the commands for going to the next stage.
  • The command "All actions" - "Save electronic document to disk" saves the current version of the electronic document with a digital print full name files in the form of an electronic document to a directory on the disk.
  • After the command is executed a file of electronic document is exported to the disk as well as the EDS files are separately saved if such were Seted on the electronic document.

Important. The mode setting of the displaying of the question on the Seted EDS list saving is made ​​in "Personal EDS Setting" of the personal application settings.

Is it possible to reject an electronic document if the user does not agree with its content?

To reject an electronic document in the viewing form of an electronic document use the command "Reject electronic document" in the group of commands "All actions" and specify the reason for the deviation which in future will be visible to other users.

Important. Operation of the electronic document deviation is irreversible, further use of this copy will be impossible, a new version of the electronic document will be required.

Is it possible to print an electronic document for the providing the external authorities with a paper copy?

To print the electronic document is possible from the content viewing form through the command "All actions" - "Print document with electronic print full names"
Electronic document will be printed in a visible on the screen form with an additional sheet that lists the EDS Seted if IS any.

Important. If an electronic document is necessary to provide to external authorities in electronic form than use the command "All actions" - "Save electronic document to disk."

Is it possible to refill an accounting document according to the incoming electronic document?

Refilling is called by the command "Refill the document" for the unposted accounting system document.
Important. The previous content of the accounting system document will be replaced.

Is it possible to "reassociate" the incoming electronic document with another accounting system document?

In the form of an incoming electronic document using the command "All actions" - "Select the document of reflection in accounting" it is possible to change the connection of an electronic document with an accounting system document.
This command does not refill the accounting system document according to the electronic document (for this purpose it is necessary to use the appropriate command in the form of electronic document).

Is it possible while exchanging with the counterparties in the application to confirm to the sender of an electronic document that the data is received?

If on the phase of the setup of direct exchange agreement with the counterparties there was set a demand of a response print full name (set a sign "To sign" on the tab "Incoming Documents" next to the desired type of electronic document), then in the viewing form of an electronic document it is possible to set the EDS for the incoming electronic document by the command "Sign and send confirmation" or send it to the person in charge for the print full name by the command "Approve an electronic document".
Important. While exchanging the primary documents in the electronic form of the FTS format that requires a response title for the incoming electronic document, after the command "Approve the electronic document" is executed the application Creatings a new electronic document of the necessary title and offers to sign by EDS.
