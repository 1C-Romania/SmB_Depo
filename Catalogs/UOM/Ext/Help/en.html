UOM catalog

All units of measure used by the companies (pieces, containers, cases, boxes, etc.) are Saved in this list.

The list does not support a hierarchy.

The information editing is done in a separate window, which is opened from the list form. It is possible to use Unit of Measure Classifier (OKEI) to enter a new item.

To add the new data from Unit of Measure Classifier it is necessary to press the "Fill From Classifier" button.

The table of Unit of Measure Classifier will be opened.

And directly from the table it is possible to add a new unit of measurement to the catalog "Unit of Measure Classifier".

The search of the necessary unit of measure in the Unit of Measure Classifier can be performed using Ctrl+F key combination.
