The catalog is used to select the name for a user registered in the operation system, when registering in the application.

Using this option, the user can log on to the application when registering in the OS.

The list consists of two parts:

  • In the left, the Domains list shows the names of the domains available on this computer.
  • In the right, the Users list shows the names of the users registered in the selected domain.

Selecting user

  • Select the domain.
  • Select a user in this domain, and then click Select. You can also double-click the required user.
