Information of access groups profiles folder.

There might be a lot of access groups profiles and they might have similar functions, therefore there is an option to create folders (groups) for simplification of the required profile searching, and to understand the principles of their creation.

Entering the access groups profiles folder

  • Enter Name
  • You can move the folder to the Parent group (folder). Select the folder you need from the list, and then click Select. You can also double-click to select parent folder.
