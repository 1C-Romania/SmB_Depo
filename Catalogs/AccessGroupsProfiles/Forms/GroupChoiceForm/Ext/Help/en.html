The list is used to select folders of the access groups profiles.

Due to the fact that there may be a lot of access groups profiles and they might have similar settings of the right limitations, there is an option to create folders in order to simplify search of the required access groups profile, and to understand the access restrictions in general.

Access group profiles can be classified based on specific features.

Selecting the folder for access groups profiles

  • Select the folder, and then click Select. You can also double-click with the left button.

Entering the folder for access groups profiles

  • Click Create group, and then enter folder name.
