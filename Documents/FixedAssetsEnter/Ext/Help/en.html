Fixed asset enter document

This document records information about a new fixed asset into the database.

Entry means that the fixed asset cost has been accumulated. It usually happens when fixed asset is being accepted for operation.

Document attributes:

  • Company - company, for which the document is Creatingd. It is selected from the Companies list. The attribute is displayed on the form if Accounting by Multiple Companies is enabled in accounting options.

  • Structural unit, warehouse - where the item, being accepted for accounting, is removed from. It is selected from the Structural unit list.

  • Cell - of the warehouse. It is selected from the Structural unit, warehouse Cells list. The attribute is optional.

  • ProductsAndServices - that is being accepted for accounting as a fixed asset. It is selected from the ProductsAndServices list.

  • Batch - of the item. It is selected from the Products and services batches list. The attribute is optional.

  • Quantity - item COUNT.

  • Amount - of item being accepted for accounting as a fixed asset.

  • UOM - unit of measure. It is selected from the UOM Classifier list or from the Units of Measure list.

  • Note - any additional information.

  • Fixed asset. Table fields:

    • Fixed asset - that is being accepted for accounting. It is selected from the Fixed assets list.

    • Useful life - which will be used for calculation of the depreciation. It is completed only for the fixed assets with the 'Linear' accumulation method.

    • Finished Goods Output - to be used for calculation of the depreciation. It is completed only for the fixed assets with the 'Proportionally to Finished Goods Output' accumulation method.

    • Amortize - if depreciation should be accrued.

    • In this month - if depreciation should be accrued as of this month.

    • GL Expense Account - GL Account where expenses for depreciation of fixed asset are accumulated.

    • Structural unit - to which expenses for depreciation of fixed asset are accumulated. It is selected from the Structural unit list with the Structural unit or tools types.

    • Business Activity - where depreciation accrual is accumulated. It is selected from the Business Activities lists.
