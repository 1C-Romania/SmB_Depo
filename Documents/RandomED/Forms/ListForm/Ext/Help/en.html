Sending arbitrary digital signature file from application to counterparty

For this purpose, you can use Arbitrary electronic document which is similar to an email and allows you to fill in a covering note, attach a necessary file, digitally sign (if required) and send a document to the counterparty.
The document can be used for unformalized documents exchange, for example, to send a contract or a reconciliation act.
Storage of arbitrary electronic document versions is not supported.
