Fixed assets sale document

The document registers property sales. The list of documents is stored in the property accounting document register.

Multiple transactions are performed whenever a document is posted:

  • Accrual of depreciation for the month of sale

  • Writing off a property from accounting

  • Show accounts receivable

To edit the document currency, click Edit prices and currency in the document form.

Document attributes:

  • Organization - an organization for which the document is created. It is selected from the Organizations catalog. The attribute is displayed in the form if Accounting by Multiple Companies is enabled in accounting options.

  • Counterparty - customer selected from the Counterparties catalog.

  • Contract - a contract regulating the document. It is selected from the Counterparty contracts catalog.

  • Comment - any additional information.

  • Property. Tabular section:

    • Property - property being sold to a customer. It is selected from the Property catalog.

    • Initial cost of the property.

    • Depreciated cost - initial cost less depreciation.

    • Depreciation total - depreciation accrued up until this month.

    • Depreciation this month - depreciation accrued for the current month.

    • Amount of sold property, including taxes according to the selected check boxes.

    • VAT rate - VAT rate selected from the VAT rates catalog.

    • VAT amount - amount of VAT.

    • Total - automatically calculated total value of property.

  • Setoff of advance payment. The table contains prepayments received from customer earlier. To edit data in the Setoff of advance payment table, click Edit setoff of advance payment. Tabular section:

    • Document - document with customer prepayment: Petty cash receipt or Payment receipt.

    • Amount of settlements - amount in contract currency.

    • Rate - rate used to recalculate Amount to document currency. The field is shown if the accounting parameter settings enable currency transaction accounting.

    • Factor - factor used to recalculate Amount to document currency. The field is shown if the accounting parameter settings enable currency transaction accounting.

    • Payment amount - amount of prepayment in document currency. The field is shown if the accounting parameter settings enable currency transaction accounting.
