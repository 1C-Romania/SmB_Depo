Budget document

The document Saves company budgeting information for some planning period.

 

Document attributes:

  • Company - company, for which the document is Creatingd. It is selected from the Companies list. The attribute is displayed on the form if Accounting by Multiple Companies is enabled in accounting options.

  • Planning Filter period - planning period which defines a periodicity, start and end dates of the projected balance. It is selected from the Planning Filter periods list.

  • Note - any additional information.

  • Openings. Table fields:

    • Account - account of balance selected from the Managerial chart of accounts.

    • Amount - balance Amount.

    • Note - transaction comment.

  • Direct Cost. Table fields:

    • Planning Company contract date - start date of the planning period. By default is equal to the start date of the planning period.

    • Account - expense account selected from the Managerial chart of accounts.

    • Source - corr. account selected from the Managerial chart of accounts.

    • Amount - Amount of expenses.

    • Note - transaction comment.

  • Indirect cost. Table fields:

    • Planning Company contract date - start date of the planning period. By default is equal to the start date of the planning period.

    • Account - expense account selected from the Managerial chart of accounts.

    • Source - corr. account selected from the Managerial chart of accounts.

    • Amount - Amount of expenses.

    • Note - transaction comment.

  • Income. Table fields:

    • Planning Company contract date - start date of the planning period. By default is equal to the start date of the planning period.

    • Bank, Petty Banking - cash assets GL account selected from the Managerial chart of accounts.

    • Source - corr. account selected from the Managerial chart of accounts.

    • Amount - income Amount.

    • Item - cash flow item selected from the Banking flow item list.

    • Note - transaction comment.

  • Outcome. Table fields:

    • Planning Company contract date - start date of the planning period. By default is equal to the start date of the planning period.

    • Bank, Petty Banking - cash assets GL account selected from the Managerial chart of accounts.

    • Direction - corr. account selected from the Managerial chart of accounts.

    • Amount - outcome Amount.

    • Item - cash flow item selected from the Banking flow item list.

    • Note - transaction comment.

  • Revenues. Table fields:

    • Planning Company contract date - start date of the planning period. By default is equal to the start date of the planning period.

    • Account - income account selected from the Managerial chart of accounts.

    • Source - corr. account selected from the Managerial chart of accounts.

    • Amount - revenue account.

    • Business Activity - selected from the Business Activities list.

    • Structural unit - selected from the Structural units list.

    • Customer Order (Project) - Customer Order document.

    • Note - transaction comment.

  • Expenses. Table fields:

    • Planning Company contract date - start date of the planning period. By default is equal to the start date of the planning period.

    • Account - account of expenses selected from the Managerial chart of accounts.

    • Source - corr. account selected from the Managerial chart of accounts.

    • Amount - Amount of expenses.

    • Business Activity - selected from Business Activities list.

    • Structural unit - selected from the Structural units list.

    • Customer Order (Project) - Customer Order document.

    • Note - transaction comment.

  • Operations. Table fields:

    • Planning Company contract date - start date of the planning period. By default is equal to the start date of the planning period.

    • Account Dr - debit account selected from the Managerial chart of accounts.

    • Account Cr - credit account selected from the Managerial chart of accounts.

    • Amount - Amount of operation.

    • Note - transaction comment.
