The application enables you to enter the reminders and specify the time and subject, which activates the list of reminders at the time specified.

Various catalogs, documents and other application objects can be the subject of the reminders. For example, you may enter a reminder linked to a certain partner, goods register etc., or a reminder about specifying or filling up any data.

You can open the reminder by running the Organizer - Remind command from various application objects. At that, there is an option to enter only one reminder for an object.

You can also enter a reminder linked to any personal event or activity in the My reminders list by running the Create command. Such a reminder can be useful, for example, to remember to wish a happy birthday to your colleague in time. The reminders combined with notes are well suited as an electronic replacement for the "stickers on the edges of a monitor".

Entering the reminder

  • Correct or type in the text of a reminder. If the reminder is linked to a document, catalog or other application object, the name of the reminder will be copied to the reminder text by default.
  • Select one of the options in the Remind field:
    • Within specified time - specify reminder date and time. Type the date manually or select from the calendar by clicking the button.
    • Interval. Select the value from the list to specify the interval of reminder actuation. There is an option to type random number of hours, e.g. in 10 hours.
    • Relative to subject - specify the remind date related to the reminder subject date, for example, related to a job completion time. If the object has at least one field with future date, this Remind field value is displayed and suggested by the application by default. IN the list select the time before the date of the reminder to display the reminder. If the date in the subject is changed, the reminder date changes automatically. If the date in the subject resets due to some reason, the reminder date will be considered as due, and the reminder will be displayed ahead of the schedule.
    • Periodically - for periodic reminders, for example, you can select the on Mondays at 9-00 values from the list or enter another schedule. To enter your own schedule select the given schedule value from the list and adjust the settings you need.

Cancelling the reminder

  • To cancel a reminder, click Stop.
  • Application displays a confirmation message. Click Stop to confirm an action. To cancel, click Continue.
