The application stores the address data, required for operation in the address classifier.

In order to provide structure and content unity for address data the Federal Information Address System (FIAS) is used. FIAS contains postal addresses of the Russian Federation. FIAS helps to fill the addresses correctly and quickly. You can not submit a report without it. For example, employees tax or pension reporting requires that the addresses are filled according to the standard, otherwise the report will not be accepted. Moreover, the addresses can be changed, and the user is not always able to track such changes himself. The application allows managing addresses and maintain them updated using FIAS.

The user can export the address data on any regions of the Russian Federation from the user support website of 1C:Enterprise application of 1C company and also from files, distributed by the Russian Federal Tax Agency.

To open the window for working with address classifier, run the Address classifier - Import classifier command in the Administration - Support and service section.

Importing the address classifier

  • You can select one, several or all regions of Russia for importing the address data when importing the classifier. It is convenient as the addresses of regions that will not be used for work may be skipped. Select the check boxes next to the names of the regions you need. To select all regions for import, click , to cancel, click. Click Search to search a region in the list. The bottom part of the list contains the number of selected regions.
  • Select Import option using the radio button:
    • From 1C website
      • You need to fill in the data for authorization on the user support website to export data. To do this, click the Enter login and password link.
    • From directory on disc
      • click to specify the path to a directory (you can also press F4);
  • Click Import after all parameters are specified.
  • During import the application displays the message on import progress:
    • number of imported regions;
    • source of import;
    • currently imported region.
  • After successful import a message is displayed. Click Close to return to the application.
  • If the previously imported into the classifier address data is imported once again, such data is overwritten.
  • The import of the address classifier is not supported in the web client.
