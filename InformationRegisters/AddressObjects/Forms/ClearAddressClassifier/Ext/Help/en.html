It is intended to delete address data (for example, data imported by mistake).

You can open the form by running the Address classifier - Clear address data command in the Administration - Support and service section.

  • Select check boxes with the region names you need.
  • To select all regions for clearing, click , to cancel the selection, click .
  • To search the regions in the list, use the Find button. The calculation process of the selected regions number is indicated in the bottom part of the list.
  • When all necessary parameters are set, click Clear.
