It is intended for storing and displaying currency rates.

Exchange rates can be automatically imported from the RBC website (using the scheduled task) or manually (see Importing exchange rates).

It is opened by running the corresponding command on the currencies navigation bar.

If the importing of exchange rates from the RBC website is not available, you can enter the rates manually.

The displayed list includes:

  • Exchange rate date
  • Currency - the currency character code according to the international standard
  • Exchange rate
  • Multiplicity

Entering currency rates manually

  • Click Create
  • Select the Currency from the list (according to the Currencies list)
  • Enter the Exchange rate date or select it with the help of the calendar
  • Enter the Exchange rate
  • Enter the Multiplicity - by default it is equal to 1. If the exchange rate of the currency versus ruble is very low (the accuracy is less than four digits of the fractional part), then it is necessary to specify the number of quoted digits. For example, the Vietnamese dong rate is 16.1574 rubles for 10000 Vietnamese dongs. IN this case in the Multiplicity field indicate the value of 10000, and in the Exchange rate field indicate 16.1574
