Designed to select template opening mode.

To open a template in the Print form templates, you can:

  • Press Enter - select a template opening mode.
  • Double-click - select a template opening mode.
  • Click Open
  • Click  Edit

Variants of opening template

Use the radio button to select how a template will open after double-clicking it (or pressing Enter):

  • Read only - the template is not marked as locked and opened with an external application only for reading. 
  • Read and edit - the template is marked as locked by you and opened with an external application for reading, editing and saving changes.

Hiding message

  • If you do not want to see this message again, select Do not ask again.
  • You can edit this setting later in the print form template list (click More - Define action when selecting print form template), or in the personal settings: Main - Define action when selecting print form temp.
