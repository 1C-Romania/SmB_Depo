The templates in the form of tabular documents can be edited in the application, and to edit the templates in the form of the documents in formats such as Microsoft Word and OpenOffice.org Writer, you should install the corresponding application on your PC.

As a rule, you do not need to take any specific actions to start the application needed, all you need is to click the template file and the application will start automatically (provided that the application is installed).

You may need to adjust the advanced settings to edit the template in the web client.

If the application has been installed already, click Continue.

The application with the help of which you can edit the template is defined by the type (format) of this template:

Template type

Application

.mxl 1C:Enterprise - Working with files
.doc Microsoft Word
.odt OpenOffice Writer
