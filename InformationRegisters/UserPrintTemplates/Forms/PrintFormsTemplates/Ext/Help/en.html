Designed to manage print templates. The list contains templates of all the print forms of the application.

Click Print form templates in Administration - Print forms, reports and processings to open the list.

The list contains:

  •  - sign if the template has been edited.
  • Print form template- full name of the print form, specifies the application used to make the template. The sign near the name indicates the application used for the template.
  • Template owner - a application object associated with the print form. Templates are grouped by application objects.

Filters

  • If there are a lot of templates, you can use the template filtering tools. 
  • You can find a template by a name or a name fragment. To do it, write the search term in the left field.
  • You can filter templates by a specific criteria with the Display field:
    • All templates
    • Standard templates (no changes)
    • Used templates with changes
    • Not used templates with changes
    • All templates with changes

Viewing template

  • Click  Open to view a template. You can also double-click and select a template opening mode. An opened MXL template is available in read-only mode.

Editing template

  • Print forms are generated from templates delivered in the application package. Meanwhile, delivered templates are editable.
  • To edit the template, click Edit. You can also double-click it and, when necessary, select template opening mode.
  • Templates in the form of table documents are edited by the application. To edit templates in the form of office documents (Microsoft Word and OpenOffice.org Writer), you need to install the relevant office software on your computer.
  • Before starting editing, you will be reminded of it with a message with a link to the pages of the relevant software to install. Click Next. 
  • To work in web client, you may need additional settings.
  • If a necessary application is installed on the computer, the template will open for editing.
  • After editing, save the template and close the office application. To finish editing and save the template in the application, click Finish editing.
  • After saving, the edited template gets the sign, i.e. the edited template will be by default used for print.

Switching between standard and custom templates

  • You can edit print form templates. Then you can switch between the edited variant and the standard one.
  • Switch using the corresponding buttons
    •  Use edited template (custom template)
    •  Use standard template. The  sign gets inactive showing that there is a custom template.

Warning

  • Some template changes may prevent display of print form. Generally, it is caused by deletion or renaming of template areas and parameters.
  • If the template has been edited and print form is no more generated, you can try to restore the template by cancelling the erroneous changes. 
  • If you can not correct the template,  Delete edited template. The standard template will be used instead.

Action when selecting print form

  • To specify an action to perform after double-clicking the template, click More - Define action when selecting print form template. Specify how to open the template. If necessary, select the relevant check box to avoid answering this question again.
