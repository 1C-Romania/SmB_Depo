Use it to specify parameters of connection to another application for data synchronization, and to check such parameters. Tabs define the parameters of connection types. You can also specify connection parameters when setting up data synchronization in the Data synchronization assistant. These settings are displayed by default and can be changed.

To go to Connection parameters, use the relevant command of the data synchronization settings.

All connections have the following common parameters:

  • Default connection - select the default connection for automatic and interactive data synchronization.
  • Check connection - click to automatically check the specified settings and try to connect to another application.
  • Compress data when sending - select the check box to limit the message size. The check box is not used for setting up direct connection to an infobase.
    • Archive password - enter a password for packing and unpacking an exchange message archive. It is supposed that, for a couple of exchanging applications, the packing password in the source application and the unpacking password in the receiver application are the same.

Direct connection

  • Select an infobase variant:
    • Correspondent file base 
      •  Select Infobase directory.
    • Correspondent client server base
      • Enter Server name
      • Enter Server infobase name
  • If necessary, select the Operating system authentication check box (it is cleared by default). IN this case, you do not fill in the following fields.
  • Enter User name
  • Enter Password

Internet

  • Enter the Internet Address of the application.
  • Enter User name and Password.
  • If necessary, select the Remember password check box.
  • Click the Internet access parameters link to configure the proxy server parameters.

Network directory

In the Directory field, Select a path to a local or network directory.

FTP resource

  • Enter Path to directory and computer Port for data transmission.
  • Enter User name (login) and Password.
  • Specify the Maximum file size to limit the size of your message.
  • If necessary, select the Passive connection check box.
  • Click the Internet access parameters link to configure the proxy server parameters.

Mail messages

  • In the Email account, check the list to find a necessary account or add a new account.
  • Specify the Allowed attachment size to limit the volume of your message.
