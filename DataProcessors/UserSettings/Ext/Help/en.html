It is intended to manage user settings.

The commands of user settings management can be found on the User settings and rights panel of the Administration section.

Administrator manages the user settings.

Available actions

  • Copy settings - copying user settings from one user to another. It can be necessary, for example if you need to adjust the settings of several new users in one group in which there are users already.
  • Clear settings - delete settings of all or selected users. IN some cases it is necessary to delete the user settings, for example if work in the application became impossible because of these wrong settings.
  • User settings - run this command to open the list to manage user settings.
