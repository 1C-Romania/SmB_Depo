Select the user settings to copy or clear:

The list consists of three tabs:

  • Appearance - appearance settings of the desktop, forms (e.g. application lists) and the application command interface.
  • Report settings - a list of report settings is grouped by report variant names.
  • Other settings - personal user settings, table document print settings, quick access settings for additional reports and data processors, the Favorites section settings, and other settings not included in other sections.

Selecting settings

  • Select necessary settings using check boxes. You can use the following options: 
    • Click to Mark all settings.
    • Click  to Unmark all settings.
  • To copy or clear settings, click Select.

Viewing settings

  • Double-click the required report (or use the More - Open command) to view the report and appearance settings for the current user.
