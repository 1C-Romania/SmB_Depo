Select one or more users to which the settings will be copied (these users' settings will be cleared).

The list consists of two parts: in the left, there are groups; in the right, there are users (external users) of these groups. The default group is All users (All external users).

To view all users of the child groups with proper hierarchy, select the Show users of child groups check box at the bottom of the list.

User selection

  • Select the necessary users with check boxes. You can use the following special options:
    • Click to Mark all users.
    • Click  to Unmark all users.
    • Hold down Ctrl or Shift and click necessary users; then click More - Mark selected.
    • Click the selected users, and then click More - Unmark selected.
  • To copy or clear settings, click Select.

Viewing active users

  • The users to which the application appearance settings are copied must preliminarily log off. Otherwise, the settings may not be applied.
  • Click the Active users button to view the users currently working in the application.
