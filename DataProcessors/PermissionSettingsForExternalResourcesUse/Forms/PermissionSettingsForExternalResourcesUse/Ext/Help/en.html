If a safety profile, limiting access to external (related to 1C:Enterprise servers cluster) resources are used in  1C:Enterprise servers cluster for this database, then it is required to make changes to the permissions structure in the safety profile for application correct work.

External resources that will be availabe for use are listed in the list.

Click Apply changes for confirmation.

Besides, it is necessary to check that:

  • access to external resources is not locked by other software installed on the server (antiviruses, traffic restriction applications);
  • necessary rights in users accounts of the operating system are configured to run working servers of 1C:Enterprise cluster by these users.
