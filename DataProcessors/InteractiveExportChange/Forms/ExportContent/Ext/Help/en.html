The report is intended for displaying the statistics on the structure of the data imported during the synchronization.

The report can be viewed by clicking the Imported data structure report link from the Interactive synchronization assistant. The report is generated automatically.

The heading of the report contains the name of the application, with which the data is synchronized, the report also contains the date and time of the report generation.

The report includes the columns:

  • Data - the list of application objects (lists and documents) that are involved in the synchronization.
  • Selected:
    • By general rules;
    • Additional - it is selected on the page of the Interactive synchronization assistant;
  • Total objects in base;
  • Will not be imported - the number of items in the selected objects that will not be imported;
  • Will be imported - the number of items in the selected objects that will be imported.

The total number of the imported items is counted in the report.

The content of the imported data may include additional objects to ensure the integrity of data.

The lower part of the list also includes:

Data synchronization rules - the detailed synchronization rules that were accepted for this database;

Additionally added data - the list of data that was added on the corresponding page of the Assistant.
