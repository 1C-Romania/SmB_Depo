It is intended for filtering data which will be imported to other application during data synchronization.

You can open the list by clicking the link in the With filter field  to add data on the Data for importing field of Interactive data synchronization wizard.

Data to be imported

  • Click Pick to choose data for importing and to fill in the list. Objects of the application for selection are shown on the form as a tree. Select what you need, and then click Select. Data is placed in the list for importing.
  • To delete data from the list, click More - Delete.
  • The list of the imported data consists of the columns:
    • Data for import - the objects of the application selected for importing are listed in this column;
    • Filter - The All objects (link) filter is chosen by default. Click the link to select necessary items of the chosen lists.
    • For import/total - the number of the chosen items and total number of the list items are counted. Click the Will be imported link in the bottom of the list to re-count the selected items.
  • Click OK to confrim the selection and return to the Assistant to continue setting the data synchronization.

Viewing the content of data to be imported

  • Click Data content to see Information on the content of data to be imported.

Saving and using importing settings

  • By clicking the Settings button you can save entered settings for selection of data or to use the previously saved settings. The application suggests using previous settings, for example, settings saved automatically, for this purpose choose from the offered list the Last importing value. You can choose previously saved setting from the list. You can save your own settings. To do this, click Save current settings. Type in the name of the saved settings, and then click Save.

Calculating the number of the objects to be imported

  • In the lower part of the list click on the link in the Will be imported field to calculate the number of the imported objects. After calculation in the For importing/Total column of the list the number of items is filled in respectively and the link becomes equal to the total number of items to be imported.
