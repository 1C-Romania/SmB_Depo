You can use the form to set the parameters of application update via Internet.

To update from the user website you need to specify the user's code and access password. Only registered users of 1C:Enterprise applications are authorized to get updates. You can find the detailed information on how you can get the user's code and password in the support section on the http://1c-dn.com/">v8.1c.ru site.

To open the form go to the Administration - Support and service - Application update menu, and then run the Setting application update via Internet command.

Select the automatic check of updates using the radio button:

  • Disabled;
  • On application startup;
  • According to schedule - when selecting this option fill the schedule of automatic check of updates. To do this, click the Not defined link or other link with earlier entered schedule.

Fill the access parameters on the http://1c-dn.com/">1c-dn.com user site:

  • User's code;
  • Password.

If necessary, select the Remember password check box. Click the How to get the user's code and password link if you are not able to update application via Internet.
