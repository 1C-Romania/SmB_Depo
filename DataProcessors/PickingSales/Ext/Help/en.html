Product selections for sales documents data processor

The data processor is used to put products and services into the unfilled tabular section of a document. You can run the data processor in documents with tables using the Select button. Products and services are selected from the Products and services catalog in the selection mode.

IMPORTANT! You can run the basic functions of operation with lists using the shortcut menu.

Functionality of lists and shortcut menu bars

Products and services is the key catalog for the selection mechanism. This catalog includes information about inventories, services and other items used in the economic activity. It is the catalog which is used to generate and display additional information in the selection data processor. The selection data processor provides the following additional information on products and services:

  • list of characteristics (if the corresponding option is selected)

  • prices of products and services according to the type of prices specified in the document from which the selection was run. You can display information about the current price type if you run the Document information command

  • stock balance, available stock balance and indication of stock reserve availability (if stock reservation is enabled). For inventory items for which reserve is available, you can see the reserve details: which documents were used to finalize the reserve (you can run the command from the shortcut menu)

    Information of lists and shortcut menu bars

    Main information is provided in the form of lists. Shortcut menu bars (the right-click menu) include all auxiliary commands which are grouped for convenience. Below there is a list of commands available on the shortcut menu bars.

    List of Products and Services

  • Add a product – to add a product into the basket

  • Create – to create a new item of the Products and services catalog

  • Change – to edit the current item of the Products and services catalog

  • Copy – to create a new item of the Products and services catalog by copying values from the current item into it

  • Move to the parent – to position on the folder being the parent to the Product and services catalog item in the products and services tree. After the positioning, the products and services list will be filtered: the user will have access only to the items included in the current folder

  • Detail the stock – to open the list of documents itemizing the stock

  • Products and services analogs – to display only the current products and services and their analogs (products and services analogs are specified directly from the Products and services catalog item card) in the products and services list

  • Create/change price – to create/change quickly the current price for a specified item

  • Update – to update data of the products and services list

  • List adjustment – to call dynamic list settings

  • Export list – to export the products and services list content in a tabular document (used, for example, for printing out or saving the list)

  • Change form - to change the layout, size, order and other properties of the selection form items

    List of Characteristics of Products and Services

  • Add a product – to add a product with the current characteristic into the basket

  • Create – to create a new item of the Products and services characteristics catalog

  • Change – to edit the current item of the Products and services characteristics catalog

  • Display products and services – to move (return) to the products and services list

  • Detail the stock – similar to the command in the products and services list

  • Create/change price – similar to the command in the products and services list

  • Update – to update data of the products and services characteristics list

  • List adjustment – to call settings of the products and services characteristics dynamic list

  • Export list – to export the products and services characteristics list content in a tabular document (used, for example, for printing out or saving the list)

  • Change form - similar to the command in the products and services list

    List of Stock Balance (If the Use of Several Stocks is Enabled)

  • Update – to update data of the list of stock balance

  • Export list – to export the stock balance content in a tabular document (used, for example, for printing out or saving the list)

  • Change form – similar to the command in the products and services list

    List of Selected Products and Services (Basket)

    Selected products and services are the tabular section of the selection data processor, therefore the shortcut menu commands are similar to the tabular section commands used in documents in terms of content and functions.

    Selection navigation and keyboard shortcuts

    The selection form has a command bar which includes additional options of form management. These options include:

  • List display management: stock balance, basket

  • Quick navigation between elements of the form with indication of keyboard shortcuts. You can use the following keyboard shortcuts when selecting products:

    Alt+F3 to move to the products and services list or the products and services characteristics list

    Alt+F3 to move to the full-text search bar

    Alt+F5 to move to the products and services tree list

    Alt+F6 to move to the list of selected products

    You can also navigate using the Tab button

  • Adjustment of selections to display only products and services with the specified price or inventories available

  • Names update. This option is available when activating the use of characteristics or several stocks and with this option it is possible to enable/disable the update of corresponding list names (products and services characteristics and stock balance)

  • Document information. This option opens a form with information of the document from which the selection is run

    Additional information about selection

    In lists, in addition to the context search, there is an option to use typical search capabilities.

    IMPORTANT! You can run a list search by Ctrl+F or by typing the text required.

    Note that the full-text search and the list search are different tools and the results of these operations are summarized. For example if you use a full-text search, you may select all products and services items which names contain the word "fasteners" (in any place, not necessarily in the beginning), and then using a list search you may select products and services which names start with the word "construction".

    Search cancellation:

  • for a full-text search, it is necessary to clear the field with the typed text (in the selection heading);

  • for a list search, it is necessary to activate the required list and press Ctrl+Q;

    IMPORTANT! You can activate a full-text search in the application settings.

    You can select goods with or without requesting quantities and prices (the Settings option):

  • when selecting checkboxes, the specified parameter of the products and services item is requested

  • if no checkbox is selected, at each repeated selection of this products and services item the quantity (and reserve) is increased by 1.

    The application automatically transfers the chosen products and services items to the tabular section of the document.

    To transfer selected items into a document use the Transfer to document option.

    IMPORTANT! Selection is not used for editing filled tabular sections.
