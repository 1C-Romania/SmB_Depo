If a safety profiles, limiting access to external (related to 1C:Enterprise servers cluster) resources are used in 1C:Enterprise servers cluster for this database, it is required to make changes to the permissions structure in safety profile for application correct work.

External resources that will be availabe for use are listed in the list.

Besides, it is necessary to check that:

  • access to external resources is not locked by other software installed on the server (antiviruses, traffic restriction applications);
  • all necessary access rights in users accounts of the operating system are configured to run working servers of 1C:Enterprise cluster by these users.
