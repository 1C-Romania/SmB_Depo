Base of goods exporting into DTC data processor

 

The processing is meant for exporting of product data into data collection terminal base.

·         The Fill button is used to fill in the products list. Only those goods to which barcodes are assigned are put on the list.

·         To export data into data collection terminal the Export button is used. The goods information exporting is made if the data collection terminal is connected and exporting parameters (exporting file format, exporting file path) are set.
