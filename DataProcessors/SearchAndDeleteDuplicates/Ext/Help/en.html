It enables you to search for and delete duplicates from all application lists (for administrators).

Users tend to enter duplicates into the application (for example if they fail to find an item and enter it again). The duplicates prevent the application from normal operation. Either the original or its duplicate is specified in documents. So when the documents are posted, incorrect account entries occur, and the balance does not match. That is why it is necessary to avoid duplicates in the application.

The application finds all duplicates and replaces the duplicate refs with the original ones in documents. Then it marks duplicates to be removed from the list.

You can run the Search and delete duplicates command in the Administration - Support section.

Search for duplicates

  • In the Search field , select one of the application lists.
  • In the Filter field, specify the filter parameters for the list items by clicking All items (default value), and then define the Filter rules.
  • Select the Field to filter and the search value, in the Comparison kind field, select a comparison operation, and then specify the Comparison value. Click OK to search for duplicates again. The hyperlink changes to the filter parameter value.
  • To delete the filter, click .
  • Predetermined rules of duplicates search in the list are automatically added to the Compare field. To add items to the rule, click the hyperlink, select check boxes for the rule items, and then click Select. Once you select, the hyperlink changes the value.
  • Once you define the duplicates search criteria, click Search for duplicates.
  • If you fail to find duplicates, change the criteria, and then click Search for duplicates.

      Duplicates list

      • A list of found duplicates consists of two parts:
        • On the left side, you can see the found items. They are shown as a tree in the list. The list includes the following columns:
          • Name - the list items are grouped by names.
          • Code
          • Usage location - number of the item usage location.
        • On the right side, you can see the selected items details:
          • By the item group (i.e. by each name) - report on the number of found duplicates of the selected item.
          • By items - report on the usage in the application ("Is not used" or a list of documents containing its occurrences).
      • One of the items used in the application is automatically selected as Original.
      • To change the item, click it, and then click Mark as original.
      • The duplicates are flagged in the list.
      • Click Open to view the selected item or double-click it.
      • You can manage the group visible in the list by clicking buttons Expand duplicate group or Collapse duplicate group.
      • In the info bar of the list, you can see a number of items and found duplicates.
      • If you are not satisfied with the duplicates you found, change the search criteria, and then click Search again.

      Deleting duplicates

      • If you agree with all the conditions listed in the list info bar, click Delete duplicates. Duplicates will be marked to be deleted, their occurrences in documents will be replaced with the selected Original.
      • To exit without deleting, click Close. Search criteria and the duplicates list are not saved.
      • To permanently delete duplicates, in the Administration - Support section, run the Delete marked items command.

      Viewing duplicates refs

      • Click the All usage location hyperlink to view report Usage location by items from the duplicates list that are used in the application documents.
