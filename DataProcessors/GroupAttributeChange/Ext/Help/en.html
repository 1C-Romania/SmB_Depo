Group change attribute

There are two options for attributes bulk edit:

  • You can run the Edit selected command for most application lists.
  • Only for administrators: in the Administration - Support section, run the Attributes bulk edit command. If it is handled externally, in the relevant section, run the Additional data processors - Attributes bulk edit command.

Editing selected list items

Change of attribute values (including additional attributes and data) for the selected list items.

Enter new values of attributes for the selected list items

  • You can see editable fields of the selected list items in the rows of the Attribute column. You can change multiple attributes at a time.
  • To change the attribute values (including additional attributes and data), fill in New value in relevant fields.
  • Double-click the entry bar to open it. Here you can:
    • Enter a value to search for in the list.
    • Press F8 (Show all) to select a value from the relevant list.
    • Run the Create command to add a new value to the list.
  • The check box is selected automatically whenever you enter a new value.
  • Click Edit attributes.

Clearing box values of the selected list items

  • To clear the attribute value, select the check box in the relevant row. Leave the New value column empty.
  • Click Edit attributes.

Editing results with errors

  • You can see a list of unchanged list items in a separate window. You can see the error reason for each list item.
  • Click Open to view the items that could not be changed.
  • Click Edit again to try to edit the list items again with the same parameters.
  • To correct the error, you might need to change the attribute values for the items.

Selecting previously saved settings for editing attributes

  • Run the Previously edited attributes command.
  • In the drop-down list, select the setting you need. 30 last settings of the attribute changes are shown in the list.

Editing locked attributes

  • In the list, attributes that can not be changed by default appear dimmed.
  • These are key attributes that can affect other lists and documents of the application. That is why they are protected from random changes.
  • If you try to enter new values, the application prompts to unlock them. If you need to change them anyway, you can confirm your selection by clicking Yes.

Editing posted documents

  • If a posted document is edited, it will be posted again.

Settings of attribute edit parameters

  • The application uses default settings to change attributes. Run the Additional parameters command to adjust the settings.

Attributes bulk edit (for administrators)

  • Administrators can edit attributes of all application catalogs and documents with tabular sections. They can also select items to edit more precisely in the Select items to edit table.

Selecting items to edit

  • In the Edit field, select a document or a catalog with items you want to change. By default, you can Edit all items of the catalog or all documents of the selected type.
  • To view the items to edit, click Edit all items.
  • With the Set filter command, you can filter one or multiple items by certain criteria. If you set a filter, the changes are made only to the selected items.
  • Once you set filter criteria, the link appears as Items selected. It shows how much items are selected to edit.
  • Once you select a catalog or document type, in the Editable attributes table, you can see the attributes that can be changed including the service ones. To show service attributes, run the Additional parameters command and select the relevant check box.
  • Once you select the attributes you need, use standard commands to edit them.
