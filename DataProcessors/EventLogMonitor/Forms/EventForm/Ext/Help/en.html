Viewing properties of a application event displayed in the registration log.

Displays full information on the application event rather than displaying only data shown in the registration log:

  • User - the full name of the user
  • Date, time - the exact time of the event
  • Application - for example, Thin client
  • Computer - the net name of the computer
  • The Data fields group:
    • Metadata - the type and name of the application object: list, document kind, etc.
    • Data - the specification of the application object: the specific list item, document, etc.

    • Presentation - the specification of the application object

  • The Event fields group:
  • Event - the event type
  • Comment - the brief description of the event
  • The Session field group - the full information on the session in which this event occurred:
    • Number - the session number
    • Working server
    • Main IP port
    • Auxiliary IP port
  • The Transaction fields group:
    • Identifier
    • Transaction completion status - for example, Fixed

      The registration log provides for the opportunity to manage content of columns displayed, which makes it possible to display only important information.
