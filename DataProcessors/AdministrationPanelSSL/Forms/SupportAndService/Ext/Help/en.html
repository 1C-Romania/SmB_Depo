﻿ This panel is designed to control the Status of the application, backup and reSave, version update, productivity optimization.

  • Event log
    		•  Viewinf of the EVENT and actions which were taking place in regulation tasks or during the work of users in the application.
  • Active users
    		•  The list of users working in the application at the momemt.
  • Blocking the work of the users
    		•  Blocking and unblocking the work of users with the application. When blocking the user the session of all the users is closing (inckuding the current user).
  • Deleting the marked objecs
    		•  Final deleting of that objects which previously have been marked for deletion.

    Regulation operations

  • Regulation and background jobs
    		•  Setting up, performing and tracking the statuses of the regulation and background jobs. Regulation tasks automatically perform the regulation operations according to the schedule.
  • Company contract date of prohibition of the data changes
    		•  The setup of the prohibition of the previous periods data change for the users of this application.: See also: Company contract dates of prohibition the data loading in Data synchronizing.
  • Totals and aggregates management
    		•  Management of the totals for the accumulationand accounting registers as well as aggregates of the turnover registers. The correct totals and aggregates setup can raise the application productivity.
  • Full-text search management
    		•  Support of the full-text search IndexA in active status.

    Backup and restoring

    To decrease the risk of the data loss it is necessary to regularly perform the backup of the application. The frequency of the backup performing depends on how often you enter the new data.

  • Application backup
    		•  Performing the backup of the application: right now, in some time or when finishing the session.
  • Backup setup
    		•  Automatic backup setup accrding to the schedule or turning off the control of the backup if it is performed by external tools.
  • Restoring from the backup
    		•  Restoring of the application from the backup.

    Performance rating

  • Performance rating
    		•  The tick of estimation of the application performance according to the APDEX methodology.
  • Performance indicators
    		•  Viewing anf estimation of the application performance.

    Classifiers

  • Import banks classifier
    		•  Exporting and updating the Russian banks Classifier from the Internet or from the 1C:ITS disc.
  • Export the currency exchange rates
    		•  Exporting the currency exchange rates from the Internet.
  • Export the address classifier
    		•  Exporting and updating the classifier of Russian addresses from the Internet, from the 1C:ITS disc or local folder.

    Personal data protection

  • Registering of the EVENT of access to the personal data
    		•  Registerinf of the EVENT of access according to the requirements of Federal Law as of 37.07.3006 #153-FZ "About the personal data"
  • Personal data protection
    		•  Viewing of the references to the personal data of the selected objects of the application.

    Update of the application version

  • Configuration update
    		•  Obtaining information about the last updates on the user website in the Internet and assistant of an automatic update setup of the application.
  • Abbreviation of the system changes
    		•  Viewing a list of changes in the Seted version of the application and recommendations for updating the application.
  • Detalize the update process in the log of EVENT
    		•  When the setting is enabled to the log of EVENT are recorded the running update processors with the specifying of the execution time.
