This panel is designed to configure the parameters of import, storage and processing of files.

  • Store files in volumes on drive
    		•  Save files in the infobase or in volumes on the drive.
  • File storage volumes
    		•  Configure connection to the file storage volumes (folders) on the file server.
  • Restrict importing files with extensions
    		•  Select the check box to restrict import of files with the specified extensions; specify the file extensions restricted to be imported in the input field separated by spaces.
  • Maximum file size
    		•  Maximum size of imported files. Files exceeding the specified size will not be imported.
  • ODF file extensions
    		•  Files extensions of the open document format for the office applications. The extensions are to be separated by spaces.
  • Text file extensions
    		•  Extensions of the files containing text data. Extensions are to be separated by spaces.
  • Digital signatures
    		•  Allow or forbid to use digital signatures and check that nobody has modified the files since the digital signature was generated.
  • Cryptography settings
    		•  Connection parameters of the cryptography tools: digital signature and encryption.
  • Scanned documents numbers
    		•  View the numbering of the scanned documents and their owners.

    See also: Application settings
