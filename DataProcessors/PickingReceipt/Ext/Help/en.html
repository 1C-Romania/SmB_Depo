Product selections for revenue documents data processor

The data processor allows you to put products and services (inventories, services) into the unfilled tabular section of a document. You can run the data processor in documents with tabular sections using the Select option. Products and services are selected from the Products and services catalog in the selection mode.

IMPORTANT! You can use the basic functions of working with lists using the shortcut menu.

Functionality of  lists and shortcut menu bars

Products and services is the key catalog for the selection mechanism. This catalog includes information about inventories, services and other items used in the economic activity. It is the catalog which is used to generate and display additional information in the selection data processor. The selection data processor provides the following additional information on products and services:

  • list of characteristics (if the corresponding option is selected)

  • prices of products and services according to the type of prices specified in the document from which the selection was run. You can view information about the current price type if you run the Document information command

  • stock balance, available stock balance and indication of stock reserve availability (if stock reservation is enabled). For stock for which reserve is available, you can see the reserve details: which documents were used to make the reserve (you can run the command from the shortcut menu)

    Description of lists and shortcut menu bars

    You can find main information in the form of lists. Shortcut menu bars (the right-click menu) include all auxiliary commands which are grouped for convenience. Below there is a list of commands available on the shortcut menu bars.

    List of Products and Services

  • Add product – you can add a product into the basket

  • Create – you can create a new item of the Products and services catalog

  • Change – you can edit the current item of the Product and services catalog

  • Copy – you can create a new item of the Product and services catalog by copying values from the current item into it

  • Move to parent – you can position on the folder being the parent to the Product and services catalog item in the products and services tree. After the positioning, the products and services list will be filtered: the user will have access only to the items included in the current folder

  • Detail the reserve – you can open the list of documents with the details on the reserve

  • Products and services analogs – you can view only the current products and services and their analogs (products and services analogs are specified directly from the Products and services catalog item card) in the products and services list

  • Update – you can update data of the products and services list

  • List settings – you can adjust the settings of the dynamic list

  • Export list – you can export the products and services list content into a tabular document (used, for example, for printing out or saving the list)

  • Change form – you can change the theme, size, order and other properties of the selection form items

    List of Characteristics of Products and Services

  • Add a product – you can add a product with the current characteristic into the basket

  • Create – you can create a new item of the Products and services characteristics catalog

  • Change – you can edit the current item of the Products and services characteristics catalog

  • Display products and services – you can move (return) to the products and services list

  • Detail the reserve – similar to the command in the products and services list

  • Update – you can update data of the products and services characteristics list

  • List settings – you can adjust the settings of products and services characteristics dynamic list

  • Export list – you can export the content of the list of products and services characteristics into a tabular document (used, for example, for printing out or saving the list)

  • Change form – similar to the command in the products and services list

    List of Stock Balance (If the Use of Several Stocks is Enabled)

  • Update – you can update data of the list of stock balance

  • Export list – you can export the stock balance list content into a tabular document (used, for example, for printing out or saving the list)

  • Change form – similar to the command in the products and services list

    List of Selected Products and Services (Basket)

    The selected products and services are the tabular section of the selection data processor, therefore the shortcut menu commands are similar to the tabular section commands used in documents in terms of content and functions.

    Selection navigation and keyboard shortcuts

    The selection form has a command bar which includes additional options of form management. You can use these options to:

  • manage the list display: stock balance, basket

  • navigate quickly between items of the form displaying the keyboard shortcuts. You can use the following keyboard shortcuts when selecting products:

    Alt+F3 to move to the products and services list or the products and services characteristics list

    Alt+F3 to move to the full-text search bar

    Alt+F5 to move to the products and services tree list

    Alt+F6 to move to the list of selected products

    You can also navigate using the Tab button.

  • set filters to display the products and services only with the price specified or inventories available

  • Update names. This option is available when activating the use of characteristics or several stocks and with this option you can enable/disable update of corresponding list names (products and services characteristics and stock balance)

  • Document information. This option opens a form with information on the document from which the selection is run

    Additional information about selection

    In lists, in addition to the context search, there is an option to use typical search capabilities.

    IMPORTANT! You can search within a list by running the Ctrl+F command or by typing the text you need.

    Note that the full-text search and the list search are different tools and the results of these operations are summarized. For example if you use a full-text search, you may select all products and services items which names contain the word "fasteners" (in any place, not necessarily in the beginning), and then using a list search you may select products and services which names start with the word "construction".

    Search cancellation:

  • for a full-text search, it is necessary to clear the field with the typed text (in the selection heading)

  • for a list search, it is necessary to activate the required list and press Ctrl+Q

    IMPORTANT! You can activate a full-text search in the application settings.

    There is an option to select goods with or without requesting quantities and prices (the Settings option):

  • when selecting checkboxes, the parameter of the products and services item is requested

  • if no checkbox is selected, at each repeated selection of this products and services item the number (and reserve) is increased by 1

    The application automatically transfers the selected products and services items to the tabular section of the document.

    To transfer the selected items into a document use the Transfer to document option.

    IMPORTANT! You can not use Selection for editing filled tabular sections.
