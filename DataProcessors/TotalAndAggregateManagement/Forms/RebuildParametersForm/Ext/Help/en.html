This form is designed to specify parameters of rebuilding the aggregate use and to receive a list of optimal aggregates.

  • Maximum relative size - maximum total aggregate size in percents of the flow table. If the parameter is not specified or is null, there is no limit for the aggregate size.
  • Minimum rebuilding effect - extra percentage necessary for the old network effect; if the new network increases the effect by the Minimum rebuilding effect %, the method really rearranges the network. If the parameter is not specified or is null, there is no requirement for the effect. This parameter is not available for obtaining a list of the optimal aggregates.
