The form is used to delete duplicate data.

You can open the from the application lists by running the More - Merge selected command. You can select and merge several items.

The list of merged items includes:

  • Name
  • Code
  • References - indicates the number of the item usage or the No references message

Merging items

  • The first item in the list  is marked with the icon meaning to be the merging target by default.
  • From the list select the "correct" item you want to leave in the application, and then click Mark as original. After that the selected item will be marked with the icon as the merging target.
  • At the bottom of the list you can see the total number of items to be merged and item name to be left in the application. It means that other items will be marked for deletion (the administrator can change this link).
  • Click Merge.
  • After merging the selected item is left in the application and other specified items are replaced in all references by the selected item and marked for deletion (the user with administrator rights can permanently delete the items).
  • If there are any errors during item processing, the list of unprocessed items will be displayed with the description of failure, for example if the items belong to different list groups with different accounting system, then merging is not possible.

  • To continue merging, you should eliminate the cause of errors or change item content.

Changing item content in the list

  • If specified items are not applicable, you can Add new item. Select an item from the opened list or add a new one.
  • If there are more than 3 items in the list, you can Delete the items you do not need from the list to be merged running the More command.

Viewing items

  • Select the required item, and then click Open.

Viewing refs to list items

  • Using All references link at the bottom part of the list you can view References report for items in the list.

Configuring item deletion (for administrators only)

  • If the user has administrator rights, the user can permanently delete all items marked by the application for deletion by changing the marked for deletion link to permanently deleted at the bottom of the list.
