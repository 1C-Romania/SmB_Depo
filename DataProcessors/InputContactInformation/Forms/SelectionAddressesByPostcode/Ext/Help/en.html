The form is used to select the address based on postal zip code.

To find the address based on the zip code, it is necessary to have corresponding region address classifier exported.

You can open it from the address input window automatically, or by running the More - Fill in based on the zip code command after zip code entry.

That part of the address rebuilt based on the zip code is dispayed at the upper part of the window. You need to correct it only. You can select as follows: streets of the city or settlements and streets of that region settlements. Select the part of the address you need, and then click Select. You can also double-click. After the street is selected, enter remaining parts of the address in the opened window.
