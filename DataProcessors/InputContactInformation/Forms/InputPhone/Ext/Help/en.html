The form is used to enter phone (fax) numbers using the template.

You can open the form from the document tabular section or item list, which are the containers of contact information, by clicking Select.

To enter phone (fax) number, it is necessary to fill in the fields:

  • Country code - the application automatically adds "+" to the country code
  • City code is automatically entered in brackets
  • Phone number is entered in arbitrary form
  • Extension number - the application automatically enter "ext." before extension numbers
  • Notes - enter additional information about the phone number

Clearing phone (fax) numbers

  • To delete wrongly entered phone (fax) number and start entering again, you can run the More - Clear phone number command.
