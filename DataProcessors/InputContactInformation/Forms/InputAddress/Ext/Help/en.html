The form is intended for the address entry by the template. When filling in the form, you can use the address classifier.

It is opened using the Select button from the Address field in the list items.

  • Country field allows to select the country of the address. Editing the addresses outside Russia is performed in the common field ignoring the parts of address. Russia is selected by the application by default, at that the field may be uneditable depending on the settings. To change the country, click Select. The country is selected from the Classifier of the countries of the world (OKSM).
  • Zip code field is intended for entering postal zip code that corresponds to the specified address. If the address is found in the address classifier then the zip code can be entered automatically. If the address classifier for the corresponding region is exported, you can fill in the address by the zip code.
  • City, Settlement field is intended for entering the region, city, etc. If necessary, you can enter the address parts in details clicking  Select. Enter data of the settlement in the opened template.  
  • Street field is used to select (click Select and find the required street in the list) or enter the street of the address.
  • Number, Building, Apartment fields allow to enter corresponding parts of the address. You can change these variants by selecting the values from the drop-down list.
  • It is also possible to Add additional fields to enter other predefined parts of address, such as: Unit, Letter, Facility, Office, Box, Room, etc.
  • Address presentation field is uneditable when filling in the address by template. The field is filled in by the application automatically during address fields entering.
  • Note field allows to add any comment to the address that won't be displayed in the address presentation and printed.

Examples of the address filling in

    • To enter the address "Saint Petersburg, Pisarevskiy prospekt, 2, Building 2, Letter Z, office 12345", it is necessary to enter:
      • In City, settlement field select "Saint Petersburg city".
      • In Street field select "Pisarevskiy prospect".
      • In Number field enter "2".
      • In Building field enter "2".
      • To enter the letter, you should add the field by clicking Add, select from the drop-list the Letter value and fill in the field with the "Z" value.
      • Instead of Apartment select Office from the drop-list, fill in the field with the "12345" value.
      • To enter the address containing different values of the fields, for example: "Moscow, Admiral Lazarev str, house property #89, Building 16, unit 1, it is necessary to:
      • In City, settlement field select "Moscow city"
      • In Street field select "Admiral Lazarev str".
      • Instead of Nunber select from the drop-list House property and enter "89" to the field.
      • In the field Building enter "16".
      • Press Add button and select from the drop-list the Unit value and enter "1"

    Check of address filled in

    Press Filling check button to analyze the correctness of the address filling in. The application checks the following:

    • the address filling;
    • the presence of invalid symbols;
    • the length of the address items;
    • the correspondence to the classifier. IN client-server base the address check to classifier correspondence is performed at the server. After the address filling is checked, the application displays the message "The address is entered correctly" or the list of errors. The filling check is not possible for foreign addresses, therefore they are not checked. The command is also unavailable if the address is entered in free format.

    Address entry in free format

    • Use Enter address in free format command in More menu to enter knowingly incorrect address at the address entry by the template (for example, it is necessary to enter incorrect supplier address specified in the way-bill, to save the address for some time and to print it, but such address can not be used in future). The address is used for storage, therefore it is recommended to use it only for individual cases.
    • When the command is selected, the application displays the warning message: "Addresses entered in free format can fail the check by address classifier". To confirm the choice click Yes, to return to the address entry by the template click No.
    • The template of the address entry is displayed but all its fields exept Address Presentation and Note fields, are inactive. Filled in Address Presentation field. When switching to Note field or clicking OK the fields of the template that the application has recognized in the address line, are filled in (i.e. these fields have no errors). IN this case the check of the address filling is ignored.
    • To confirm the address press OK. The application will recognize that the address is filled in free format and will be viewed in such form.

    Address fillling by postal zip code

    • If the address zip code is known and the corresponding classifier for the region is exported, then using More command - Fill in by postal zip code you can find the address by zip code. Only addresses corresponding to the entered zip code are displayed to be selected. Select the required part of the address and filled in blank fields.

    Address clear

    • Using More menu - Clear address you can clear the wrong address.

    Classifier Import

    • Select More command - Import classifier. After address classifier for the required region is updated, it is possible to edit the address. When filling in some fields, the application will automatically offer to export the classifier.
