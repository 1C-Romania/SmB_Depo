Cashier workplace (CWP)

Cashier workplace is used for fast and convenient processing CR receipts, cash register return receipts, return receipts note and cash withdrawal receipts.

Cashier workplace includes two tabs: Sale and Return.

The Sale tab is used to issue CR receipts.

The Return tab is used to issue cash register return receipts, return receipts notes and cash withdrawal receipts.

Each tab has two areas. Working area is to the left, an additional bar is to the right.

You can hide the additional bar.

In the working area on the Sale tab there is the Inventory tabular section, corresponding command bar, total data of the register receipt and the Accept payment button.

The toggle changing the content of the bar items is on the Sale tab of the additional bar. The toggle is in the Logbook position by default and the cash documents log is on the additional bar: CR receipts, cash register return receipts and retail sale reports. If the toggle is in the Quick sale position, then the quick sale buttons are displayed. If the toggle is in the My cash register position, then the user can edit attributes of the receipt head like cash register, POS terminal etc.

The list of CR receipts and buttons for the return document creating based on the receipts is in the working area on the Return tab.

The field displaying the content of the CR receipt in the text form is located under the list of receipts.

The list of cash register return receipts is displayed on the Return tab of the additional bar.

You can change the cashier workplace buttons and key combinations corresponding to these buttons in the settings.

The cashier workplace settings are saved and edited in the Cashier workplace settings catalog. To open the catalog item go to the More menu (Logbook toggle), run the Settings command, and then click the Settings (Quick sales toggle).

Processing the sale

After cashier workplace form is opened, a cashier can fill in the tabular section with products and services, change number and discount in the lines.

You can fill the table in several ways:

  • Line-by-line - insert lines one by one pressing the plus button, running the Add command of the shortcut menu or Insert key on the keyboard.

  • By selection - you can open products and services selection window using the Select button.

  • By search - search box used to enter description, number or code of products and services is to the left over the tabular section. If products and services are found, then the line will be added to the tabular section.

  • Barcode scanning - barcode scanning using barcode scanner. If it is needed to enter barcode manually, click the corresponding button with barcode located over the tabular section.

  • Quick sale - using quick sale buttons. To configure and use quick sale, click Quick sale at the right bottom corner.

    If it is necessary to change the number in the line, you can do it using several ways:

  • Change the number directly in the line.

  • Using "+" and "-" buttons on vertical command bar located to the right of the tabular section line.

  • Using thebutton with calculator image on the vertical command bar located to the right of the tabular section. IN this case the number entry window is opened.

    When the product information is entered, click Accept payment or Ctrl+Enter.

    Payment method where it is necessary to enter the amount of payment in cash and/or using bank card (cards) is opened.

    If it is necessary to print sales receipt with the fiscal receipt, then select Sales receipt check box in the payment form.

    Then click Get receipt.

    The receipt will be printed by fiscal register.

    Then you can continue processing register receipt at the same window.

    If it is necessary to open previously created document or print corresponding sales receipt, you can do it using the additional bar on the Sale tab.

    Additional bar displays the cash documents logbook (Logbook button).

    Find required CR receipt.

    If it is necessary to open it, you can double-click.

    If you need to print the receipt, then highlight the document by clicking it, click More, and then select the Print sale receipt command.

    Processing the sale return

    To process the sale return from the customer, go to the Return tab.


    If the customer wants to return a product before end of shift and customer has fiscal receipt on hands, then you should do the following.

    Find the CR receipt you need by fiscal receipt number using search box.

    Check if it is the receipt you need by viewing the content of the document under CR receipt.

    Click Create return CR receipt.

    After cash register return receipt is created, the button becomes unavailable and corresponding hyperlink to open the Banking register return receipt created document is displayed at the left.


    If the customer wants to return a product after the end of shift and the customer has fiscal receipt, then you should do the following.

    Find the CR receipt you need by fiscal receipt number using search box.

    Check if it is the receipt you need by viewing the content of the document under CR receipt.

    Click 1. Create return receipt note.

    When the cash withdrawal receipt is created, the button becomes unavailable, and corresponding hyperlink to open created document is displayed at the left.

    Click 3. Create cash withdrawal receipt on receipt basis.

    When cash withdrawal receipt is created, the button becomes unavailable, and corresponding hyperlink to open created document is displayed at the left.


    If the customer wants to return a product after the end of shift and the customer doesn't have fiscal receipt, then you should do the following:

    Click 1. Create return receipt note.

    When the cash withdrawal receipt is created, the button becomes unavailable, and corresponding hyperlink to open created document is displayed at the left.

    Click 3. Create cash withdrawal receipt on receipt basis.

    When the cash withdrawal receipt is created, the button becomes unavailable, and corresponding hyperlink to open created document is displayed at the left.

    If it is necessary to open previously created cash register return receipt document, you can do it using additional bar.

    The Cash register return receipt document is displayed in additional bar.

    Find the CR receipt you need and double-click.
