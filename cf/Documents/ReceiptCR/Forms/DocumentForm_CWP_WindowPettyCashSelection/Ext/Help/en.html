Cash for cashier workplace (CWP) selection window

It is required to choose the cash register and POS terminal on the window.

When a cash register is selected, click Open cashier workplace or press the Ctrl+Enter.

The application stores the values that were selected by the user and populates them during the subsequent openings.

If a user always uses only one cash register and POS terminal, there is an option to select the Don't show this window any more check box on the form.

Later the user may clear the check box, change the cash register and POS terminal in the cashier workplace (My cash button).
