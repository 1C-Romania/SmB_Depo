Tax accrual document

The document Saves the information about the manual accrual and reimbursement of taxes.

 

Document transaction types:

  • Accrual.
  • Reimbursement.

Document attributes:

  • Company - company, for which the document is Creatingd. It is selected from the Companies list. The attribute is displayed on the form if Accounting by Multiple Companies is enabled in accounting options.

  • Note - any additional information.

  • Taxes. Table attributes:

    • Tax Type - for which the tax accrual (reimbursement) is done. It is selected from the Tax types list.

    • Amount - of accrual (reimbursement).

    • GL Expense Account - GL account to which tax accrual (reimbursement) is attributed.
