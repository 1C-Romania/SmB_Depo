Pick by documents form

You can use the form to pick up the inventory from one or several documents of the Supplier invoice or Expense report kind into the corresponding tabular section of the Additional costs document.

To fill out the tabular section, you need:

  • to place the documents from which the products and services will be selected for costs distribution into the Selected documents tabular section;
  • if you need to select only some part of the products and services for distribution of additional costs, in the Selection of products and services tabular section you need to specify the list of products and services that are available for transfer, and to select the Use filter by specified products and services check box;
  • the selected products and services will be added into the Selected products and services for document tabular section;
  • click OK to transfer the data from the Products and services selected for the document tabular section to the tabular section of the document.

If you select the Show inventory selected from current document check box, only the eventory selected from the Selected documents tabular field of the current document will be shown.

If you select the Add new items into table footer check box, new rows will be added to the Selected products and services for document tabular section footer when filling the data from the selected documents, even if there are duplicates. If you selected the check box, the data from the new row will be added to the row with the dupliactes. You can search for duplicate rows using the columns as follows: receipt document, products and services, characteristic, batch, measurement unit, VAT rate, price.
