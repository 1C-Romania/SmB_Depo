Fixed asset output document

The document registers property Depreciation in the database in cases when Depreciation is calculated proportionally to output of finished goods.

Document attributes:

  • Company - company, for which the document is Creatingd. It is selected from the Companies list. The attribute is displayed on the form if Accounting by Multiple Companies is enabled in accounting options.

  • Note - any additional information.

  • Property. Table fields:

    • Property - fixed asset being depreciated. It is selected from the Fixed assets list.

    • Quantity - of depreciated fixed assets. It is specified in the natural units.
