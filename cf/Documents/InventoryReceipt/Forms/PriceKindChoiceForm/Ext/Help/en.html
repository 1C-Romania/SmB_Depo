Setting prices in the document.

Additional dialog "Prices" for filling and changing the information about setting of payment procedure when pressing the button “Edit prices” in the document form.

When opening the dialog you see the set for this time parameter values:

  • Type of prices - it is selected from catalog "Type of prices". Prices of ProductsAndServices positions, added to document’s specification, will be set according to the given to them values of selected type ("Retail", "Wholesale", etc.). IN any moment it’s possible to choose the other price type for filling the prices to document’s specification. To change already filled prices it necessary to set the flag "Refill prices".
