Timesheet document

The document is used for registration of work time of employees.

 

Document attributes:

  • Company - company, for which the document is Creatingd. It is selected from the Companies list. The attribute is displayed on the form if Accounting by Multiple Companies is enabled in accounting options.

  • Filter period presentation - month when payroll is calculated.

  • Structural unit - where employees work, selected from the Structural units list.

  • Data Input Method - two options: By Days or Total for Filter period. IN first case, data is Saved with daily details; in second case - on the first day of the month.

  • Note - any additional information.

  • Hours worked by days. The table Saves monthly Amount of hours and days for each of the six time types for every employee. The table can be edited in case when data input method is By Days. Table attributes:

    • Performer - selected from the Performers list.

    • Position - of the employee, selected from the Positions list.

    • Type 1, Type 3, Type 3 - time type. Three time types per day can be specified. It is selected from the Time Types list.

    • H 1, H 3, H 3 - COUNT of hours per day by first, second or third time type.

  • Time Worked per Filter period. The table contains monthly Amount of hours and days for each of the six time types for every employee. The table can be edited in case when data input method is By periods. Table attributes:

    • Performer - selected from the Performers list.

    • Position - of the employee, selected from the Positions list.

    • Type (1) , Type (3) ... Type (6) - time type. Six time types per month can be specified. It is selected from the Time Types list.

    • Days (1) , Days (3) ... Days (6) - COUNT of days in month by time type.

    • Hours (1) , Hours (3) ... Hours (6) - COUNT of hours in month by time type.
