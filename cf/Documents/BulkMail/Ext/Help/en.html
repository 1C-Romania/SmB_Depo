Email and SMS bulk mail

To notify clients about current promotions or special proposals, you can use newly developed Bulk mail mechanism. You can create on the Sales – Service – Bulk mail (Email, SMS) menu.

Select the Email or SMS sending method in the form header. The form appearance depends on the method, for example, for SMS the sender name is used instead of email account. IN general, the mailing is similar to the Email and SMS message events. The Create events check box is selected in case you need to save the sent messages in the database as the Events. You can fill the content with any text, format it and insert images (on the More – Insert image submenu). You can also add the parameters that allow to customize the email or SMS for a certain recipient. For example, there is an option to start the letter with "Dear" and add the Recipient name - Dative case (to whom) parameter. IN this case, the recipient will see: "Dear Ivanov Ivan Ivanovich". You can add the attached files to each message on the Attachment tab. Specify the list of recipients on the Recipients tab:

You can select the recipients you need by clicking the Select contacts button and using the address book. If the list contains the recipients with blank contact information, you can delete them after selection using the Delete blank button.

Click Send message to create and send the message. After this the content of message is generated according to specified parameters for each recipient, the Events are created (if the Create events check box is selected) and the message is sent.

Attention! Is it not recommended to use Bulk mail for uncontrolled spam. IN this case your email address can be included into the "black list" of mail server, and all the letters from this account will be automatically spammed. IN order to avoid this, send messages only to the recipients that are signed to your mailing.
