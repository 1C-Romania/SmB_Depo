Configuration update setup

To Set the configuration update it is necessary to perform the following steps

  1. If  in the application work other users then it is recommended to choose a convenient time, when nobody works in the application in order to Set the configuration update.
  2. Quit all the users.
  3. Quit your own session.
  4. Run 1C:Enterprise in Designer mode.  
  5. In Designer in the menu Configuration select Update data base configuration (or press F7).
  6. Close Designer.
  7. Run 1C:Enterprise and continue to work in the application.
