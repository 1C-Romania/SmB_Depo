This report enables you to view information about using list items in application documents.

When users work in the application, they can duplicate items in error and it is necessary to get rid of them as it results in errors in accounting. When you use these duplicate items, there can be links to one item, then to another, as a result errors occur, for example, cash can be written off incorrectly in the accounting etc.

To delete duplicate items from the application, you need to delete all references to them in the documents. This report shows which documents of the application contain these objects.

To generate the report, in the application list, run command More - Usage locations.

The report displays the following:

  • Name, Code of selected list items
  • Usage locations - document, its number and date
  • Number of Occurrences.

Generating report

  • Select one or several items in the list, run More - Usage locations command, the report is generated automatically.
  • The report is grouped by selected list items. For each item application documents are displayed, which contain references to this item as well as number of occurrences; total number of occurrences for each document and total report amount are also calculated.
  • You can go to the corresponding list items or document, which has reference to these items, by double-clicking the corresponding report cell.
