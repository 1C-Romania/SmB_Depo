Shows a complete list of user rights.

You can open the report from a user card or from an external user card (users groups or external users groups) on the Access rights page by running the Access rights report command.

Report can be printed or saved on the computer in the format you need by using standard commands.

The following information is displayed in the report:

  • IB user properties:
    • Access to infobase, login, user name of operating system and other information.
  • User access restrictions:
    • Group name, responsible person, profile.
  • User roles by profiles
  • View objects:
    • Includes 3 columns table: Object/Rights/Restrictions, Access group/Profile/Roles, Restrictions on records level/Kinds and types of access value.
  • Edit objects:
    • Includes 3 columns table: Object/Rights/Restrictions, Access group/Profile/Roles, Restrictions on records level/Kinds and types of access value.
  • Settings rights to Folders.

Get detailed report

  • Select the Detailed information about access rights check box, click Generate.
