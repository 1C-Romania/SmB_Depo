It is intended for viewing reference, normative and other information supplied within the application.

For basic versions, additional information is always displayed in the beginning of application operation.

You can open it by running the Information at start command of the Information group on the Main page. When it is opened, one of additional information pages immediately appears.

The command bar has navigation buttons Back and Forward used to navigate through pages already viewed, as well as menu items corresponding to names of additional information sections. When you select a section on the command bar, a list of additional information pages related to the section is opened. When you select a menu item with a page name, the page is displayed.

Page text may contain links to other additional information pages (internal links), as well as links to sections of websites in the Internet (external links). To go to external links, your computer should have access to the Internet.

For PROF versions, the setting to display additional information in the beginning of application operation is available. If additional information is displayed in the desktop mode, then the Display at start check box is selected in the command bar. If the checkbox is cleared, additional information will not be displayed in the beginning of operation. The Display at start check box is cleared automatically twice a month and after updating the application version.
