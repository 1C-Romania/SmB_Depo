Full-text data search finds any information both in the card fields and the attached files.

Start full-text search right from your Desktop.

You can use such search options as:

  • Transliteration (to write Russian words with Latin letters according to the Russian standard GOST).
  • Replacement support (to write Russian words with English keyboard layout).
  • Fuzzy search (when some letters in the resulting words may be different).

     

    To start search in the application, enter a word or its first letters. Click Search. You can enter one or more words to search. Do not use search operator such as AND, OR, and NOT in the search line. You can insert a navigation link to the data in the search line. IN this case the required form will be opened without data search. To find a navigation link to any infobase object, use the command in the title of each 1C:Enterprise 8 window.

    Index update

    In the file infobase variant, the full-text index is not updated automatically and, therefore, requires periodical manual update. If a full-text search index is not relevant (i.e. some data is not yet indexed), you can use the Update index button in the file infobase variant.

    Examples

    Search example Result
    "Invoice 00016433" An "Invoice" document with number "00016433"
    "PO 120000" Payment orders for amounts of 120000 and 120000,50
    "Smith Purolator" All documents by Smith for the "Purolator" counterparty
    "Invoice Purolator" All "Invoice" documents for the "Purolator" counterparty
    "Purolator 07.05" All documents of the "Purolator" counterparty issued on July 05
    "Shoes" All entries of "shoes" product stated in order:
    • White shoes (Product)
    • Black shoes (Product)
    • Receipt of goods and services # 00086822
    • Sale of goods and services # 00016422

    Tips:

    • If too much data has been found, make your query more precise by adding additional conditions (date, amount, counterparty name)
    • If too little data has been found, simplify the query by excluding some of the words
    • For precise search use:
    • Quotes - to find an exact phrase, for example: "mini cooper".
    • OR - to find one of two words, for example "Purolator OR Fedex" - all occurrences of Purolator and Fedex companies will be found.
    • Brackets - to group expressions, e.g. search "Smith (Purolator OR Fedex)" will find all documents issued by Smith to Purolator and Fedex companies.
    • To include synonyms, use "!"; e.g. search "!number" will find all documents with the word "number" or "quantity".
    • To start "fuzzy" search, use "#"; e.g. "Fedex#3" will find all documents containing words which differ from word "Fedex" for one or two letters.
    • To search by the beginning of the word, use "*"; e.g. "Fed*" will find all documents, containing words that start from "Fed" - "Federal", "Fedex", etc.
