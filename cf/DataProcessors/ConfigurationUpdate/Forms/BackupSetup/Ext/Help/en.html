It is intended to set the backup of infobase when updating the application.

Before starting the application update it is recommended to create a backup copy of infobase. The backup copy is used for automatic recovery of infobase in case of abnormal situation.

You can open the form by the Create temporary backup IB and perform rollback in case of abnormal situation link on the Search and install updates window.

Select parameters to create a backup:

  • Do not create backup;
  • Create temporary backup. IN this case there is an option to select the Restore infobase check box, in order you can make rollback in case of abnormal situation while application is updating;
  • Create backup and save in specified directory. IN the directory field write a path to create backup, for example, C:\Backup.

After backup parameters input, click OK to return to the installation of updates.
