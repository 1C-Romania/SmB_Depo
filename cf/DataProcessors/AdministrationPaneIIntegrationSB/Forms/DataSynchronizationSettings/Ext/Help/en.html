This panel is used to configure data synchronization with other applications.

  • Data synchronization
    		•  Check box of data synchronization with other applications, for example, with 1C: Enterprise Accounting, version 3.0.
  • Data synchronization setup
    		•  Setup and performance of data synchronization with the other applications.
  • Warning when data synchronizing
    		•  Viewing of the unposted documents, unfilled attributes, conflicts and unaccepted by the prohibition date.
  • Importing prohibition dates
    		•  Configuring import prohibition for past period data from other applications. See also: Data change prohibition dates in Support and service.
  • This infobase prefix
    		•  Prefix is added automatically to the catalog item codes and document numbers to differ applications in which they were entered and to avoid data duplication.
  • Contract management
    		•  The option specifies the system operation at posting documents when the counterparty contract doesn't correspond to the document operation. The following options are available:
    • Do not post. Message of the contract non-compliance is displayed and the document is not posted.
    • Only warn. Message of the contract non-compliance is displayed but the document is posted.

    Counterparty contracts are not monitored if data synchronization with the Enterprise Accounting is not configured.

  • Use exchange with sites
    		•  This option enables you to configure the exchange with the online shops.
  • Setting up exchange with websites
    		• Setting up and execution of data synchronization with sites (online shops).
  • Prefix to exchange with the website
    		•  Prefix is added automatically to customer order numbers exported from the website to differ them from the orders created in the infobase and to avoid data duplication.
