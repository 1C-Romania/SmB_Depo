Print labels and price tags

The processing is designed to print the labels and price tags for the goods. Tags and labels are printed according to the patterns that can be added and edited.

Creating (editing) of labels and tags patterns is performed using the button Label And Tag Templates.

 

To print the labels and tags using the command Show the filter it is possible to filer the goods in terms of different criteria (by the items positions, items group, stock status etc). Using the command Show the filling mode there is displayed the Information about the modes of the filling the the tags and labels (in case of the selection of the printing mode Print tags and labels).

Printing is performed by the prices types which are necessary to specify in the corresponding field.

Using the command Install for the selected lines it is possible to specift for the selected lines the COUNT to be printed and the labels and tags template.

The printing of the tags and labels can be performed using the corresponding command from the Supplier invoice document .

The mark of the selection for printing can be set only for that items for which is defined at least one template for the tag or label printing and set the COUNT to be printed.
