The interactive synchronization assistant is designed for data synchronization with optional preview and mapping of exported data. Use the assistant to map two synchronized infobase objects. After the objects are mapped, data is imported and then exported.

Use the interactive data synchronization mode if before synchronization one object could have been added to both infobases independently. IN other words, use the interactive data synchronization mode for the first synchronization of two infobases already having entries. Such infobases are likely to contain similar objects. IN this case, the interactive mode allows you to manually map such objects.

Open the assistant with the  Synchronize data with defining advanced parameters button in the data synchronization list.

Connection parameters

  • On the first assistant page, select the exchange message transport type (you can change the existing connection variant indicated on the page), by clicking  Configure. If necessary, go to the information exchange directory and prove that the required files exist, or check the exchange message transport parameters.
  • Click Next to go to the next page.

Mapping data

  • The second assistant page shows the object mapping statistics. Before opening the page, the application tries to match the data automatically.
  • Automatic mapping covers the default mapping fields with strict mapping rules. It is recommended to check automatic mapping results. The list contains:
    • Data, total received, pcs.
    •  - some remarks on object mapping
    • In this application, pcs.
    • Not mapped, pcs.
    • Mapping, %
  • Use the Map command to preview the imported data and map objects manually. Mapped manually, the object of the two infobases are saved to be mapped automatically next time. For the interactive import mode, it is recommended to map the objects manually first and then map the remaining unmapped objects manually.
  • The list is not dynamic, so you need to click  Update to view the new data received and mapped  after you opened the list.
  • Use the More - Receive data  command to import data from the selected table rows. Use the data import rarely, only when you need to map related data tables.
  • The stated commands are available in the context menu of the mapping statistics table. Executed from the context menu, these commands are functional for multiple selected table rows.
  • After mapping, the rules are saved and the page is not displayed anymore.
  • Click Next to start data synchronization.

Exporting data

  • On the third page of the assistant, state the data to send to the other application. If you do not specify data, the data modified since the previous synchronization session will be sent by default. 
  • By default, the Do not add variant is selected - data will be sent according to the common settings.
  • To export documents additionally, switch to Add documents.
    • In the Period:  field, Select the dates of the data you want to send.
    • By default, the Filter: field is filled in with All documents (link). Click the link to define the filter criteria: enter the field to filter, define Comparison type, select the Value to compare. There may be multiple criteria; in this case, they are grouped with logical AND, OR, and NOT.
  • To add additional data, switch to Add data.
    • Pick up the data: click the Additional data is not selected link (by default). Pick up export data from the relevant lists and click OK. The link displays the enumeration of the selected data.
  • View the export data statistics using the Export data content report link.
  • Click Settings to use predefined or save new settings.
  • Click Next to begin data synchronization with the selected parameters.

Data synchronization complete

  • The final page of the assistant indicates the synchronization state:
    • Failed to receive data or Data has been received successfully, etc.
    • Failed to send data or Data has been sent successfully, etc.
    • Use the More link to view the synchronization procedure log  and error reports.
  • If necessary, you can set data synchronization schedule in automatic mode (if the current synchronization is not included in any of the existing synchronization schedules). ClickConfigure.
  • To finish your work with the assistant, click Done.
