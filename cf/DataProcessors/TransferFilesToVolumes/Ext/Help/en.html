Designed to transfer files stored in an infobase to file storage volumes. 

To transfer files:

  • In the application settings, the Administration - File operation settings settings, select the Store files in volumes on drive check box.
  • In the file storage volumes, add a necessary number of file storage volumes of the infobase.

You can monitor the following information:

  • Number of the files in the infobase
  • Size of the files in the infobase (MB)

On the basis of this data, you can calculate the necessary number of file storage volumes.
