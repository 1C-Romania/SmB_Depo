Employee calendar

To manage the EVENT on interaction with the counterparties use the command Performer calendar which is caled by the same-name link on the actions panel with the following sections: Marketing and sales, Supply And Purchases, Works, toolss, tools.

Using Performer calendar the user can analyze the EVENT and work tasks planned for the current period (day, week, month) as well as plan the new ones. Analysis of the planned EVENT and tasks is performed on the tab My agenda.

To plan the EVENT and tasks (not only for yourself  but also for the employees) use Schedule.

To plan a new event in the schedule form "Day" it is necesary by double clicking on the schedule scale and with a righ mouse button call the form of the document Event" or Job order".

Events can be viewed also in the Month presentation. For this please push the Month button.
