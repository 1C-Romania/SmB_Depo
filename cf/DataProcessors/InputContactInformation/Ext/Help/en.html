After all required kinds of contact information used for the organization or company are set, and the address classifier is dowloaded, you can start entering contact information to the lists and application documents.

You can enter contact data:

  • in a form of arbitrary string
  • using templates

Use of templates allows to unify data entry, improves its perception when viewing and search in the lists.

If you use the address template, the address fields can be filled in using the address classifier. Use of templates and address classifier significantly accelerates data entry, reduces errors in the process of data entry, and therefore facilitates reporting to different organizations. Moreover, the address classifier is always updated, therefore, staff addresses are always relevant. The application provides the possibility to enter some addresses with errors in free form as an exeption when using address templates.

Use of template when filling in phone and fax numbers facilitates data entry, reduces errors during entering.

When entering contact data both predefined fields and the fields added to Kinds of contact data list while working in the application are available.

If the Allow entering several values is selected at setting one or several kinds of contact information, then Add button is provided in the entry window. When you click it, the list of possible kinds of contact information to be added is displayed. For example, the company may have several email addresses, web pages, etc.

To work with contact information, you can use commands of the shortcut menu by clicking right button of the mouse:

  • Select (F4) - enter contact information using the template, similar to Select button . Address, phone or fax numbers can be entered using corresponding template
  • Open (Ctrl+Shift+F4) - view of entered contact information (for example, too long address can be displayed in full using this command)
  • Cut (Ctrl+X)
  • Copy (Ctrl+C)
  • Insert (Ctrl+V)
  • Clear (Shift+F4) - fields for address and phone number entry using the template can also be cleared. IN the Country field of the address, "Russia" is entered automatically
  • Select all (Ctrl+A)
  • Add comment - using this command you ca enter or edit comment without opening the field editing window. Notes can be added for all kinds of contact information, except Other text field, because this field is used to enter all necessary data. Use the command, then enter text of the comment, click OK and then continue work in the application. If comment field is not empty, it is displayed under contact data field.
