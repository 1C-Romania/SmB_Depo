This form is used to input phone (fax) numbers by template.

You can access it from tabular section of a document or list item containing contact information, by clicking  Select.

To input a phone or fax number, fill in the following fields:

  • Country code - "+" character will be appended automatically;
  • Area code - this code will be bracketed automatically;
  • Phone number - no format restrictions;
  • Extension - "ext" will be added automatically before the extension digits;
  • Comment - enter additional information here, if any.

Clearing phone (fax) number

To delete a wrong phone (fax) number and input the correct number, execute the All actions (or More, in Taxi interface) - Clear phone number command. To access this command from Taxi interface, click More.
