This form is used to input a settlement address by detailed template.

To access this form, go to the City field in the address input template and click  Select.

In the State field, fill in the name of the U.S. state where the settlement is located. When filling in the address classifier is not available for the selected state and you possess sufficient rights to load the classifier, you will be offered to do so. If the classifier is loaded, you can select a partial address from list by clicking  Select.

The County, City, Settlement fields are used to store the related address blocks. The list of mandatory fields varies depending on the address specifics. For example, some addresses do not include county, city, or settlement name. If an obsolete name is selected, a warning message will be displayed. Click Yes to confirm and continue with the address input. To remove the obsolete addressed from the lists, select the Hide obsolete addresses check box located in the settings for this contact information kind.

When finished typing in a city or settlement name, click OK to proceed with the address input.

If the address classifier is loaded for an address block, the application will offer you suggestions for the current address block as you type, based on the fields you filled in earlier.
