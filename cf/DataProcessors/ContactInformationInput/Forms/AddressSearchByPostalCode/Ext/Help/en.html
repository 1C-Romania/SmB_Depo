This form is used to search for addresses with known postal code.

Search by postal code cannot be performed until the address classifier for the corresponding state is loaded.

To access this form, fill in the postal code in the address input window and execute the All actions (or More, in Taxi interface) - Fill in by postal code command.

Partial address corresponding to the postal code will be displayed at the top of the window. You will have to fill in the missing address information manually. You can choose between city streets and streets of smaller settlements in the state. Select a section of the address and click Select, or double-click the section. After selecting the street, fill in the remaining address data in the new window.
