This form is used to input contact information according to a template. Address classifiers can be utilized to further streamline the procedure.

  • Country - select the address country in this field. Addresses outside of the U.S. are edited in a common field, as they are not broken into standard subfields like "Street" or "Apartment". "USA" is the default country. This field can be disabled via settings. To select a different country, click  Select. You will be able to select a country from the Classification of countries of the world (OKSM) list.
  • Postal code - type in a postal code corresponding to the address. If the address can be found in the address classifier, postal code will be filled in automatically. You can also fill in an address based on the postal code, provided that the state address classifier is loaded and available.
  • City - enter the city name here. To enter detailed information, click  Select and type in the detailed city information into the template form that will be displayed.
  • Street - select (click  Select and pick a street from list) or type in a street name.
  • Building, Unit, Apartment - enter the related address information here. You can change these fields by selecting additional values from dropdown list.
  • You can also  Add additional fields to store other predefined address blocks, such as Room or Office. 
  • Address presentation - in case of address input by template, this field is disabled. It will be filled automatically as you fill in address fields.
  • Comment - you can type in any comments to the address. These comments are never included in the address presentation or printed.

Address input examples

  • To enter "1515 Connecticut Ave, Office 202, Washington DC, 20008, USA":
    • City - select Washington, DC.
    • Street - select Connecticut Ave.
    • Building - type in 1515.
    • Apartment - first, you need to change the name of this field to Office. Select Office from dropdown list and then type in 202.
  • To enter an address with a different set of fields, for example, "45 East 52nd Street, Unit 2, Room 111, New York, NY, 10022, USA":
    • City - select New York.
    • Street - select 52nd Street.
    • Building - type in 45 East.
    • Click  Add, select Unit from the dropdown list, and type in 2.

Address validation

Click Check filling to validate the address you have entered. The following criteria are validated:

  • address completeness;
  • absence of prohibited characters;
  • length of address attributes;
  • compliance with address classifier. In client/server mode Infobase, compliance of addresses with the address classifier is validated on the server side. When address validation is complete, the "Address is correct" message or list of errors will be displayed. Validation is not performed for addresses outside of the U.S. Validation is also not performed for free-format addresses.

Entering free-format addresses

  • In certain cases you may need to knowingly enter an invalid address (for example, an address on an incoming recepit is invalid and cannot be used, but you still have to enter it). For this, execute the All actions (or More, in Taxi interface) - Free-format address input command. The invalid address will be further used for storage, so this option should only be used as a last resort.
  • Once you execute the command, the "Free-format addresses are not validated by address classifier" message will be displayed. Click Yes to confirm, or click No to go back to the template-based address input form.
  • You will be offered a standard address input template where all fields, except Address presentation and Comment, are disabled. Enter the address into the Address presentation field. As soon as you start filling in the Comment field or click OK, the address you have entered will be automatically parsed and some of the disabled template fields will be filled in. The standard address validation procedure will not be performed.
  • Click OK to confirm the address. It wil be saved as a free-format address, to be viewed without any changes.

Address input by postal code

If you know postal code for an address and have the address clasisifier for the related state loaded, you can find the address by using this postal code. For this, execute the All actions (or More, in Taxi interface) - Fill in by postal code command. This will display a list of addresses related to the postal code you have specified. Select the required address from list and fill in the remaning fields manually.

Clearing addresses

To delete a wrong address, execute the All actions (or More, in Taxi interface) - Clear address command.

Loading classifiers

To load an address classifier, execute the All actions (or More, in Taxi interface) - Load classifier command. Once the address classifier for the selected state has been updated, you can continue editing the address. When filling in certain fields, you will be automatically offered to load the classifier.
