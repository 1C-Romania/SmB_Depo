As soon as you have completed setting up the contact information kinds used by the company or organization, and loaded the address classifier, you are free to proceed with entering contact information into the application lists or documents.

You can enter contact information:

  • as free-format text lines,
  • by filling in template fields.

Be advised that data input via templates ensures data uniformity, resulting in easier readability and navigation.

When entering contact information into address templates, you can use address classifier to fill in the address fields. Using templates and address classifier expedites data input and reduces the input error rate, resulting in higher quality of reporting documents. Moreover, the address classifier is updated on a regular basis, which ensures that addresses of your employees stay valid at any time. In certain cases you may need to knowingly enter an invalid address. This cannot be done with templates; to enter an invalid address, you need to switch to free-format address input.

Using templates to enter phone and fax numbers also expedites data input and reduces the input error rate.

When entering contact information, you can have access both to the predefined fields and to the fields maunally added to the Contact information kinds list.

If the Allow multiple value input check box is selected for a contact information kind in its settings, the Add button will be available in the input window for this contact information kind. Clicking this button will call up a list of contact information kinds you can add to the template. For example, you may need to add multiple fields for email addresses, webpages, etc, for a specific contact.

When editing contact information, you can use  the following commands available via context menu accessed by right-clicking:

  • Select (or F4 keyboard shortcut) - enter contact information by template, identical to the Select button. Templates can be used to enter addresses, phone or fax numbers.
  • Open (or Ctrl+Shift+F4 keyboard shortcut) - review the contact information entered previously (for example, if an address is too long and cannot be fully displayed in preview, use this command to read the full address);
  • Cut (or Ctrl+X keyboard shortcut);
  • Copy (or Ctrl+C keyboard shortcut);
  • Paste (or Ctrl+V keyboard shortcut);
  • Clear (or Shift+F4 keyboard shortcut) - this will also clear fields used for address or phone input by template. The Address country field takes on the USA value automatically.
  • Select all (or Ctrl-A keyboard shortcut).
  • Enter comment - use this command to enter or edit comments without having to open the field editing window. Comments are available for all contact information kinds except the Other text field, because this field itself is used to store comments. Execute the command, type in the comment, and click OK to proceed. If the comment field is not blank, the comment will be displayed above the contact information field.
