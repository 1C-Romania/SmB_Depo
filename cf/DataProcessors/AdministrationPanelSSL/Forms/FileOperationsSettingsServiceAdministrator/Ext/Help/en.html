This panel is used for setting the import parameters, files storage and processing.

  • Save files in volumes on disk
    		•  Selection between files storage in database or in volumes on a drive.
  • Files storage volumes
    		•  Connection settings to files storage volumes (folders) on file server.
  • Prohibit files import with extensions
    		•  Select the check box to disable importing of files into the application with specified extentions, and specify prohibited for importing files extentions in the input field with space between extensions.
  • File maximum size
    		•  Max allowable files size, imported into the application. Files exceeding the specified size won't be imported into the application.
  • Files extensions ODF
    		•  Open documents format files extensions for office applications. File extensions are specified with space between them.
  • Text files extensions
    		•  Files extensions containing text data. Extensions are specified with space between them.
  • Electronic print full names
    		•  Allows using electronic print full names and checks if a file has been modififed since the electronic print full name generation.
  • Cryptography settings
    		•  Cryptography means connection parameters: electronic print full name and encryption.
