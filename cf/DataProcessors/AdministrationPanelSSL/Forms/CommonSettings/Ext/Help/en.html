Use this panel to configure main parameters of the application.

  • Application title
    		•  Text displayed in the application title.
  • Time zone
    		•  Reference time zone of the application. All dates of the documents and application registers are stated in the same time zone.
  • Current session time
    		•  This command allows you to see how the client's time differs from the server time corrected for the time zone.

  • Object versioning
    		•  Check box of revision history (versioning) of catalogs and documents.
  • Versioned objects 
    		•  Versioning variant settings for catalogs and documents.

  • Additional attributes and data
    		•

    Check box allowing additional properties for catalogs and documents.

     

     

  • Sets of additional attributes and data
    		•  Selection of additional attributes and settings of catalog/document data.

     

    		 .

  • Contact information kinds
    		•  Adding and configuring contact information kinds of different system objects, e.g. partners, counterparties, employees, etc.
  • Internet access parameters
    		• Settings of proxy-server parameters on the 1C:Enterprise server.
  • Infobase administration parameters
    		•  Configuring access to the server cluster to perform administrative actions with a client-server infobase.
