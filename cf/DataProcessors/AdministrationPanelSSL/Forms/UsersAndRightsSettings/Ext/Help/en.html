This panel is used to administer users and set access rights.

  • Users
    		•  Administration of the users working with the application.
  • User groups
    		•  Check box for accounting users in terms of groups. One user may be a member of multiple groups. Access rights set to a group are applied to all its members.
  • Access groups
    		•  Setup of access rights and restrictions for the user list and user groups.
  • Restrict access at entry level
    		•  Check box of flexible setup of access rights of the groups to catalogs, documents and other application data.
  • Access group profiles
    		•  Setup of predefined templates of user access groups. The access group profiles consist of the roles defined in the application metadata.

  • Settings copy
    		•  Copying settings of the application operating context between two users. It may be useful when you add a new user.
  • Settings reset
    		•  Settings reset for a user or a user group. It restores the integrity of the settings and sometimes helps to stabilize sessions of a user.
  • User settings
    		•  View and control of the user settings.

  • Allow access to external users
    		•  Check box for using external users in the application. When it is selected, you can configure access of external users and use them in the application.
  • External users
    		•  Account of the users working with the application from outside, e.g. Partners. External users are administered just like standard application users.

  • Internet address of infobase publication
    		•  The Internet address of the infobase. Use it to create navigation links to infobase objects in your email messages.
