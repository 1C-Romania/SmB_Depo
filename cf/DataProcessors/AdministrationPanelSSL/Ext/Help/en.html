The Administration section contains tools using which you can change operation parameters, perform scheduled jobs and monitor the application state. To provide effective operation, configure them properly.

If necessary, you can show the frequently used settings in the command interface of the Administration section in the settings of the navigation panel or the action panel according to the setting type.

See also:

  • Support and service -  keep the application functioning (for administrator only).
  • General settings - configure basic settings of the application.
  • User and rights settings - manage access and configure user rights.
  • Organizer - configure email, notes, reminders and business processes.
  • Data synchronization settings - configure data synchronization with other applications and organize their cooperation in a certain infobase.
  • File operation settings - configure the parameters of file import, storage and processing.
  • Print forms, reports, and data processors - configure print forms, report variants, additional reports, and data processors.
