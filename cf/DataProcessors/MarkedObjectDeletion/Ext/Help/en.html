Designed to delete selected objects permanently.

While working with the application, you can mark objects for deletion (e.g. when they were entered by mistake). The user having administrative rights can permanently delete objects marked for deletion.

Click Delete marked objects in the Administration - Support and service section.

Deletion does not affect link integrity, i.e. if a application object (a document, list item, etc.) is referred to by other objects, the object will not be deleted. Remember that deletion of marked objects is an irreversible operation.

Delete previously marked objects in one of the two ways described below:

  • Delete all objects automatically - try to automatically delete all the objects marked for deletions.
  • Delete objects selectively - preliminarily select the objects before you delete them.

Automatic deletion of all marked objects

  • Click Delete.
  • In this mode, all marked objects will be found and deleted. After a while, a number of deleted objects is displayed. User intervention is only required if there are links to some objects and therefore they can not be deleted.

Selective deletion of objects

  • Click Next.
  • In this mode, a list of the objects marked for deletion is created.
    • In the list, objects are grouped by type; by default, all objects are marked for deletion.
    • Clear the check boxes in the list to keep the objects that were marked for deletion.
    • Double-click the object or use the  Edit button to open the object and decide whether to delete it.
  • For permanent deletion, click Delete.
  • After permanent deletion, you see the message showing how many objects were deleted.
  • If there are links to the objects, you see a warning and a list of the objects that can not be deleted and the objects preventing from deletion.

Actions when object can not be deleted

The list of the objects that were not deleted consists of two parts: on the left - a list of the objects, on the right - the Usage places list. 

  • To delete objects permanently, firstly:
    • Double-click in the list or click More - Edit to open the object preventing from deletion, e.g. to manually delete links to the target object from such an object.
    • Specify a link to another list item. Click Replace in all usage places to go to replacing items. To replace links, select some other list item instead of the one to delete.
    •  Mark for deletion all the objects linked with the item to delete (in the left part of the list - Usage places).
    •  Unmark for deletion an object marked for deletion.
  • Then click Redelete to try to delete the objects again.
  • Click Home to select another deletion method.
  • Click Close to return to the application without further deletion attempts.
