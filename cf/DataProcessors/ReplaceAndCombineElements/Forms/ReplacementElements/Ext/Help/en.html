It allows replacing one or several items to the selected item.

Such option is useful for item deletion, for example, to delete the item completely it is necessary to delete all references to it.

It is opened using Replace and delete button in the application lists. It is also possible to run the Replace for all use points command of the Delete marked objects window.

You can automatically replace all references to removed items by references to selected item. All selected items will be marked for deletion, or deleted directly if the user has administrator rights.

All items of the corresponding list including items marked for deletion are displayed.

Viewing the list item

  • Run the Change command of the shortcut menu by clicking right mouse button or using More menu to view the selected item.

Work with documents

  • A selection based on the period is provided for the documents. If there are many documents in the list, it is possible to apply this selection. To do it run the Set period command of the shortcut menu by clicking right mouse button or using More menu.
  • Moreover, the More - Post (cancel posting) commands are provided for the documents .

Items replacement

  • Select the item which is to be replaced by the selected items.
  • Attributes of the item selected to be replaced are indicated at the bottom of the list in information line. Besides, users with administrative rights are provided an option to select. Click on the link and specify that replaceable items will be:
    • marked to be deleted
    • deleted permanently
  • Click Replace to replace all references.

In case of errors occurred during items processing, the list of unprocessed items is displayed. Abbreviation of the reasons of failure to process the selected item are displayed in the bottom of the list. IN the list it is possible to:

  • Open item to be viewed by clicking Change, or double-click
  • Try to replace the element again clicking Repeat replacement
  • Click Back, and then select another item for the replacement
  • Click Close to return to the application
