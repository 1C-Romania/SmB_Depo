Exchange rates are imported from the RBC website.

The exchange rates import window consists of two parts:

  • Filter period:
    • From - the date of beginning of the exchange rates importing period
    • To - the date of ending of the exchange rates importing period
  • List of currencies - filled by default from the list of Currencies only with currencies for which the Upoaded from Internet check box is selected. The list displays the following items:
    • Import - select the check box to import exchange rates for the specified period
    • Character code - established by the international standard and used as the main presentation of the currency
    • Currency - the full name of the currency
    • Exchange rate date - the date of the imported exchange rate
    • Exchange rate - the exchange rate displayed as on the specified date 
    • Multiplicity - the number of quotation items; it is not displayed if it is equal to 1
  • you can also add additional fields into the list using the More - Change form command:
  • Digital code - a unique numeric code established by the international standard; it serves for identification of the currency when rates are imported from Internet

    Importing selected exchange rates

    • Specify the beginning and ending of the period for importing exchange rates in the 'From' and 'To' fields
    • Select desired currencies using the 'Import' check box or all currencies using the  button
    • You can clear all selected currencies using the button
    • Click Import
