It is intended to view the list of additional delayed data handlers which run after version update in a background mode.

You can open the list by running the Update results and additional handlers command in the Administration - Support and service section, and then by clicking the relevant link in the Application update results window.

The list includes:

  • Data handler - the name of data additional processing procedure and its detailed description.
  • Status - brief information on the results of data processing.

Select data handler to view the detailed procedure progress at the bottom of the list.

Filtering

  • select the status you need in the Status field ("All handlers", "Error", "Running", "Completed");
  • type a word or its part into the Search field.

Running the data processing additional procedures

  • To run data processing additional procedures, click Start. The button is visible if data processing additional procedures are in progress or there are some non-completed procedures in the list.

Viewing the event log

  • To view the progress details of data handlers in the event log after their completion, click the Technical information on data processing results link.
