Designed to delete user settings.

Sometimes you might need to delete user settings, e.g. when incorrect settings prevent the application from operating normally. You can clear all settings including personal report variants.

Find the command clearing user settings in the Administration  - User and rights settings section.

Clearing user settings

  • Use the Clear settings radio button to select the users whose settings are to be cleared:
    • For selected users - click the Select link.
      • Select the Users list (External users).
      • In the User selection list, select the relevant check boxes and click Select.
      • After you selected the necessary check boxes, the link looks like: the user name (if you selected one user) or the number of the selected users (if you selected multiple users).
    • For all users - here you can clear All settings only.
  • Use the Settings to clear radio button to select the settings to delete:
    • All settings - if you select multiple or all users, you can only clear all settings of each user.
    • Specific settings - if you select only one user in the Clear settings field, you can select specific settings. 
      • Click the Select link.
      • In the User settings selection window, select check boxes for the settings to clear.
      • After that, the link looks like a name of one user setting or a number of the selected settings.
  • To start clearing with defined parameters, click Clear and close to return to the application or Clear to go on clearing.
