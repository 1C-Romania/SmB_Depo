Designed for a user to copy the settings of another user. You can copy settings within one list - Users or External users.

You might want to use it if you configure multiple new users in one group that already contains users.

Find the command that copies user settings in the Administration - User and rights settings section.

Copying user settings

  • In the Copy settings from user field, select the user (or external user) whose settings will be used by another user (external user):
    • Click  Select and specify the Users (External users) list where settings will be copied. 
    • In the User selection (External selection) list, select the necessary user (external user) and then click Select. You can also double-click the necessary user.
  • In the Copy to field, select the users for which you will copy the settings. You can switch to:
    • Selected users - you can select users. Click the Select link.
      • In the Select users for settings copying window, select check boxes for necessary users (external users).
      • When selection is complete, the link looks like a name of one selected user or a number of selected users.
    • All users - once you select the value, you can go to the next step.
  • In the Settings to copy field, select the settings to copy to another user or other users:
    • All settings - after that you can go to copying settings.
    • If you select Certain settings, you can select settings from a list. Click the Select link.
      • In the User settings selection window, select check boxes for the settings to copy.
      • When selection is complete, the link looks like a name of one user setting or a number of the selected settings.
  • To go to copying with the selected parameters, click Copy and close to return to the application, or click Copy to go on copying.

Limitations

  • Even when you click All settings, some user settings are not copied.
  • The users to which the appearance settings are copied must log off. Otherwise, the settings may not be applied.
  • If you copy report variant settings, users must have rights to the relevant reports.
  • Personal report variants can not be copied to other users.

In such cases, you see a warning informing that some settings are not copied. Click the Show report button to view the skipped settings.
