Designed to manage user settings.

Use the list to view, clear and copy the user settings of reports, application visual appearance and other parameters. 

Open the settings list with the User settings command in the Administration - User and rights settings.

  • In this case, the list includes the User field. By default, the User is Administrator. Anyway, this field can not be empty. If the application has external users, their settings can be managed, too.

The settings list of each user (external user) can be opened with the Settings command of the navigation panel.

  • In this case, there is no User field.

To find necessary settings, you can enter the name or its part into the Search field.

The list includes three tabs:

  • Appearance - appearance settings of the desktop, forms (e.g. application lists), and the command interface.
  • Report settings - a list of report user settings grouped by report variant names.
  • Other settings - personal user settings, table document print settings, quick access settings for reports and data processors, the Favorites section settings, etc.

Copying settings

In the User field, select the necessary user. You can:

  • Copy to other users - copy the settings of the selected user to one or more application users.
    • Select the necessary user.
    • Select the necessary copying settings in the list tabs by holding down Ctrl and Shift and clicking the required settings.
    • Select check boxes for necessary users in the User selection list, and then click Select. The settings will be copied.
  • More - Copy to other users to settings group - the command is similar to the previous one, but you can also select the following settings groups in the command menu:
    • Report settings
    • Appearance settings
    • Report and appearance settings
    • Report, appearance and other settings - to copy all the user settings.
  • Copy from - use this command to copy settings to the selected user from another user:
    • In the list, select the user from which you want to copy settings.
    • Select the settings to copy. The action is similar to the Copy settings command in the Administration - User and rights settings section.

Limitations

Even when you click All settings, some user settings are not copied.

  • Users for which appearance settings are copied must log off. Otherwise, the settings may not be applied. To find out if the users are logged on right now, you can click the Active users button in the User selection list.
  • When copying report variant settings, the users must have rights to these reports.
  • Personal report variants can not be copied to other users.

In such cases, you see a warning informing that some settings are not copied. Click the Show report button to view the skipped settings.

Clearing selected settings of selected user

In the User field, select the necessary user. You can:

  • Clear - clear the selected settings of the selected user (click the target settings and hold down Shift or Ctrl).
  • Clear all - select the settings groups of the selected user:
    • All users
    • Report and appearance settings

Clearing all settings of selected users

  • Hold down Shift or Ctrl and click the required settings.
  • Click More - Clear all - Selected settings of users.
  • To confirm your selection, click Yes.
  • Select check boxes for necessary users in the User selection list.
  • Click Select button to reset the settings.

Clearing all settings of all users

  • Click More - Clear all - All settings of all users.
  • To confirm your selection, click Clear all.

Viewing current user settings

  • Only the current user can view settings. IN this case, the user can view the application reports and appearance with the selected properties.
  • Select the required setting.
  • Click More - Open.
