This panel is used to control personal settings and to manage request display when closing the application.

  • User info
    		•  View information of the current authorized user. Moreover, you can change your own password to enter the application in the opening form.
  • Full-text search in the data
    		•

    Transition to the fulltext search form. Functions are enabled in the Service and Support section.

  • Set extension for file operations
    		•  It specifies the file work extension. It informs that the extension is selected in the start mode of thick and thin clients.
  • Equipment connection and setup
    		•  The form is used to connect and configure external equipment specified in the External Equipment catalog.
  • Configure access to the proxy server
    		•  Additional proxy server settings.
  • Scanned files numbers
    		•

    Proceeding to the information register which contains the scanned files numbers and data about the owners.

  • Update system parameters
    		•  It updates the command interface, desktop and opened forms taking into account current values of functional options and the parameters. It is necessary to call the method to activate changed parameters of functional options.
  • Confirmation when exiting application
    		•  Request management when application closing.
