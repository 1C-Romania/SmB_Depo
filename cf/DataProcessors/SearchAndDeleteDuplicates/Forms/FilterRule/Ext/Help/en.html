It enables you to filter list items.

It is opened from the Search and delete duplicates window by clicking a reference in the Filter field.

Entering filter criteria

  • Click Add new item.
  • Select a Field from the list to filter by.
  • In the Comparison kind column, select a comparison condition. Depending on the selected list, you can access standard comparison kinds (Equal to, Not equal to, More, Less, etc.) as well as more complicated ones:
    • You can specify a list of possible filter values using In a list comparison kind. Not in a list comparison kind is a negation of the previous.
    • You can use filter by part of the field (combination of letters) using Contains or Not Contain operations.
    • You can determine if there is a value in the field using Filled or Empty operations.
  • Set Value. To do so, depending on the selected field and comparison kind Select a value from the list, enter a value or create your values list (for IN the list and Not in the list operations).
  • To delete the filter, click Delete.
  • To confirm and return to the application, click OK.

Grouping conditions

To refine the filter, you can apply multiple conditions. They can be reordered by dragging or clicking and .

  • If ungrouped multiple conditions are set in the rule, AND logical operation defaults to these conditions.
  • You can apply And, Or, Not logical operations to selected conditions. To do so, Group types are applied in the application: And group, Or group, Not group.
  • To use conditions grouping, select one or several conditions, and click Group conditions. Conditions are included by default to And Group.
  • You can change grouping operation if needed. To do so, open the Group type cell selection by double-clicking Group. Select another Group type. You can also change group type manually.
  • Drag other conditions to the group or create new conditions within the group.
  • To cancel grouping, click More - Ungroup.
