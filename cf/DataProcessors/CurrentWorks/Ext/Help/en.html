It enables you to output to-do list of the user to the desktop.

The application allows you to show a to-do list of the user on the desktop: new emails, tasks, claims, non-approved orders, etc. To-do list may include important and ordinary works, single-line and multi-line works, with and without quantitative indicators. To-do list is grouped by sections command interface, you can set visible and order of display, automatic update according to specified interval. Important works are highlighted.

Works appear as links, and by clicking them you can open lists, documents or other application objects.

Using Set button in the lower part of current to-do list, you can control to-do list and sections list.

Displayed to-dos

  • You can control the visible of to-do list on the tab of the current works settings using the check boxes.
  • Using button You can make all sections and works visible by selecting all check boxes.
  • Using button You can hide all sections and works by clearing all check boxes.
  • Using buttons and you can change the sequence of works and sections in the list.

Updating to-do list

  • You can adjust the update frequency of current works on the Other tab.
  • Select the Update automatically every 5 minutes check box (by default). Specify other number of minutes if needed.
  • You can also Update to-do list at any time by using the button .

Excluding information duplicates

  • If any of works duplicates the information displayed on the desktop, you should select one of the application functions:
    • Disable to-do from the Current to-do list

or

    • Hide information from desktop.
