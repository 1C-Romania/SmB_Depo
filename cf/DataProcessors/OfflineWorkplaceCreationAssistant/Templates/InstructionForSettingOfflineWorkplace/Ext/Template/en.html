Setting of an offline workplace on the computer

Attention: To work offline without connecting to the Internet it is necessary to purchase and locally Set the [ApplicationName] application on the computer. For details please see the information below:

  1. Create a new folder which will contain the info base of an offline workplace. E.g., in the folder "My documents".
  2. Open the exported file Offline work.zip and copy (extract) all the files to this directory.
  3. Run 1C:Enterprise and in the opened window "Run 1C:Enterprise" press the "Add..." button.
  4. Specify the variant "Add an existing info base to the list" and press "Next".
  5. At the next page of the assistant specify the variant "On this computer or on the local network computer" and enter the name of the info base. It is recommended to specify the same name as a name of an offline workplace.
  6. At the next page specify the subdirectory Infobase in the directory Creatingd at the step 1 and then at the final page of the assistant press "Done".
  7. Run the application using the command "1C:Enterprise" (at the right top corner of the window "Run 1C:Enterprise").
    In the window of an access to the info base in the field "User" select from the list the user on behalf of which there was Creatingd an offline workplace. The field "Password" leave empty.

After, to open the application it is necessary to run 1C:Enterprise and in the opened window "Run 1C:Enterprise" select in the list this info base.When working in the application to configure the synchronization of the data with an application in the Internet it is possible to use the command "Offline work" in the section "Administration".

3. How to set an offline workplace for the work of other users?

At once after the first running the offline workplace is ready to work. Your account has administrative right in the application in the Internet as well.

If this offline worplace is going to be used by other users then it is necessary to set for them an access to the infobase in the "Users" list in the section "Administration". IN this case they will be assigned the same access rights as in the application online. To get more information on maintaining the list of users and setting up an access see the application documentation .

Also if it is necessary for the users without your participation to synchronize the data with the online application then change the Internet connection settings, set by default. To do this use the command "Set up the connection... " in the "Offline work" window in section of "Administration". IN the form of settings specify the user on whose behalf it is necessary to synchronize data . For this user in the online application it is necessary to specify the profile "Synchronize the data with other applications."
