It is used to configure scheduled job for performance measurements export.

This can be necessary, for example, to analyze performance in another application.

You can call it by running the More - Settings command of the Performance estimation window.

Settings of performance measurements automatic export

  • Select the Export check box, at that the Export to local directory , Export to FTP check boxes are selected automatically.
  • Select the path in the Export local directory field (if the local directory is not used for the export, then clear corresponding check box);
  • If necessary, specify ftp-server address or clear the Export to FTP check box.
  • To go to further export configuring, click Set up export schedule, complete the schedule.
  • To confirm and return to the application, click Save and close.
  • After this, performance measurements are exported automatically. Formed external files and files containing settings used to receive these measurements are automatically packaged in ZIP-archive.
