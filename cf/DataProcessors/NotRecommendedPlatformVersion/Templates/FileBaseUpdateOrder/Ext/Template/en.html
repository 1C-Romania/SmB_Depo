Update to 1C:Enterprise 8.3 version

1. Export 1C:Enterprise 8.3 distribution package

Export the distribution package of the latest version of the 1C:Enterprise 8.3 platform with the 1C:Update service on the 1C:ITS portal (portal.1c.ru). If necessary, enter a login and a password.

Select the active (recent official) version of the process platform.

To export the file, go to 1C:Enterprise process platform for Windows:

The Export distribution package link opens a dialog for saving the file on the disk. Remember the folder where you save it:

2. Install new 1C:Enterprise 8.3 version

Note: installation in Windows requires administrator rights. Unpack the exported archive. IN the folder to which you unpacked the file, find and start Setup.exe:

Follow the instruction of the Assistant:

3. Create a backup of your infobase

Open the directory that contains the necessary database:

4. Start your work with the new platform version

Use the 1C:Enterprise button to start your work with the selected infobase on the new platform version.
