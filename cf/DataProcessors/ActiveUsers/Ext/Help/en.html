It contains the list of users working in the application at the moment.

In this list both users working with the database using "Thin client", "Web client" or "Designer" applications and users such as "COM-connection", "WS-connection", "Background job" etc. are dispayed.

The list is displayed by running the corresponding command of the Administration section - Support and service.

The list contains the following data:

  • User - User full name. External users are also displayed
  • Application - Application type is indicated, for example, Thin client. Connection mode is also specified. Moreover, sessions set for other connection modes, for example, connections (COM) or web services, are indicated
  • Operation start - date and time of operation start
  • Computer - computer network name used to enter the application
  • Session - session number and data separation

Total number of active users is indicated in the bottom line of the list.

Bold font is used for the line corresponding to the user working session using which the list of active users is opened.

For easy view of the list you can use any column sorting.

Filters

  • Using Show field you can select any filter based on one of the predefined values from the list. All sessions value is selected by default.

Viewing the user

  • To view user data, use double mouse click over the desired line.
  • Besides, you can select the desired line and run More - Open command.

Viewning the log file

  • To analyze the selected user operation you can open log file using Log File command. The log file is filtered based on selected user.

List update

  • The list of active users is not the dynamic list. To view active users included in the list after its opening, click Update the current list or click F5 button.

  • End of session (client-server mode)

    • In client-server operation mode End session command is available. This command ends user sessions selected from the list forcefully.
    • To run this command in a right way, first it is necessary to set application administration options (settings window opens automatically).
