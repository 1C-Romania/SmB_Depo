Security profiles - mechanism of 1C:Enterprise platform version 8.3 that provides managing the external (relative to 1C:Enterprise server clusters) infobases resources, such as file system directories, COM objects (for Windows-based systems), and external (relative to application) software modules based on 1C:Enterprise scripts, such as external reports and data processors, Run and Calculate methods calls, external components, operating system applications, and Internet-based resources. You can use security profiles to isolate any infobase and lock it from using any external resources not permitted by the administrator.

Previous versions of the platform required security profile with full access to external reports and data processors. IN the latest version, this limitation was removed and there is an option to specify stricter permissions for external code to use external resources (including code for external reports and data processors) than for infobases.

Now permissions to access external (relative to 1C:Enterprise server) resources are managed automatically. Administrator no longer has to maintain manually the list of external resources required for the application to operate. Instead, whenever the application requires access to an external resource, the user sees a dialog with the list of required permissions. If the requested permissions are approved by the administrator, they will be added to the security profile on servers' cluster. The security profiles generated for the applications are as strict as possible (for example, Full access to external reports and data processors check box is not selected by default).

When working with thin client under Linux, it is necessary to install RAS administration server in order to enable automated permissions control for security. When working under Windows, there are two options available: either using a RAS administration server or COM connection.

To enable automated permission management, adjust the settings of the security profile usage mode in the Administration - Support and maintenance - Security section.

Security profiles are only available to owners of CORP client and server licenses for 1C:Enterprise 8 platform.
