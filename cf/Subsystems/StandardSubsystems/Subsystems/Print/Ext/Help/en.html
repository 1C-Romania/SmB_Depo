Many configuration documents have a printed form. All the printed forms use a single mechanism to edit the layouts of printed forms without changing the configuration itself. Printing objects forms can be formed according to the table layout (MXL) or to the layouts of the office documents in the formats of Microsoft Word and OpenOffice.org Recordr.

In the forms of the documents with the possibility of displaying of the printed forms in the command panel there is a Print command where are located the printing commands of the specific printing forms. Similar commands exist in the form of the document list. IN the list it is possible to select multiple documents at the same time and execute the print command for the selected documents.

As a result of this command the printing forms will be generated and opened for viewing in the special form of "Print the documents". To print the generated printing form is possible by using the Print command.

  • Send via email - generated tabular documents can be sent as an attachment to an email.
  • Go to the document - is possible to scroll the window of the printing form to the printing form of the selected document if it is not displayed in the printing form.window.

The setting of the printing forms is performed in the section Setting and administration by the command Printing forms layouts.
