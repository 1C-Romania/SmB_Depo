﻿Configuration allows you to exchange data between different infobases according to the exchange rules.

Exchange rules are created by configuration "Data Conversion, edition 2.1."

Data exchange can be one-way or two-ways and can be performed automatically (according to the specified schedule) or manually on the user's demand. Various communication channels can be used for data exchange: network location, an email or FTP.

To customize exchange settings, in section Administration, in the navigation panel, click Data synchronization.
