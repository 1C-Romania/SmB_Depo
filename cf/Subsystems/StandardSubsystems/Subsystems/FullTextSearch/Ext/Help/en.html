Configuration allows you to perform a full text search in any configuration objects by searching the specified string in all object attributes. If there is any attached file containing text information then the search is performed in them as well.

Full-text search provides such opportunities as transliteration support (entry of Russian words with Latin characters according to GOST), replacement support (entry of the part of the characters in Russian words by Latin characters with the same key); fuzzy search (characters in found words may vary).

To enable and configure the full-text search, it is necessary to use the form of System Parameters Settings in section Settings and administration.
