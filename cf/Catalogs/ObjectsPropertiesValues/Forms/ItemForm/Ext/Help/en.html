Th form is intended to enter the additional attribute (data) values.

The application provides the use of several types of additional attributes (data). The data type is selected when setting an additional attribute (data). The application suggests entering values when selecting the Additional value data type. At that, the list of values is used for each additional attribute or data type.

Entering values

  • Specify the Name of an additional attribute (data) value.
  • Using the Included into group field a new value can be included in one of the groups. To enter a new group, click Create (F8 key), and then type the name. To select from the available groups, click Show all (F4 key), and then select the necessary group from the list.
  • In case the Values are also characterized by weight factor check box is selected for an additional attribute (data), enter numerical value in the Weight factor field.
    • This field can be used further for the analysis in reports. The logic of using this field is to be defined by the user. For example, for the Reliability property values you can set weight factor as follows: set the weight factor to zero for values from one to three, and then set the factor to one for values four and five. With such weight factors you can filter the reliable clients (with weight value more than zero) quickly in the report.
