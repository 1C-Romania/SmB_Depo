The list defines values of additional attributes and data.

You can use additional attributes (data) of different types. Select data type when configuring additional attributes (data). When you select the Additional value data type, you are prompted to enter a value into this list. Each additional attribute or data of this type uses an individual value list. Use this data type to describe various properties of the application objects.

The Additional value data type is used quite frequently. Mostly you enter information by selecting values from the list, so the process is quick and error-free. It is used like an ordinary or grouped value list.

Selecting values

  • Select a necessary item.
  • Click Select.
  • You can double-click the required item.

Entering values

  • To enter values, click Create and fill in required fields.

Grouping values

  • If there are a lot of values, you can group the list.  Enter a required number of groups. The groups can be independent or substitute to each other.
  • To add groups to the list, click Create group and fill in the required fields. 
  • Drag values to relevant groups.
