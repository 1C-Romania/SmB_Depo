You can decrypt data in this form.

The following data is displayed:

  • Data that will be decrypted is specified in the link on the top of the window.
  • You can select a certificate installed on a computer in the Certificate field. You can decrypt data with the help of this certificate.
    • If there are no certificates in the drop-down list, it means that you need to install the certificate with the closed part of the key on the computer.
      The certificate to be installed shall be from those certificates with the help of which the data was encrypted (for example, you can check the file certificates in its card on the Allowed to decrypt tab).
    • If the list of certificates by which the data was encrypted is empty, but it is known, with which certificate the data was encrypted, then in the drop-down list click Show all, and then select this certificate.

Decrypting data

  • Enter the password for the selected certificate in the Password field.
  • Select the Remember password check box in order not to enter the password next time.
  • Click Decrypt.
