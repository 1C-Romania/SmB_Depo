Use it to locate one or more reports variants in the application sections.

Use the Place in sections command to open in the Reports list.

  • The Reports variants field lists the options to be placed.
    • UsingSelect you can add the report variants for placement.
    • Use the Value list to Add a single variant of the report. Click Pick up to open the list of the report variants and select some report variants, and then click Select.
    • Click OK to confirm your selection and return to the variants placement. If you want to edit the list, click Cancel to return to the placement of the report variants.
  • The Section, group column indicates the tree of sections and groups of the software command interface.
    • Select and clear the check boxes to add report variants to the sections/groups of the command interface and delete the variants from the sections/groups.
    • On the More menu, you can use the Clear check boxes command to cancel your selection and restart placing.
  • Define the report option importance using   Select:
    • Define as Important and thus highlight the report variant in bold in the report panel.
    • Place the report variant in the reports panel in the See also group.
  • Click Place to confirm your selection.
