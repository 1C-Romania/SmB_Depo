It is used to support users working with the application externally. As a rule, their access is limited. External users do not take active part in accounting processes, work separately and do not interact with each other. For example, employees in the human resources base, customers of the online shop, respondent of the questionnaire base.

You can open the external user list from the Administration section - User and right settings. You can also open it from the Questionnaire (Access management) section.

The application external users are of different types so their information is stored in different application lists.

The external user list is maintained by the administrator or the user responsible for the external user list.  

Only the administrator can allow to enter the application.

The person responsible for the external user list (with the right to add new external users) can maintain the external user list to a limited extent, configure the external user before the application login is allowed by the administrator. The responsible person can use the External access command, edit the External users and External user groups lists.

External user entered by the user responsible for the list receives Prepared status. The application administrator can select such users using this criteria, check the correctness of data filling and allow access to the application.

After the administrator allows to log in the application, the person responsible for the list can not change the login settings, except for disabling permissions to log on to the application.

All external users can view information of themselves and also update some properties, namely: password, application language, and contact information.

In the distributed infobase the application logon is configured separately for each infobase of the application.

Entering external user

  • The main way of entering and editing external users is by means of the External Access command located on the command bar of the relevant application lists and objects. When running the command the external user connected with this object is immediately created or the existing external user is opened.
  • You can also enter the external user to the list. Click Create. IN this case it is necessary to fill in the Object connected with this external user.

Viewing external users of lower-level groups

  • To see all external users included in child groups taking hierarchy into account it is necessary to select the Show external users of the lower-level groups check box in the lower part of the list. The check box is selected by default.

Viewing inactive external users

  • External users can be marked as inactive. To view such external users, select the Show inactive external users check box in the lower part of the list. The check box is cleared by default.

Viewing infobase users

  • To control the entering correctness or in case of incorrect entering of external users, you can move to the record control and external user list synchronization with the infobase users list on the More menu by the Infobase users command.

Grouping external users

  • If you can use the user groups in the application, the list consists of two parts: the required number of groups is in the left part, the list of external users is in the right part. If you select a group, external users included into the group are displayed at the right part.
  • If there are many external users in the application, you can group the list, for example, by the kinds of relations with external users: partners, counterparties, etc. 
  • An external user can be included into several groups.
  • You can include an external user in the groups using several ways:
    • When entering the external user, you can select the group from the list and then click Create.
    • To add large number of external users to the group, you can open the group window and click Pick. At that, the External users list consisting of three parts is used for the selection.
    • Mark an external user in the list by running the Assign groups command on the More menu, include it into one or several groups. You can also find this command in the shortcut menu using the mouse right button.
    • You can drag one or several external users to the group you need. To use this option, you shall clear the Show external users of the lower-level groups check box.

Entering external user group

  • Click Create group, fill in all necessary fields.
  • You can also use the Create command of the shortcut menu by clicking the right mouse button in the left part of the list.

Moving external user group to another group

  • Select the group at the left part of the list, use the Move into the group command on the shortcut menu of the right mouse button.
  • Select the group to be moved.

Selecting external user group participants

  • If the group members are selected from the external user list, it consists of three parts.  The groups are displayed at the left upper part of the list, the external users of the selected group are displayed at the lower part. The selected users are displayed at the right part of the list.
  • For convenient selection already available groups of external users are displayed at the upper left part of the list. Selecting the required group (or All external users group) you can view users of this groups at the lower left part of the list.
  • Select one or several users at the left bottom part of the list and move them to the Selected users right part using the arrow.  You can also move the external user from the left lower part to the right part using the mouse double click.
  • To cancel your choice remove one or several undesired users from the right part of the list using similar actions.
  • Using double arrow << you can immediately delete all external users from Selected.
  • The application counts the selected external users.

Viewing external user data report

  • You can use the report to analyze the login settings (IB user properties) together with the other properties of the external user in the catalog. It includes 3 report variants - user report, external user report and report of all application users.
  • Click More - Information of external users, generate the report.
