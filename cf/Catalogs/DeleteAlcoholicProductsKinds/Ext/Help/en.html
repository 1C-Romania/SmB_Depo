Alcohols products kinds

The catalog is intended to store the information on the kinds of alcohols and alcoholic products that are used for filling the sales volume register of alcohols and alcoholic products.

It is required to fill a product type code in the catalog form, the name and a license type.

For example, products type code 404, the name "Wine with protected geographical location or with the protected name of the place of origin", license kind - "Alcohol".
