Designed to save a file version.

Displayed when the file is edited, i.e. after running the Save changes or Finish editing commands and creating a file version in the following lists: Files, Attached files, Edited files, List of locked files at logoff, Files in main work directory.

Saving version

  • The file to save is displayed under the window title.
  • In the Comment to version field, you can write a short comment describing your changes in the file. 
  • If the Create new version check box is not selected, the updated version will be written over the old one. It is recommended to create versions, since if your file is damaged, you can restore an earlier safe version. By default, the check box is cleared. You can change the settings with the Store versions check box in the file card or when loading files (folders).
  • To confirm, click Save.

View the file versions list with the relevant command on the navigation panel of the file card.
