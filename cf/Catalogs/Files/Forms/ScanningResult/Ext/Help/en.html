Use this window to scan one or more images. Use the Create file - From scanner command in the Files list (Attached files).

Scanned images

When the first image is scanned, it is shown in the Assistant window; the information bar under the image shows the format and other image parameters. You can use the following commands:

  • Scan more - run this command to scan multiple documents.
  • Rescan - delete the current image and scan again.
  • Accept - create a file from the image (the command is available when scanning one image). The file card is automatically created and displayed. The assistant window closes.
  • Receive as one file - the command is available when scanning two or more images. It creates a multi-page file (TIF or PDF) from all the images. The file card is automatically created and displayed. The assistant window closes.
  • Receive all as separate files - the command is available when scanning two or more images. It creates a separate file from each image. The file cards are created automatically. The assistant window closes.
  • Cancel - give up further scanning and quit without saving the scanned images.
  • Set up - edit the session scan parameters defined in the personal application settings. Define the parameters for this session and click OK. The images you scanned before are not deleted.
