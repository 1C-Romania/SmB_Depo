Designed to select a file creation method.

Open the window with the Insert button or the Create command of the context menu from the file list or the attached file list.

File creation methods

Switch to the necessary variant:

  • From template -  create a new file from the selected template.
  • From file on the drive - create a new file from the existing file on the local or network drive.
  • From scanner - create a new file from a graphic file from scanner (if the computer is connected with a scanner). Before exporting files from scanner, set up scan.

Click Create to add a file.
