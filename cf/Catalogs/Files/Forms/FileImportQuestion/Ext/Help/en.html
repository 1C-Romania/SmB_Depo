You can simultaneously load multiple selected files or multiple folders from your computer drive. To do it, use the relevant commands in the  file list.

If one or more selected files are too large, you will see the warning with a list of the files you can not load.

Click Proceed to:

  • Import the remaining files of proper size.
  • Get an error report with a list of the files that failed to be imported.

Click Cancel to return to the application without further attempts. If you were planning to load multiple files and other files meet the requirements, such files will not be loaded.

You can increase the maximum file size in the application settings: in Administration - File operation settings, specify Maximum file size. By default, it is 50 MB.
