The list is used to manage application files.

The list displays the following data:

  • - availability of digital signature and encryption
  • Name - full attachment file name and description. Icon next to the name specifies the file type
  • Location - the file list folder or application object where the file is attached;
  • Author - the user who added the file
  • Creation date - time and date of the file adding to the base
  • Size (Kb) - file size
  • Extension - file extension
  • Editing - the user opened the file for editing
  • Version - file version number
  • Edited - the user added the file version
  • Volume - volume of the disk where files are stored
  • Path to the file in the volume

The following actions are available:

  • View - Open file for viewing using external aplication
  • Cancel editinging - clear the sign showing editing of the file
  • Open a card of the file using corresponding command on the More menu
  • Save file in the specified catalog on hard disk or network drive using the More - Save as command
  • Delete a file permanently using the More - Delete command (Shift + Del)
