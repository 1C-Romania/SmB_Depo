Designed to import a folder from the computer drive. IN Windows file explorer, select a folder in the file list and use the Export folder command of the context menu. IN the file explorer, specify a path to the folder, and then click Select folder. Fill in the required fields: 

  • The Directory on drive field is filled in automatically; select another folder if necessary.
  • The Folder field automatically specifies the selected folder of the file list to store the import folder. If necessary, you can   select another folder.
  • To save disk space, select the Delete files after adding check box (by default, it is cleared).
  • Enter Description of new files.
  • Select the check box to Save versions (recommended; by default, it is cleared).
  • For text files, you can Select encoding from the list.
  • To confirm, click Export folder. If the folder to be exported is empty, the message "No file to export" appears.
You can also drag the export folder from the hard or  network derive - just drag the file from Windows file explorer to a necessary folder of the application file list.
