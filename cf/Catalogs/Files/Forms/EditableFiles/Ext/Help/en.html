This list displays only files you locked for editing. You can edit files in the files list or in the attached files list for the application objects.

Open it with the Editable files command in the Organizer section.

The list displays:

  • File - the attachment file name.
  • Location -  the folder containing the file or object to which you attach the file.

Available actions

  • View - open the file to view.
  • Finish editing - save the file version and makes the file available for other users.
  • Open card - open to view the file data.
