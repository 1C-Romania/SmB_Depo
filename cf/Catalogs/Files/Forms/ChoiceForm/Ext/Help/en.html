It is intended for selection of a file template. You can select a file template from the File templates folder.

The application is supplied with a File templates predefined folder. The folder contains files which are used as templates for creating new files. For example, a template of a contract document can be placed in this folder to be used to create contracts wih specific counterparties.

It is opened from the Files or Attached files lists by the Create - By template command.

The list consists of two parts: on the left there is the File templates folder, on the right - the list of file templates included in this folder. The list consists of columns as follows:

  • - existence of a digital signature or encryption
  • Description - the file format and name. Information on the file is displayed as well
  • Author - the user who created the file
  • Date of creation - the date and time when the file was included in the application

Creating file from template

  • Select the needed template in the list, click Select. Alternatively you may double click the template

  • The application will automatically create the file card display it. Complete the required fields, click Save and close

Adding files template

  • To place a new template into the Templates folder, click Create file, using the radio button select:
  • From file on hard drive. Then select the required file
  • From scanner. Follow instructions of Scanner Assistant
  • The application will automatically create and display the file card. Complete the template information, the Folder field is by default completed as "File templates"

Displaying information on file

  • Click More - Open card to display the file card
