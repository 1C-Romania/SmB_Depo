Use the embedded text file editor to view and edit a file, compare it with the original while editing and open it with the standard application if necessary.

Open the editor from the Files list by clicking Open or Edit file if it is allowed in the personal application settings.

By default, the embedded editor opens TXT, INI, XML text files. You can expand this extensions list in the personal settings. On the Working with files tab, in the File opening settings table, you can specify the file Extensions for the embedded text editor (separated with a space). You can also specify Open with - Standard application or Embedded editor (by default).

Editing file

  • If a file is opened for viewing, you can not use the Save and close and Save buttons. Click Edit to edit the file.
  • The embedded text file editor functions like an external application. The file is copied to the local computer and the infobase marks it as locked. Other users can not use it until you click More - Finish editing. 
  • After you finish editing, click More - Finish editing to clear the "locked" mark. Depending on the settings, a new file version is saved or the file is updated in the storage. By default, a new file version is created. To change this setting, clear the relevant check box when creating a version.

Viewing tasks

  • Click More - Tasks to view a list of tasks based on this file.

Viewing file card

Click More - Open card to view the file information.

Comparing versions

  • Click More - Show differences to compare file versions. If you did not preliminarily select the comparing application in the personal application settings (Working with files), you are prompted to select it: Microsoft Word or OpenOffice.org Writer.

Saving file

  • Click More - Save as to save the file in the specified directory on the hard or network drive. 

Opening file in standard application

  • Click More - Open with standard application to go to the external application (Windows notepad) and edit the file. 
  • If you need to open text files with specific extensions with the external application by default, edit the personal application settings. Do the following:
    • In the personal application settings, click the Working with files tab.
    • Edit the File opening settings. IN the Open with column, select Standard application.

Changing text file encoding

  • Text files may be saved in various encodings. The main encodings for text files in Russian are:
    • IBM866 (Cyrillic DOS)
    • KOI8-R (Cyrillic KOI8-R)
    • Windows-1251 (Cyrillic Windows
  • If you open a text file and see nonsense characters, click More - Change encoding, and select another encoding from the list.
