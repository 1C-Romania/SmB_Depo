The main working directory is a special directory that contains the files you view and edit. The list is designed to manage files in the working directory.

You can open the list in personal settings (the Working with files tab) by running command Configure main working directory - Open file list.

The list shows:

  •  File is being edited - the check box is selected if the file is being edited.
  • File - the file format (as a sign) and name
  • Size (KB)
  • Date modified

Use the More - Change form command to add extra fields to the list.

Available actions

  • To delete unnecessary files from the working directory, click Delete. Thus you will only delete files that were not changed.
  • To save the file changes and open it for other users, click Finish editing.
  • To free the files you occupied, use the Cancel editinging command.
  • Click Open file directory to view the folder containing the file you are editing.
  • Use the relevant command of the context menu or the More menu to Open the file card.
