You can save files coming directly from the scanner. Customize scan functions to work with scanner.

Use the relevant command in the personal settings window (the Working with files page).

Before setting up the computer, connect the scanner. Available parameters depend on the scanner capacities. Click OK to confirm the settings.

Scan component

  • Before you set up scan, click Install scan component.
  • If your component does not have a scan, you do not have to install the component.
  • If the component is already installed, the installed Scan component version will be displayed instead of this button.

Scanner name

  • In the Scanner field, select the necessary scanner from the list of available devices (if only the scanner is connected, it is selected by default).

Conversion to PDF

  • Select the check box to Use ImageMagick to convert to PDF. You can convert the scanned images to PDF with ImageMagick. Ensure that the client computer with scanner has these free applications: ImageMagick (use Win32 dynamic at 8 bits-per-pixel) and Ghostscript.
  • Specify the Path to PDF conversion application - a path to the exe. file of ImageMagick (by default, "convert.exe"). If this name conflicts with other applications, use Select to specify the full path like "C:\Application Files\ImageMagick-6.6.3-Q8\convert.exe".

For single-page images

  • Format - you can select one of the supported formats (BMP, JPG, GIF, PNG, TIF, PDF) to place the file in the infobase. If the Use ImageMagick to convert to PDF check box is selected, the list also offers the PDF format (then you can select PDF for storage, other formats - for scan).
    • Quality can be configured if the JPG format is selected (100 - maximum quality and maximum size,  1 - minimum quality and minimum size). By default, it is 100.
    • Compression can be configured if the TIF format is selected. Possible values: LZW, RLE, CCITT3, CCITT4, no compression. By default: no compression.
    • Type (for scan) - select one of the supported formats (BMP, JPG, GIF, PNG, TIF) for the images from the scanner. You can use this option when you select the PDF format of the single-page image. IN this case, PDF is used for storage.

For multi-page images

You can edit its parameters if the Use ImageMagick to convert to PDF check box is selected. By default, it is TIF.

  • Format - you can select one of the supported formats (TIF, PDF) to place the multi-page file in the infobase. It is used if you select Receive all as one file command for multiple scanned images when working with Scan assistant.

Scan parameters

Parameters depend on the capacities of the connected scanner. Some of them may be unavailable.

  • Resolution allows you to select the scan resolution (DPI). The list of available resolutions depends on the scanner you select.
  • Color index of the images from the scanner (monochromatic, grayscale, colored). It depends on the scanner you select.
  • Image rotation allows you to rotate the image when scanned. This option is not available for some scanner models.
  • Paper size allows you to select the size of the paper put in the scanner. This option is not available for some scanner models.
  • Scan both sides - if you select the check box, both paper sides are scanned. This option is not available for some scanner models.

Scanner dialog

  • Show scanner dialog before scanning. When selected, this check box shows the dialog of advanced scan settings before the scan session begins (e.g. you can select the scan area on the page). By default, it is selected.

Configuring default parameters

  • Click Configuring standard parameters to customize the default scanner settings. If the computer does not have PDF conversion applications, keep the recommended default formats.

Viewing list of scanned file numbers

  • Click Open numbers of scanned files to view the relevant list.

Note: Under Windows Server 2008, you can use scanner after installing the Desktop Experience component.
