Designed to configure settings for a given scan session (the settings will not be saved). Open it with the Setup command of Scan assistant.

Depending on the scan capacities and the long-term scan settings, some parameters may be hidden. For example if the "conversion to PDF" option is disabled, you can not change the format of a multi-page image. By default, the TIF format is selected.

Single-page images

  • Format allows you to select one of the supported formats (BMP, JPG, GIF, PNG, TIF, PDF) to store in the infobase.
    • Scanned image type allows you to select one of the supported formats (BMP, JPG, GIF, PNG, TIF) for the images from the scanner. You can use this option if your single-page image is in PDF format.
    • Quality can be configured if the JPG format is selected (100 - maximum quality and maximum size,  1 - minimum quality and minimum size). By default, it is 100.
    • Compression can be configured if the TIF format is selected. Possible values: LZW, RLE, CCITT3, CCITT4, no compression. By default: no compression.

Multi-page images

The parameter can be used if the "conversion to PDF" option is enabled in the scan settings.

  • Format allows you to select one of the supported formats (TIF, PDF) to store in the infobase. Used when the Receive as one file command is selected.

Scan parameters

The parameters are available if the scanner has relevant options.

  • Resolution allows you to select the scan resolution (DPI). The list of available resolutions depends on the scanner you select.
  • Color index of the images from the scanner (monochromatic, grayscale, colored).
  • Image rotation allows you to rotate the image when scanned. This option is not available for some scanner models.
  • Paper size allows you to select the size of the paper put in the scanner. This option is not available for some scanner models.
  • Scan both sides - when selected, both paper sides are scanned. This option is not available for some scanner models.

Scanner dialog

  • Show scanner dialog - when selected, shows the dialog of advanced scan settings before the scan session begins (e.g. you can select the scan area on the page).
