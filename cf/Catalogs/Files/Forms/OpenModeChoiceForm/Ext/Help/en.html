Designed to select a file opening mode.

To open a file in the file list:

  • Click Enter - requires selection of file opening mode, unless other settings are defined.
  • Double-click it - like in the previous case.
  • Click  Open file
  • Click  Edit (F4)

Variants of file opening

  • Switch to a necessary file opening mode:
    • Read only - the file will not have an indication that you are editing it and will open in an external application in the read-only mode.
    • For editing - the file will have an indication that you are editing it and will open in an external application for editing and saving.
  • To confirm your selection, click Open file.

Disabling opening selection mode

  • If you do not want to choose opening mode anymore, select an opening mode and select the Do not ask again  check box.
  • If the personal settings are configured for text files to open In embedded editor, you do not have to select a file opening mode: the editor opens immediately and opening modes are switched in the editor itself.
  • If the file was locked for editing, you do not have to select a file opening mode: the file opens in the editing mode.
  • You can enable mode selection in the application personal settings, on the Working with files tab.
  • In the application personal settings (the Working with files  tab), you can select Action when selecting file:
    • Open file (default)
      • Select the Select opening mode when selecting file check box. By default, the check box is selected.
    • Open file card - in this case, you press Enter or double-click to open the file card; you do not select a file opening mode.
