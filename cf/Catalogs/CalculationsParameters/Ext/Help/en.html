Calculation parameters catalog

The preconfigured data selection rules from the different accounting sections are Saved in this list. The data is used for the payroll calculation.

Every item of this list contains a query template for obtaining some value from the database. The query may be composed in two ways. If the Custom Query flag IsSet, then the query text and parameters are available for editing; a query assistant is available in the fat client only. If the flag is not set, the following data should be specified: data source, filter periods, indicator (one), filter parameters. IN this case, the query will automatically be generated. IN case when the Specify Value on Payroll flag IsSet, the item does not require configuration; the parameter value will have to be specified before calculation in the Payroll document

The query parameters may be defined in several ways:

  1. Fixed parameter values could be specified on the Filter tab, or if the Custom Query flag IsSet - on the Parameters tab.
  2. The query parameter values are filled automatically from the Payroll document. Such parameters are: Company, Currency, Structural unit , StructuralUnit, RegistrationFilter period, PointInTime, BeginOfFilter period, EndOfFilter period, Performer, OccupationType, PerformerNumber, TabNumber, Assignee, Ind, Individual, Position, AccrualDeductionKind, CustomerOrder, Order, Project, GLExpenseAccount, BusinessActivity, Size, DaysWorked, HoursWorked. It is not required to define values of the listed parameters. The specified values will not affect the calculation.

The following attributes have to be filled:

  • Id - calculations' parameter presentation in formula.
  • Source of data selection.
  • Specify Value on Payroll - if the parameter value will be entered at the moment of a payroll calculation.
  • Custom Query. The option has to be set if the query and its parameters are edited manually.

On the Data tab, the data selection period is specified and the indicator is specified.

On the Filter tab the filters are applied to a data selection.

On the Query tab the calculation parameter configuration result can be reviewed and adjusted if required. The query can be edited if a button Edit Query is pressed. Query Assistant can be used via the Query Assistant button in the fat client.

  • Query - query text.
  • Parameters - query parameters:
    • Name of the parameter. Being filled from the query text, it does not contain spaces.
    • Presentation of the parameter. The presentation is displayed in the Payroll document.
    • Value of the parameter.

The item may be filled by a template. To do this press the Fill by Template button and pick one of the options:

  • Norm Days.
  • Norm Hours.
  • Worked Days.
  • Hours Worked.
  • Hours Worked by Jobs.
  • Tariff Rate.
  • Fixed Amount.

The item will be configured according to the selected template after one of the option is picked. The item can be adjusted with custom settings later.
