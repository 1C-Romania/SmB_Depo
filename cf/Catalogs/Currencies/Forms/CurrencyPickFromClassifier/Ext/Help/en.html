Contains general information about currencies and is designed to add new currencies (recommended for correct entering of currency data).

All-Russian Currency Classifier indicates:

  • Numeric code - an international standard, the unique numeric code is used to identify the currency when exporting its rates from the Internet (from the RBC website).
  • Character code - an international standard, used as a short name and the main representation of the currency.
  • Currency name - a full name of the currency.
  • Countries and territories - countries and territories using this currency as the primary one.
  • Import from the Internet - select the checkbox to enable loading the currency data from the RBC website.

Adding currency from classifier to currency list

  • Select one or more currencies, and then click Select.
  • Double-click a currency to add it to the currency list.
