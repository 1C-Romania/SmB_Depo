Properties of the currency you use.

Any company uses currency extensively: all money-related documents (invoices, payment orders, contracts, waybills) include an expressed or implied Currency field. This field is filled out according to the company's business area: it may be ruble, some foreign currency or conditional unit.

General currency information

When you add a currency from the All-Russian Currency Classifier, the following fields of the new currency are filled out automatically:
  • Currency name - the full name of the currency.
  • Numeric code - an international standard, the unique numeric code identifies the currency when exporting its rates from the Internet (from the RBC website).
  • Character code - an international standard, used as a short name of the currency and its main representation.
  • Currency rate - if the exported from the Internet checkbox is selected, the currency may be exported from the RBC website.

More about currency rate radio button

  • change the Currency rate is exported from the Internet radio button only in special situations:
    • If you add a currency from the classifier (so you may export its rates) and the software suggests using this currency as the conditional unit with the rate entered manually;
    • For technical reasons, when the currency rate update is available only when the radio button is selected.
  • For your national currency, change the Currency rate radio button to entered manually, since the ruble-to-ruble rate will always be 1, and, for example, a Russian company should select entered manually in the Currency rate field for ruble.

Linking currency rate to another currency

  • The company may use this option to calculate its internal rate of a specific currency.
  • Change the Currency rate radio button to linked to another currency.
  • Select from the list or enter the other currency which rate will determine the rate of the present currency.
  • Specify the Markup, % ratio - the coefficient for the rate of the main currency to calculate the rate of the present currency.

Calculation using formula

  • You can add a currency which rate depends on multiple currencies simultaneously.
  • To do it, change the Currency rate radio button to calculated by formula.
  • Define the rate calculation method in the relevant field. For example, (USD+EUR)/2.

Currency wording parameters

Define the wording parameters for the integer/fraction parts of the currency to use them in print forms.

  • Integer part - define the declension parameters for the currency integer part: Grammatical gender and the relevant singular/plural templates.
  • Fraction part - define the declension parameters for the currency fraction part: Grammatical gender and the relevant singular/plural templates.
  • Decimal places - if the fraction component is represented with a number, define the number of decimal places shown in the text. For example if there is one decimal place, the document amount will be shown as 150.2 euros, two decimal points correspond with 150.20 euros.
  • Amount wording sample - the number entered into the Amount field will be written according to the specified currency wording parameters. Use this option to check your currency wording parameters.

By default, the currency wording parameters are available for the basic currencies: ruble, euro, and dollar. Fill in the parameters of other currencies on your own.

Viewing currency rate

  • To convert the amount into rubles, use the currency rate.
  • Click Currency rates on the currency navigation panel. You can specify the review period for easy viewing of the currencies rates.

Entering currency rate manually

  • You can export the main currency rates from the RBC website. If the rate of some currency can not be exported, the Currency rate radio button is automatically changed to entered manually.
  • To enter a rate manually:
    • Click Currency rates on the currency navigation panel.
    • Click Create, fill in the currency rate as of a necessary date (by default, it is the current date).
