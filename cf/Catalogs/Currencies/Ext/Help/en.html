Use this list to specify the currencies used in the application. Find the necessary currencies in Russian Currency Classifier - RCC (recommended) or add them manually.

Open the currency list in the Catalogs section.

The list displays:

  • Currency name - the full currency name.
  • Numeric code - an international standard, the unique numeric code to identify the currency when exporting the rates from the Internet.
  • Character code- an international standard, used as the main currency representation.
  • Exchange rate- the exchange rate value as of the current date.
  • Multiplicity - by default, it is "one" and is not displayed. If the rate of a certain currency is very low (accuracy less than four decimal places), this field specifies the rate units amount. For example, the Vietnamese dong (VND) is 16.1574 RUB per 10,000 VND. This block includes three small auxiliary columns that contain this expression:
    • RUB per - a logical link.
    • Multiplicity - the amount of rate units.
    • Short name - the currency character code.

Adding currency from All-Russian Currency Classifier (RCC)

  • Click Find in Classifier (recommended).
  • The following fields of the new currency will be filled out from the Classifier automatically: Name, Character code, Numeric code. The Currency exchange rate will be selected automatically too (loaded from the Internet, if the currency rate can be loaded from the RBC website).

Entering currency without classifier (manually)

  • Click Create.
  • Click No when the application invites to use the Classifier.
  • Fill in the currency data.

Importing currency rates

  • Click Import currency rates to load the currency rates from the RBC website.
  • The application also features a scheduled job "Importing currency rates" to load the RBC website currency rates automatically.
