Designed to view and manage file versions.

To view a list of all file versions, open its card (in the file list, press F2 or click More - Open card) and click File versions on the navigation panel.

The hierarchical versions list is shown as a tree to provide better understanding. Thus, the files revision history is easy to monitor. Later versions are subordinate to the earlier ones. You can not delete the first file version from this list.

When the file is edited, you are recommended saving your file version and entering information about this version. By default, this option is enabled (we recommend saving versions to restore an earlier functioning version in case of serious errors that damaged your file). You can save versions by selecting the relevant check box in the file card or when loading your files (folders). Later, you can edit some version data, e.g. a comment.

The list displays:

  • Author - a user who added a file version. This column also displays the file type (as an image).
  • Version number - versions are numbered automatically when you edit them; the latest saved version has the greatest number.
  • Comment - additional information about the file version.
  • Creation date - date and time when the file version was added to the base.
  • Size (byte) - a size of the file data in bytes.

Use the More - Edit form command to add extra fields, e.g.:

  • Full attachment file name - a full name of the file.
  • File storage type - possible values are In infobase or IN volumes.
  • Extension - a file format; and some other fields.

Viewing file version

  • Click View to open a selected file version with a relevant application; you can view and edit it (you can also open a file version by double-clicking the necessary line or with the More - View command).

Viewing file version data

  • Click Open card to open the data of a selected version. You can edit Comment of the version card.

Selecting new current file version

  • Select the necessary file version in the list and click Make active to change the active (current) version. The versions are numbered as before, but the versions positions are new.

Comparing file versions

  • Select two versions in the list and click Show differences to compare them. The versions comparison is supported for the following file types: HTML, TXT, RTF, DOC, DOCX, and ODT.

Saving file version to disk

  • Click More - Save as to save the version  to disk or some other device. Select a directory to save the file.
