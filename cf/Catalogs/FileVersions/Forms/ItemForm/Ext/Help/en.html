You can use the form to view the information on file version.

Application suggests saving the file version and entering data about this version after completion of file edition (by running the Complete command in the files list). This option is enabled by default (it is recommended to use versions saving as in case of serious mistakes that may lead to file damage, there would be an option to return to earlier, non-damaged version). Subsequently there is an option to edit the comment and if necessary to change the version author.

Information on file version

  • Owner - file owner name, is automatically filled in when adding a version, it is not subject to edition;
  • Name - version name, is automatically filled in when adding a version, it is not subject to edition;
  • Parent version - select the necessary version from the list to subordinate one version to another. The versions history is clearly presented as a tree in the application;
  • Size (byte) - file version size in bytes.
  • Extension - file version extension (format). There is also a picture showing the file version format, th picture is defined by the application automatically.
  • Comment to version - enter additional info about file version (it is also possible to enter when saving a version after completion of file edition).
  • Author - the user who added a file version. If necessary, it is possible to select other author from the users list.
  • Creation date - date and time of file version addition in the base.
  • Version No. - versions are numbered by the application automatically when changing a file, thus the last saved version has the maximum number.
  • Change date - file version change date (local time).
  • Encoding - value is set to Default automatically.

Viewing file version

  • Click View to open the selected file version with the corresponding application for viewing or editing (there is an option to open a file version double-clicking on the line you need or on the More menu also).

Saving file version to a disk

  • To save file version to a disk or other device, click Save as. Select directory to save the file to.
