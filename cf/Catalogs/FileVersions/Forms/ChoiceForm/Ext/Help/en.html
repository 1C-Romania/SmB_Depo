You can use the form to select a file version.

To run the form go to the file version card when choosing a Parental version.

The list includes:

  • Version No. - versions are numbered by the application automatically when changing a file, at that the last saved version has the maximum number.
  • Author - the user who created a file version. Also the file type is displayed in this column in the form of a picture.
  • Creation date - the date and time of file version created in the base.
  • Size (byte) - file version size in bytes.

Selecting file version

  • Select the version you need, click Select. You may double-click also. At that there is an option to choose one file version only.
