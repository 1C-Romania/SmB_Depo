You can select the application to compare file versions. The application enables you to compare HTML, TXT, RTF, DOC, DOCX, ODT files versions.

You can select a text editor that will be used to compare files versions. To open the editor, run the Show differences command from the file versions list if the text editor for comparison is not selected by default.

Selecting version comparing method

  • In the Compare file versions using field, select the application to compare files from the drop-down list: Microsoft Word or OpenOffice.org Writer are available for selection.
  • The selection is saved in personal application settings.

Default settings

  • You can select the text editor to compare versions in the Compare versions using field in the application personal settings (on the Work with files tab). You can select Microcoft Word or OpenOffice.org Writer. This setting is not set as default automatically.
