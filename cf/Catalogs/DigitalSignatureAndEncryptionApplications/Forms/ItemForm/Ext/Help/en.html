Information of digital signature and encryption application.

To open the application, click Add in the Digital signature and encryption settings list on the Application tab.

Entering application information

  • Select the application you need from the list in the Presentation field. Frequently used applications are displayed in the list.
      • The rest fields are filled in automatically.
    • If there is no application you need in the list, select <Other application>
    • In this case, the rest fields are filled in manually
  • All fields are mandatory.
  • Application name - full name of the application.
  • Application type - the number indicating type of digital signature provider.
  • Signature algorithm - digital signature algorithm is selected from the list provided by the digital signature provider.
  • Hashing algorithm - is selected from the list provided by the digital signature provider.
  • Crypto algorithm - is selected from the list provided by the digital signature provider.
