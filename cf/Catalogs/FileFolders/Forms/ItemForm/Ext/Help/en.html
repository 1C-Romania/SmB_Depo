Viewing the files folder data.

Folder data

  • Fill out the folder Name. The field is mandatory;
  • One folder can be included into another. Select a parent folder from the list in the In folder field if necessary;
  • The Full path field is filled by the application automatically;
  • Specify the detailed folder information in the Abbreviation field;
  • Select a user responsible for the file folder from the list in the Responsible field;
  • In the Creation date and time are filled automatically in the Creation date field.
  • Select Working directory for a files folder if necessary . IN this case after editing, the files of this folder will be saved separately in the specified working directory.

Access rights to a folder

  • Various access rights can be assigned to folders. To work with folder access rights, click Rights settings.

Quick switch to files list

  • Click Open folders and files list to go to  the Files list.
