Alcohols product kinds catalog

You can use the catalog to store the information on alcohols and alcoholic products kinds, used for filling out the sales volume register of alcohols and alcoholic products.

You need to fill out the kind code of a product, the name and the kind of license on the catalog form.

For example, product kind code is 404, the name is "Wine with protected geographical location or with the protected name of the place of origin", license kind is "Alcohols".
