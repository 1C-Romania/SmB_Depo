Description of the contact information field properties.

The application is delivered with predetermined contact information fields for some documents and lists. You can change only settings of the predetermined contact information fields; other details are filled in by default and copy protected.

Contact information types

You can store the following contact information types:

  • Address - a field with this contact information type is designed to store addresses as an arbitrary line or an address template. State the address with the template using the address classifier. If you state the address with the address classifier, it states the address postcode automatically, after you finish entering the address. The predetermined actual and legal addresses are used in the document print forms.
  • Phone - a field with such contact information is designed to store phone numbers as arbitrary lines or according to a phone number template. The predetermined phone number is used in document print forms.
  • Email address - a field with this contact information type is designed to store an email address. Use it when sending email messages from the application.
  • Web page - a field of this contact information type is used to store web page addresses.
  • Fax - a field with this type is designed to store fax numbers as arbitrary lines or according to a fax number template. Use it in document print forms.
  • Other - a field with this contact information type can be used to store arbitrary text information, e.g. you can explain how to find your office. You can enter up to 1024 characters into this field.

Adding contact information field to list or document

Enter required information:

  • Select a contact information Type from the list.
  • Enter Name which will be shown with the contact information field in a list or a document.
  • Enter your ToolTip text that will be shown when you hover the mouse pointer over the contact information field.
  • Select Group, unless it is defined by default. The group corresponds to the document or list to which a new contact information field will be added.

Customizing contact information fields

  • Select the Required filling check box to automatically check filling of a contact information field when entering documents or lists. You can not save a database without filling in this field.
  • The Edit only in dialog setting is used for the following contact information types: Address, Phone, and Fax. With this setting, you can use a template to constrain entering a new or editing an existing address; your address will be limited with the address classifier. When the check box is selected, you can enter your address in any form. It may be necessary in extreme cases, e.g. when the address data in some documents is wrong, but it has to be specified. Such address will be stored with a mark showing that it was entered in a free form, without constraints. As soon as you get proper information, you can edit the address using the template.
  • Select the Allow multiple values check box to enable entering more values, e.g. a company or an individual can have multiple contact email addresses or phone numbers.

Customizing address field

  • Select the Only Russian check box to restrict selection of other countries when filling in the address. Use this option when all your company employees have only Russian addresses to speed up the procedure. IN this case, extra settings are available:
    • Restrict entering incorrect - when the check box is selected, you can not write an incorrect Russian address (recommended only when you use data-entry templates and the address classifier; enable this option by selecting the Edit only in dialog checkbox).
    • Check with RAC
    • Check with FIAS - with the FIAS classifier, your addresses are stored with extra fields (e.g. District).
    • Hide obsolete addresses - select this checkbox to hide obsolete addresses according to the last address classifier export. Such addresses are marked with  in the list. By default, the check box is cleared.
  • Include country in the representation - select the check box to specify country in the Address field when automatically creating a full address.

Customizing email address field

  • If a field has the Email address type, the email address is verified. Select the Restrict invalid addresses check box to prevent entering incorrect email addresses.
