The catalog contains the description of properties description of the Access groups folder.

Since there may be a large number of the Access groups and the groups with similar settings of the rights restrictions, there is an option to create folders for easier search of the Access groups you need and for better understanding of the access rights restrictions in general.

You can group the access groups in folders according to various attributes.

Entering access groups folder

  • Enter folder Name;
  • You can include folder into Parent group (folder). Select necessary folder from the list, and then click Select button. You can also double-click to select a parent folder.
