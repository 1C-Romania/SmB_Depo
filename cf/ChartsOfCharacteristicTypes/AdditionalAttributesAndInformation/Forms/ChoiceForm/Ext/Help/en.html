The list is used to select additional attributes (data).

You can open the list using the Additional attributes (data) list by running the Add - By template or Add - Common commands as follows.

Adding according to template

There is an option to use previously entered additional attributes and data from other sets, add attributes with common list of values for several single-type attributes (for example, Color of door by the Color attribute template). IN this case you can use the list of values from the attribute-template. It is convenient to configure centralized setting of the values list for several single-type attributes simultaneously.

  • Select already existing additional attribute (data).
  • Click Select.
  • You can also double-click.
  • The type and all values of the additional attribute (data) selected as a template will be copied to a new item.

Adding common additional attributes (data)

The common attribute is included in several sets. This option is suitable for the cases when not only the list of values shall be common, but the attribute as well. The common attributes are grouped and displayed as one attribute in one column or field in the items lists and reports of different catalogs and documents. For example, using common Color attribute there is an option to select all yellow clothes and shoes.

  • Select already existing additional attribute (information). If no required atribute is in the list, click Create. Fill in all necessary fields.
  • Click Select.
  • You can also double-click.
