It enables you to view time measurements that are made to assess application performance.

To open the list, in the navigation bar of the key operation, run command Time measurements.

The following data is displayed in the list:

  • Key operation - full name of the key operation
  • Measurements start date - date and time of measurements start
  • Session number
  • Execution time in seconds
  • Record date - date and time of record
  • End date - date and time of measurements end
  • User - full user name
  • Local data record

Click More - Change form to add additional fields, such as:

  • Name - name of key operation in application
  • Priority - displayed from the list of key operations
  • Minimum allowed level - displayed from list of key operations
  • Target time - displayed from the list of key operations.

Application fills data on measurements automatically on performing the key operations. However, you can enter data on time measurements manually if needed. You can use this opportunity for debugging, for example. Click Create, fill in required fields.
