The application enables you to enter the reminders and specify the time and subject, which activates the list of reminders at the time specified.

Various catalogs, documents and other application objects can be the subject of the reminders. For example, you may enter a reminder linked to a certain partner, goods register etc., or a reminder about specifying or filling up any data.

You can open the reminder by running the Organizer - Remind command from various application objects. At that there is an option to enter only one reminder for an object.

You can also enter a reminder linked to any personal event or activity in the My reminders list by running the Create command. Such a reminder can be useful, for example, to remember to wish a happy birthday to your colleague in time. The reminders combined with notes are well suited as an electronic replacement for the "stickers on the edges of the monitor".

Reminder list

At the time specified in the reminder, the application opens the Reminders list. The list contains all newly created and delayed reminders, that are about to be actuated. The text and overdue time are displayed for each reminder. If a reminder is linked to any subject date (for example, with a job deadline), then the reminder displays the time left till this event.

There is an option to go to a catalog item or another application object linked to the reminder directly from the list (by running the Open command or double-clicking). If the reminder is linked to the application object, the reminder will be opened.

You may click the Stop button to cancel the selected reminders, and delete them from the list. A periodic reminder remains active according to configured schedule after clicking the Stop button. You may cancel such a reminder in the My reminders list. You can postpone the reminders in the list for some time specified in the Remind after field by running the Postpone all command. You may fill the Remind after field with a value from the list, or type in a new value at any time.
