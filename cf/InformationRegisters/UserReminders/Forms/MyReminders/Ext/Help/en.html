It is intended to work with all reminders, running in application at the moment. At that, the user sees only his/her own reminders.

The list includes:

  • Reminder time - reminder actuation time;
  • Description - text of the reminder;
  • Event time - time that was set up for the reminder at the moment of its entry, or time in a reminder subject, in case Reminder time is relative;
  • Source - application object linked to the reminder.

Entering the reminder

  • You can enter a reminder linked to any personal event or action not associated with any specific application object.
  • Click Create, and then fill in required fields.
  • Source column of such reminders contains Not specified in the list.

Viewing the reminder

  • To view or edit the reminder, select the line you need in the list, click Modify.

Canceling the reminder

  • Select the line you need in the list, click Stop.
  • Application displays a confirmation message. Click Stop. To cancel your action, click Continue.
