It is intended to display actuated reminders.

At the time specified in the reminder, the application opens the Reminder list. The list contains all newly created and delayed reminders, that are about to be actuated. For each reminder the text and overdue time is displayed. If the reminder is linked to any subject date, for example, with a job deadline, then the reminder displays how much time is left till this event.

There is an option to go to a catalog item or another application object linked to the reminder directly from the list (by running the Open command or double-clicking). If the reminder is not linked to a application object (the reminder will open).

All reminders in the list can be postponed for some time specified in the Remind after field by the Postpone all reminders command. To fill a field select the value from the list or type any time.

Periodic reminder keeps running according to the configured schedule after clicking the Stop button. It is possible to stop such reminder completely in the My reminders list.
