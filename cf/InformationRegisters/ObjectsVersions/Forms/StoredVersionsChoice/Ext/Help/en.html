It is used to view the history of the object changes and select saved versions. Version history let you answer the questions as follows: "Who changed the object?", "When was the object changed?" and "What was changed in object?".

Using the list you can log the history of document and list changes, view any version of an object or compare any versions of an object.

To work with object change history, it is necessary to perform prior configuration of the versioning. Application administrator configures and view history of versions.

To view version history of any list or document, you can use the Change history command of navigation bar. The command is available only for the application administrator and only for the objects with versioning configured.

The list displays:

  • No. - version number; versions are listed in number descending order, first version is at the bottom of the list;
  • Change author - a user who edited the object;
  • Company contract date - date and time of editing;
  • Note - write detailed information regarding this version. The application automatically fills in this field, for example, when going to another version it is indicated what version you have selected and when you have opened this version.

Filtering versions

  • Using the Attribute changes filter you can display changes of certain attributes. Using theSelect open the attribute list, using check boxes select the attributes which changes shall be displayed in the version list.

Viewing object change events

  • By the Technical data of object change link you can view the log file filtered based on the events associated with the object immediately after introducing and saving changes.

Viewing version

  • Click Open version to view the report of object status at any moment.

Comparing versions

  • To view the report of object status change, select two or more versions in the version list, click Compare versions.

Returning to previous version

  • To return to one of the previous object versions, select the required line with the object version, click Go to version. Changes made after saving this version will be cancelled.
