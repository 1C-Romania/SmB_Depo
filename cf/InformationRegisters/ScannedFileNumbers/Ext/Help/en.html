It is intended to view the numbering of the scanned documents and their owners.

You can view the list by running the Scanning setup - Open numbers of the scanned files command on the Working with files tab in personal application settings.

The application automatically assigns the number to the scanned files. You can monitor the files movement in the list and view the deleted files.

The list includes:

  • Owner - object of the application. If the number was used in several objects of the application, you can view it in the list;
  • Number - number of the scanned file.

Nulling the numbers of the scanned files

  • When scanning it can be necessary to reset the numbering of the scanned files (start from zero) for some owner of the files (files folders or list item), for example if some images were scanned for test purpose, and after testing you need to scan final images with correct numbers.
  • To do this, you need to find the owner of files and set the value to 0.
