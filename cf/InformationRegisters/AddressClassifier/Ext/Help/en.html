The address classifier is used to store address data required to work with the application.

The address classifier ensures validity of structure and contents of address information in the application. The classifier contains a nation-wide list of street addresses. Using the classifier ensures quick and error-free address input, which is absolutely required to submit tax reporting documents correctly. For example, if you introduce any errors in the address format for an employee tax report, the report will be rejected. Besides, addresses tend to change without notification. The application offers address management functionality based on the address classifier.

The address classifier is available on the 1C:Enterprise user support website, on 1C:ITS discs, or as a separately downloaded file.

To access the address classifier management window, execute the Address classifier command located in the Administration - Setup and maintenance - Classifier or Catalogs section.

This window displays the application address data summary (number of regions for which the address data is available) and lists the basic address data management operations: downloading, clearing, checking for online updates.

Loading the address classifier

  • Click Load.
  • You can choose to load the full classifier, or for specific regions only. This allows to save time and storage on loading address data for the regions you do not need. Specify the required regions by selecting the corresponding check boxes. The regions that are not yet loaded are disabled and color-marked in grey. To select all regions for download, click . To cancel selection, click . To look up a region in the list, click Find. Total number of currently selected regions will be displayed at the bottom of the list.
  • A hyperlink at the bottom of the window specifies the load source. The default hyperlink value is From 1C:Enterprise user support website.
  • Before you can download the classifier, you need to complete authorization on the user support website.
  • To change the classifier load source, click the hyperlink. You can also load the classifier:
    • From a disc directory.
      • specify path to the directory;
    • From 1C:ITS disc.
      • specify path to the disc.
  • Once the required parameters are set, click Load.
  • The load progress indicator will display the following information:
    • total number of regions to be loaded;
    • load source;
    • region being loaded now.
  • Once loading is complete, a notification will be displayed. Click Close to go back to the application.
  • When re-loading previously loaded address data, the earlier copy will be overwritten.
  • Web client does not support the address classifier loading functionality.

      Checking for updates

      • Execute the Check for updates command to check for availability of address classifier updates on the 1C website. This command only checks 1C website for updates, regardless of the loading sources selected earlier.
      • If an update was found for any regions, you will be offered to update the outdated classifier data.
      • To proceed to classifier download, click Update. Check boxes for the regions that need to be updated will be selected automatically.

      Clearing the address classifier

      • Click Clear to delete the address data you do not need (for example, data loaded by mistake).
