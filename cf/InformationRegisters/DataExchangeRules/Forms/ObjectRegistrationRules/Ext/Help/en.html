Designed to control and change data registration rules. Data registration rules are provided for all data synchronization kinds. 

By default, data synchronization is configured according to the Standard exchange rules. The exchange rules include object conversion rules and  object registration rules.

To open Object registration rules, click More - Import object registration rules in the synchronization settings window.

Rules source

  • Use a radio button to select where to get object registration rules:
    • Use standard rules (default)
    • Import from file on computer (for debugging only)
  • Below are given the names of the currently used rules.

Rules data

  • You can monitor the rules data: name, creation date, and application version.

Importing rules

  • Select Import from file on computer.
  • Click Import.
  • If necessary, specify a path to the rule file.

Exporting rules

  • If necessary, you can export registration rules to a file.
  • To do it, click Export registration rules to file,
  • Select the export type:
    • Normal file
    • .zip archive
  • Select a path to the file on the computer or network drive. The name is created automatically. Click Save.

Replacing standard object registration rules with rules from external file

  • Select the Import from file on computer radio button, click Import, and then select a rules file on the computer or in a network folder.
  • Check that you are importing correct rules compatible with your versions of the source and receiver configurations. The Rules data field shows the rules creation date and other useful information.
  • Click Save and close. The imported rules will be applied immediately.

Note: object registration rules are used for all same-type data synchronizations (based on the same exchange scheme; e.g. the Complete synchronization exchange scheme is the basis of two data synchronizations: Synchronization with Chicago office and Synchronization with Atlanta office). Therefore if there are multiple same-type synchronizations, you only have to change rules once (by selecting any of the configured data synchronizations).
