Staff list information register

The register Saves actual information about the staff list.

 

Available fields:

  • Company - for which the staff list is entered. It is selected from the Companies list. The attribute is visible on the form if in Accounting by Multiple Companies is enabled.
  • Structural unit - for which the staff list is entered. It is selected from the Structural units list.
  • Position - for which the staff list is entered. It is selected from the Positions list.
  • Number of Rates - which is being planned for the specified position.
  • Accrual kind (deduction) - selected from the Accrual and deduction types list.
  • Tariff - of the staff list unit.
  • Currency - of the tariff rate, selected from the Currencies list.
