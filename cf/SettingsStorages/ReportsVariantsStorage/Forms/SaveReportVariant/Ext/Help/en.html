Designed to store a report variant.

When a report is customized and generated, you can store it in a new report variant. To do it, click Report variants - Save variants or More - Save variant.

List of existing variants

  • When saving a report variant, you see the list of the existing predefined variants and user variants of this report.
    • The list includes:
      • Standard, predefined report variants. By default, they are common.
      • Common report variants opened for common access by other users.
      • Personal report variants saved before.
    • The list also shows the report variants marked for deletion.
  • You can Edit the report variants existing in the list with the relevant command in the context menu. For a predefined report variant, you can edit the description or location settings. For a user variant, you can edit the name, too (if you have enough rights).
  • To delete an existing user report variant, select it and press Del or use the Mark for deletion command of the context menu. You can not delete predefined report variants.

Naming report variant

  • To save your new report variant, enter a Name not occupied by another variant of this report. The saving procedure checks that all the report names are unique.
  • When you overwrite a variant, you can save the settings in the existing variant: by default, you select the report variant for which the Save variant command is executed. To overwrite, select any personal report variant from the list. The report variants that can not be overwritten (including the predefined variants), automatically get "-copy" to their names. If the replaced report variant was marked for deletion, you see a warning. Click Yes to confirm your choice.
    • The administrators and the users responsible for reports can overwrite reports by other authors.

Availability

  • The report variant administrator can select value Only for author in the Available option box to limit the accessibility of a report variant.
  • If you select accessibility for All users, you can allow its visible for other performers. To show the report variant directly in the report panels for all users, select the Shown in report panel by default check box.

Saving report variant

Use one of the following methods to save a report variant:

  • When the basic parameters are defined, click Save. Thus, you will use the default settings and put the new report variant to the same section of the reports list as the original variant.
  • Click Next to edit the location settings for the report variant. 

Placing in report panels

  • Select one or more check boxes in the Section, group column to place the report variant in the relevant application sections or groups. When placed in a section, the report variant will be displayed at the top of the report panel without placing in the groups.
  • In the Importance column, select the method to select a variant in the group:
    • Important – the report variant will appear bold.
    • See also – the report variant will be displayed in a separate group at the bottom of the report panel.
  • Enter a Description that will be shown under the report name in the report panel and in the report lists.
  • Click Save to save the report variant with the selected settings.
