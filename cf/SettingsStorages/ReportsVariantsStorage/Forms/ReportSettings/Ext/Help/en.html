The form enables you to set up the report.

You can open it from the report form by clicking Settings.

Use the View button to switch the settings into one of the modes:

  • Common - you can change available options only, add or delete them using the check boxes;
  • Extended - you can configure the settings: add options, change fields and report structure.

The configuration window includes several tabs depending on the report complexity:

  • Filters - you can enter the conditions for value filters in the report;
  • Fields - you can add or delete fields from the report using the check boxes;
  • Sorting - you can add, delete or disable fields for sorting the report;
  • View - you can change the view of the report;
  • Structure - you can change the structure of the report.

Filters

  • The user can set various options and filters when generating reports.
  • You can change filter parameters on the tab. Use the check box to select or clear the parameter of the filter (for example, Filter period).
  • If available, use the Select button to select the specific values from the list.
  • Using button you can select values from a clipboard.
  • Extended mode allows switching to all available report parameters setup.
  • You can Add filter by clicking the relevant button.
  • To change the report filter settings, use the table consisting of the following columns:
    • Field - filtered fields enumeration;
    • Condition - selection of filtering operation for each field;
      • The Equal, Not equal, Belongs to group, Not in group operations provide the selection of a single value (single filter).
      • The In list, IN list group, Not in list, Not in list group operations provide the selection of several values (multiple filter).
      • The Filled and Not filled operations do not require value selection.
    • Value - you can specify one or several values of each filtered field depending on operation.
      • Double-click the field to enter a value.
      • Enter parameter value using theSelect button. Select the check boxes for the parameters to be filtered. You can use all or some filter options by selecting the check boxes.
      • Click Select to select several values. Open the list, and then select several values from the list. Select check boxes for filter options you need. Enter parameter values for the filter or select from the list.
    • Placement - defines the visible of the parameter or filter in the report window. IN order to change filter visible you can double-click to select one of several values:
      • In report header - use this value to display the parameter or filter in the report header and in the report settings header;
      • Only check box in report header - use this value to display only filter check box in the report header and report settings header. Using the check box you can only enable or disable this filter. If you use this placement, write full and easy-to-understand header describing the filter value as the filter value is not displayed. It is applicable to filters only.
      • In report settings - use this value to display the parameter or filter in the settings form. It is used by default.
      • Only check box in report settings - use this value to display only filter check box in the report settings. Using the check box you can only enable or disable this filter. If you use this placement, write full and easy-to-understand header describing the filter value as filter value is not displayed. It is applicable only to filters.
      • Not displayed - use this value to display the parameter or filter in the extended mode of the report settings only.
    • Description - you can enter the title for the filter.
  • You can group the filters. To do this, select several filter conditions, run the Group the filters command in the shortcut menu by clicking the right mouse button or on the More menu. The filters can be grouped into the And, Or, Not groups. To cancel the grouping, run the Ungroup command. For more details on reports and filter operations see 1C:Enterprise 8.3 User manuals.

      Fields

    • You can add or delete fields from the report using the check boxes on the tab. Extended mode allows inserting additional fields by clicking the Add button.

        Sorting

      • You can set up report sorting on the tab:
      • In common mode you can enable or disable the sorting by selecting the check boxes.
      • In extended mode you can specify the sorting order and add fields for sorting using the Add button.

          View

        • You can change some report generation parameters, e.g. select or clear the Background color check box to apply the color to all fields.

            Structure

          • You can change parameters on the tab only in the Extended mode.
          • On the tab you can add grouped fields into the report. You can add or change fields, change their places by clicking the relevant buttons.
          • Using the button, you can change the structure of the report.

              Receiving report data

              • Specify the parameters you need, and then click the Close and generate button. The current settings will be saved automatically. If you applied settings by mistake, you can either change or reset them by running the relevant report form commands.
