It is intended to enter the path to archive with initial image files (for storage files in volumes).

If volumes on disk are used to store files, then an archive with the files that are saved in volumes will be automatically created together along with the creation of the initial image.

Then it is necessary to receive the files from this archive when creating a subordinate node infobase from the initial image in the application receiver. To do this, it is necessary to use the Place initial image files command on the actions bar of the Organizer section. The files in volumes will be automatically involved in the subsequent iterations of data synchronization.

Specify the Path to archive with initial image files in one of the fields:

  • For the 1C:Enterprise server running under Microsoft Windows - specify the path to archive clicking the button;

or

  • For the 1C:Enterprise server running under Linux - write the path to archive.

Click Place.
