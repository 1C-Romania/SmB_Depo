It is intended to assign the groups to a user (external user).

The list of user groups can be opened by the Groups command of the user (external user) navigation panel.

Assigning groups to user

  • Select a check box corresponding to group you need.
  • Click Save and close.

Deleting from all groups

You can use this option if groups were assigned mistakenly. Delete the user (external user) from all groups quickly, and then re-assign the groups.

  • Click the Exclude from all groups. If necessary, click Write.
