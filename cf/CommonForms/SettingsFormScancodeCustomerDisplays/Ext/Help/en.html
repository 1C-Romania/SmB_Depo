SCANCODE: Customer display driver handler setup form

Form is designed to set the parameters of the customer display, plugged in with driver "SCANCODE: Customer display driver". The form allows you to configure the following parameters of the device:

  • Model
    Model of connected device.
  • Port
    COM-port, to which device is connected.
  • Speed
    Data transfer rate for the port, to which the device is connected.
  • Use ticker
    Display greeting line by default
  • Ticker
    Text of greening line

After changing of parameter values, press the button "Record and close" to Save made changes, or the button "Cancel" to refuse their keeping.
