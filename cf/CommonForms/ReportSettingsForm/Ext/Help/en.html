It enables you to customize the report.

You can open the form from the report forms by clicking Settings.

In the Type field, you can select one of the following setting modes:

  • Standard - you can change only available options, add or delete them using check boxes.
  • Extended - you can adjust the settings: add options, change fields and the report structure.

Depending on the report mode, the configuration window may contain the following tabs:

  • Filters - you can enter report value filter conditions.
  • Fields - you can add or remove fields from the report using the check boxes.
  • Sorting - you can add, delete or disable fields to sort the report.
  • Theme - you can change the report theme.
  • Structure - you can change the report structure.

Filters

  • Various options and filters can be used to generate the reports.
  • You can change filter parameters on the tab. Select or clear the check box to enable or disable the parameter filter (for example, Filter period).
  • Click Select to select the values. Once you select the data type, you will be prompted to select a particular value.
  • Click button to select values from the clipboard.
  • In the Extended mode, you can customize all predetermined report options.
  • You can use a table to change the filter settings. The table includes three columns:
    • Field - fields that are used to filter data.
    • Condition - select an action for each field filter.
      • If there are operators Equal to, Not equal to, IN group, Not in group, you can select only one value (single filter).
        • Click Select to enter the parameter value. Select the check boxes to enable the parameters to be filtered. You can use all filter options or a part of them by selecting and clearing the check boxes.
      • If there are operators In list, In list group, Not in list, Not in list group, you can select multiple values (multiple selection).
        • Once you select the operator, you can select values. Click Select to select several values. Open the list, select several values from the list. Select check boxes to enable filter options you need. Enter parameter values for the filter or select them from the list.
      • If there are operators Filled and Not filled, do not select any values.
    • Value - you can specify one or several values of each filtered fields depending on the operator.
  • Filters can be grouped. To do that, right-click multiple filter conditions, and then run the Group filters command. You can add the filters to groups AND, OR, NOT. To ungroup the filters, click Ungroup.
  • For more details on the filter reports and operators, see 1C:Enterprise 8.4 User Guide.

Fields

  • On the tab, you can add or delete fields from the report using check boxes. IN the extended mode, you can insert additional fields by clicking Add.

Sorting

  • On the tab, you can customize the report sorting. IN the extended mode, you can add fields to be sorted by clicking Add. You can also specify the sorting order. IN the standard mode, you can enable or disable sorting using check boxes.

Theme

  • On the tab, you can customize the report theme settings. For example, you can select or clear the Background color check box to color the fields.

Structure

  • On the tab, you can add grouped fields to the report. You can add or change fields, change their location by clicking relevant buttons.
  • Click button to change the report grouping.

Receiving report data

  • Specify the options you need, and then click Close and generate. Last settings will be saved automatically. If incorrect settings are applied, you can either change or clear them.
