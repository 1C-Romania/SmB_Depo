An update description includes a list of modifications in the last application version and the revision history of the previous versions.

It is displayed once, when a new update is finished. You can see the update description later: use the Application modification description command in the Information panel of the Main section. 

Available actions

  • You can save information about recent application modifications and upgrades  in a file: click Save as and specify the path to the file on your computer. You can also print them out with the relevant button. 
  • For better navigation, you can collapse or   Expand all groups (titles).
  • You can also use the search command to quickly find necessary information about the new version.

Information about infobase version update progress

  • If an update is finished, but some data is still being processed, all users see the warning: Additional data processing is in progress. The functionality of the application is temporarily limited, as the update is not yet complete.

Options for administrator

  • The administrator can read a list of additional instructions recommended after the obligatory part of the infobase update process. To begin executing the instructions, the administrator can use the links in the text describing the new version.
  • Follow the Additional data processing is in progress link to learn more about the additional data processing progress.
  • Follow the Technical information about update results link to view the registration log events that happened during the update process.
