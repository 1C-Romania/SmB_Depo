Designed to configure proxy server parameters.

You can get files from the Internet via HTTP, HTTPS, and FTP protocols and save them on your client, server, or temporary storage.

Some operations (e.g. exporting currency rates, exporting an address classifier, data exchange via an FTP resource, online website configuration update, etc.) may require Internet access. If you are online via a proxy server, specify its parameters in the application. There are client and server settings of a proxy server.

Generally, the proxy server parameters are necessary for the server mechanisms receiving files from the Internet via HTTP, HTTPS, and FTP protocols.

The client mechanisms (e.g. exporting of currency rates in thin and thick clients) require configuring at a workplace. You do not have to configure a proxy server to work with email. Normally, proxy server parameters are specified for each workplace individually, as the Internet access settings may differ for different workplaces.

However if the organization uses a client server version of 1C:Enterprise, the proxy server parameters must be specified on 1C:Enterprise server, too (Administration - General settings).  The proxy server parameters for a workplace and for 1C:Enterprise server may differ.

To provide Internet access to the client computers, proxy server parameters are defined at each client workplace (the Personal settings command in the Administration section).

Attention: when security profiles are activated, you can configure proxy server parameters only with the configuration file inetcfg.xml.

Configuring proxy server parameters

  • Switch to a necessary variant of proxy server function:
    • Do not use proxy server - the default value for the first application start.
    • Use system proxy server settings - settings are automatically filled with the values of the OS proxy server settings. IN this case, the completed settings can not be edited.
    • Use other proxy server settings - settings are configured manually:
      • Write the Server address and Port (a common parameter for 1C:Enterprise server and workplaces).
      • For each workplace and server, specify proxy server User and their Password (they may be different, depending on the organization's network policy).
      • Select the Do not use proxy from local addresses check box to minimize load on the proxy server operating in the organization local network.
  • To confirm proxy server parameters, click OK.
