The parameters are configured for systems working with web services.

If necessary, specify the proxy servers addresses for separate protocols and Internet server addresses with direct access.

The Use default proxy server for all protocols check box is selected by default.

The main parameters of settings (server's Address and Port) will be used as well as additional parameters by default.

Setting additional parameters

  • To specify proxy servers addresses for separate protocols, clear the Use default proxy server for all protocols check box;
  • In the HTTP, HTTPS, FTP corresponding field, correct the proxy server name which will be used for each protocol;
  • If necessary, specify Port;
  • In the Do not use proxy server for addresses list by the Add button specify the addresses with direct access;
  • Click OK to confirm entered parameters.
