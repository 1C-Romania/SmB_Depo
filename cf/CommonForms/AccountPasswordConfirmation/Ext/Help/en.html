Designed to enter a password of an email account. The window opens when you try to send a message from accounts without the password entered.

Entering password and sending email message

  • To send an email message, enter a Password and its Confirmation.
  • Click Next to confirm the password and send the message.
  • Click Cancel to cancel the password and sending the message.

Saving password

  • Select the Remember password check box to save the password in the infobase system settings. For system accounts, it is recommended to remember the password to allow automatically send messages without user intervention. Still, you need to decide whether to store passwords in the infobase individually for each case.
