It is intended to save the print form. There is an option to save print forms in the required formats on a computer or to attach them to a document.

  • Select the format of the print form to save using the radio button:
    • Save to folder on a computer.
      • Specify path to folder on a computer.
    • Attach to document.
  • Select check boxes for one or several formats, in which it is necessary to save (attach) a print form.
  • Select the Compress to archive (zip) check box to reduce the size of saved files.
  • Click Save. Settings will be saved also.
