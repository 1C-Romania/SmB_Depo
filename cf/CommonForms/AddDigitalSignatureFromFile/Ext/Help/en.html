The form is used for adding an electronic signature from file to the items of some lists of the infobase. You may do it, for example, in case the file in the form of signature is received later than the signed data.

You can open the from the file list (list of the object files attached), Attached files list by running the Electronic signature and encryption - Add electronic signature from file command. You can also find this command on the More menu of the file card (attached file card).


The lists contains the items:


  • Electronic signature file - path to the file containing an electronic signature.
  • Comments - information which clarifies added electronic signature.

Adding electronic signature from file


  • Choose one or several files with electronic signatures.
  • Click Add and specify signature file path on hard disk drive or network drive with button.
  • Write Comments to the signature.

Removing electronic signature


  • Click More - Remove to Remove the signature file from the list.
