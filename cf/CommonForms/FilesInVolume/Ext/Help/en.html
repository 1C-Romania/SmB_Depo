Intended for viewing of the files stored in a volume on the disk.

Files in each volume are stored in subdirectories. A separate subdirectory is created for each creation date of file version. All files are stored with original name and extension, and version number is added between them, for example, Order2.2.txt.

You can open list with command Files in volume on the navigation panel Volumes.

The list displays:

  • Path to file in volume - full name of the file includes the subdirectory in which it is stored
  • Size - size in bytes
  • Author - user who added the file

Files filter

  • You can filter files by application object to which they are referred. Select the appropriate value from the list in the field Files owner. For example, you can select Attached files (Meeting) as owner, and all files attached to the documents Meeting will be shown.
  • Files can be stored in the application without their binding to any objects. You can view information about them in files list. To view files from this list, select File versions as owner.

Viewing file information

  • Double-click to view information about the file. Depending on the owner (specified in the field Files owner), attached file card or file version card is opened.
