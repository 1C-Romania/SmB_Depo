This form is intended for displaying, editing and printing out generated printing forms, and for sending a printing from via electronic mail and saving to a file.

It is called from documents and documents lists of the application upon the Print command.

There are the following opportunities:

  • Printing out of documents - a printing form common for all documents.
  • Package printing
  • Adjustable package

Printing out

  • Click Print to print out the required number of copies of the generated printing form

Number of copies

  • Specify the number of Copies. The application saves the selected number of copies. For document packages, number of copies depends on the package settings

Editing printing form

  • With the Edit button, you can activate the mode of editing of this printing form. IN such case, the printing form becomes occupied and unavailable for editing by other users. You can deactivate the editing mode by clicking this button one more time

Saving printing form

  • Using the Save button, you can save printing forms in required formats to a folder on the computer or attach them to a document. Using a radio button, choose the method of document saving and required formats of saving

Sending via electronic mail

You can Send generated printing forms of a document as attachments to an email.

  • Before sending, you need to indicate with the help of checkboxes the format(s) in which the document shall be sent
  • To reduce the attachment size, it is recommended to archive the printing form (.zip format) prior to sending. Click Select. The application will save the settings
  • Complete the To and Subject message fields and write the message text. Click Send
  • When sending a documents package, the application generates a separate attachment for every printing form. If you print at once a package of printing forms for several documents from a list, it is recommended to split the documents by counterparties. IN this case it is convenient to send the whole package to one counterparty, for example if a package includes two printing forms, then the first file would include all invoices, and the second - all consignments

Navigating to required document

  • You can select several documents from a list for printing
  • In this case the application displays several printing forms. For quick navigating to the required printing form, click Navigate to document (press F7)
  • Choose the required document from the suggested list and click OK

Editing layout

  • Click More - Change layout, to go to editing layout of the printing form. The printing form becomes inactive. When you complete editing, the printing form will change automatically

Navigating to printing form layouts list

  • Run the More - Navigate to printing forms layouts command to open the list of layouts

Printing package

  • It is possible to print out a package of various documents. Printing forms of the package are displayed on separate pages, whereas you can view any particular form by selecting the needed one from the list of printing forms on the left. You can change the number of Package copies

Adjustable package

The application allows you to print out a document package selectively.

  • On the left in the Adjustable package window, there is a bar for setting the number of copies, which consists of the following columns:
    • - in this column you can make selection for printing using checkboxes
    • Printing form
    • Number of Copies
  • To customize package printing, click More - Package adjustment. With the commands of the drop-down menu, you can Select or Clear all checkboxes, change the sequence of printing forms in the package (Move up/ down) or Clear package settings
  • Select the required printing form from the package for displaying. Select the checkbox for printing this printing form. Specify the required number of copies for each printing form. The application will save the settings
  • Besides you can change the number of Package copies
  • For convenient viewing of a printing form you can disable the copies number adjustment bar with theShow/ Hide copies number adjustment button

Sum of numbers

  • Select the needed cells with numbers
  • Click to Calculate the sum of numbers in the selected cells
