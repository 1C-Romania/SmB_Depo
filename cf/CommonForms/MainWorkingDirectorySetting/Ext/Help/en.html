Designed to set up parameters of the main working directory.

The main working directory is the folder on your computer used for temporary storage of files received from the application for reading and editing.

Open the directory by clicking the Main working directory settings ref in the personal settings (the Working with files tab).

Location of main working directory

  • The Location field specifies the path to the main working directory on your personal computer.
  • It is defined automatically, but can be changed if necessary. To do it,  Select another folder on your computer. When you select another path to the main working directory, select an empty directory (with no files or subdirectories).
  • To return to the initial settings, click  Restore default value.

Maximum size

  • Use the Maximum size (MB) field to state the maximum size of all files in the main working directory (by default, 100 MB). When it is 0 MB, file size is not limited. Size is stated in megabytes (MB).

File deletion request

  • If the Confirm when deleting from main working directory check box is selected, a file deletion request will appear before every file deletion. By default, the check box is cleared.

Deleting files after editing

  • If the Delete files after editing check box is selected, after a file is copied to the application, its original copy is deleted from your computer. Use this setting to save your hard disk space or to secure your data. By default, the check box is cleared.

Current state

  • You can view the Current state of the main working directory - number of contained Files and their Total size (MB).

Viewing file list in main working directory

  • Click Open file list to view the list of files currently stored in the main working directory.

Clearing main working directory

  • Click Clear main working directory to delete all files from the main working directory except for those you locked for editing. Click Yes to confirm.
