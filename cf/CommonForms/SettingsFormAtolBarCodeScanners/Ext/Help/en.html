ATOL:barcode scaner driver handler setup form

Form is designed to set the parameters of the magnetic card readers, connected with an external component "ATOL: Driver of input devices". The form allows you to configure the following parameters of the device:

  • Model
    Model of the connected device.
  • Port
    Serial port number (physical or emulated), to which device is connected.
  • Speed
    Parameter of a port, to which device is connected; data transfer rate.
  • Data bits
    Parameter of a port, to which device is connected; number of data bits in a transmitted byte.
  • Stop bits
    Parameter of a port, to which device is connected; number of stop bits.
  • Parity
    Parameter of a port, to which device is connected; method of parity check.
  • Sensitivity
    Parameter of magnetic card reader, connected through the keyboard interface; sensitivity.
  • Prefixes
    Prefixes, added by device to the value of card read. Is a sign of the beginning of reading or a new track.
  • Suffixes
    Suffixes, added by device to the value of card read. Is a sign of the end of track or the end of reading.

After change parameter values, press the "OK" button to save made changes, or button "Cancel" to refuse their keeping.
