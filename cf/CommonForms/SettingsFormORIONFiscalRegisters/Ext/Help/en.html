ORION: Fiscal register FRO1K driver handler setup form

You can use the form to set working parameters of the fiscal register supported by the ORION: Fiscal register FR01K driver. The form allows setting the following device's parameters:

  • Port
    Serial port number to which the fiscal register is connected.
  • Data rate
    Data exchange rate with the fiscal register.
  • Timeout
    The maximum waiting time for the answer from the fiscal register.
  • User password
    FR user password. It is applied for the receipts printing on FR.
  • Administrator password
    FR administrator password. It is applied when performing operations with FR demanding administrative rights.
  • Payment1
    The text for printing the description of non-cash payment No. 1. For example, "Payments card".
  • Payment2
    The text for printing the description of non-cash payment No. 2. For example, "Credit".

After changing the parameters, click OK to save changes or Cancel to discard changes.
