It is intended to configure the settings of the events registration of access to personal data.

You can open it by running the Personal data access events registration command from the Support and service - Administration section.

Almost all the events (except events registration of access to personal data) are registered by the application by default.

The application administrator should select the Personal data check box to enable the events registration of the access to personal data of individuals.

Select the Passport data check box to enable the registration of access to passport data. After that, the application automatically registers access events to this data in the events list.
