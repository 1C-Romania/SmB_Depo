It is necessary to enter registration data received on the users support website of 1C:Enterprise to get the access to online resources:

  • Enter Login;
  • Enter Password.
  • Select the Remember password check box in order not to enter this data in future.
  • Click the Go to the registration on user support website link to get access to 1C online resources.
  • Click Continue for authorization on the support website and return to the application.
