KasbiFR01K: fiscal registers driver handler setup form

The form is intended for setup of fiscal registers parameters supported by the driver "Barcode-M: Driver FR". The form allows you to set up the following parameters of a device:

  • Model
    Model of connected device.
  • Port
    Number of the serial port to which you connected the fiscal register.
  • Speed
    Speed of data exchange with fiscal register.
  • Timeout
    Maximum response time from fiscal register.
  • Password of user
    Password of FR user. Applied when checks are printed on FR.
  • Administrator password
    Password FR administrator. Applied for operations with FR which require administrative privileges.
  • Payment1
    Text for printing the names of cashless payment # 1. For example, "PAYMENT CARD"
  • Payment2
    Text for printing the names of cashless payment #2. For example, "CREDIT".
  • Record driver operations log
    This property is applied when problems with driver and FR occur. With the enabled property, an operations file will be created in the process, to which all the operations as well as results of operations with the driver will be written.

After changing the parameter values, click OK to save the changes or click Cancel not to save them.
