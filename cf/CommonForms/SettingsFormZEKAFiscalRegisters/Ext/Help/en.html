Configuration form of drivers handler "ATOL: Fiscal registers"

Form is designed to set the parameters of fiscal registers, supported by driver "ATOL: Cash register driver". The form allows you to configure the following parameters of the device:

  • Model
    Model of the connected device.
  • Port
    Number of serial port, to which the fiscal register is connected.
  • Speed
    Speed of data exchange with fiscal register.
  • Timeout
    MAX waiting time of fiscal register.
  • User password
    Password of FR user. It is used to print receipts on the FR.
  • Device password
    Password to access the device, needed to work with Cash register when transferring all commands from the PC to the cash register. Attention! Password for the device is not related to the passwords of Cash register operators.
  • Print tax in receipt

    If this property is enabled tax rates will be printed on the receipt.

After changing parameter values, press the button "ОК" to store made changes, or the button "Cancel" to refuse their keeping.
