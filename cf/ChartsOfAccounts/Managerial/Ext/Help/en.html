Managerial chart of accounts

Chart of accounts of Headial accounting. Account names and information accumulated on these account are defined by the Headial policy of the company.

To open the list of accounts, browse Finances / Navigation panel / See also / Chart of Accounts.

User is able to add new accounts and subaccounts in the "1C:Enterprise" mode. Properties available for an account:

  • currency accounting setting,
  • active, LIABILITIES, or active-LIABILITIES flag,
  • off-balance flag,
  • type.

Accounts can be analysed using the report:

  • Turnover Balance Sheet.

How to add subaccounts for the existing account:

  1. Convert account into subaccount, i. e. change its code, order and description.
  2. Enter new account-group, i. e. define its code and order, as for account-parent, and also specify account type - Belongs to group of Accounts.
  3. For the subaccount (s. 1) in the Subordinate to account field specify account-group, Creatingd on second step.
  4. Create additional subaccounts for the account-group if it is required.

Example: it is required to accumulate more detailed information of Goods in tools by material expenses and employees' salary. To do it we have to Creating two subaccounts for the Goods in tools account: Material Expenses and Salary of tools Performers. Here are the steps:

  1. Open the Goods in tools account, change code 40 to 40.01, order 40 to 40.01, description Goods in tools to Material Expenses.
  2. Add new account, define its type as - Belongs to group of Accounts, code - 40, order - 40 and description - Goods in tools.
  3. For the Material Expenses account, in the field Subordinate to Account, specify the Goods in tools account.
  4. Add new subaccount, define its type as - Goods in tools, code - 40.04, order - 40.04 and description - Salary of tools Performers, in the field Subordinate to account specify the Goods in tools account.
