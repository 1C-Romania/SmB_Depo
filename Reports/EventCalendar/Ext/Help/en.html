Events calendar report

Using an EVENT calendar a Head will be able to check the list of the current work, to check the list of overdue work, and also to plan the time proceeding from scheduled EVENT.  It is possible to include EVENT for a certain period in a calendar.

In the Creatingd report all EVENT are conditionally partitioned into three units:

·         Overdue EVENT.  Those EVENT which expiration date is overdue in relation to current date are shown.  Overdue EVENT are displayed by red color.

·         Current EVENT are the EVENT which must be done today.  The current EVENT are displayed by a bright color.

·         Future EVENT are the EVENT which execution is planned to future dates.

It is possible to open any event directly from the report.
