It is intended to receive statistics on the Saved versions of application objects.

You can open the report by running the Number and size of object versions saved command from the versioning settings.

If the disk space assigned to store application data is limited, there is an option to control the size of the object versions saved when matching, and to make the decision to reduce the number of saved versions with the help of the report.

The report includes:

  • Size of versions saved by object types chart;
  • Number of versions saved by object types chart;
  • Numerical data by number and size of object versions saved.
