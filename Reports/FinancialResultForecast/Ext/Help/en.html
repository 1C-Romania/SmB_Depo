Financial result report

The report is available in two options:

  • Financial result (forecast)
  • Financial result (planed-actual analysis)

Financial result (forecast)


The report contains information about forecast of the financial results for the set period of the selected company.

The report is generated on the basis of the Budget documents data.

Financial result (planed-actual analysis)


The report shows information about comparative analysis of the target and actual data of a financial result for the given period for the selected organization.

Target data is generated based on the Budget documents data.
