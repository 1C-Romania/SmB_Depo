Designed to select report variants.

Open it from the report variant form with the More - Select variant command.

You can select from:

  • Standard predefined report variants. By default, it is common and shown in the list with character .
  • Common report variants that other users opened for common access. IN the list, they are displayed with character  . You can choose variants of all the authors.
  • Pre-saved personal report variants.

The selection list may contain three report variant groups:

  • Report variants in use (not separated as a group, located at the top of the list).
  • Hidden in report panels - this group consists of predefined common reports with a cleared By default shown in report panel checkbox.
  • Marked for deletion - this group is always at the bottom of the selection list. When you select a variant marked for deletion, you see a warning message.

The list shows:

  • Name - a full name of the report variant shown in the lists.
  • The Available for all users  image:
    • Shown - all users can work with this report variant.
    • Hidden - only the author can work with this report variant.
  • Author - a full name of the user who created the report variant.
  • At the bottom, you see the Description, detailed information about the selected report variant.

Selecting report variant

  • To open a report variant, select it in the list and click Select.
  • Otherwise, double-click it.

Filtering

  • If the report variant list is too long, you can use the filter to select a necessary variant.
    • In the Author field, select the author of the report variant from the users list.

Customizing report variant

  • Select a report variant, click More - Edit, and then edit necessary fields.

 Deleting report variant

  • You can mark for deletion a user report variant with the command of the More menu. You can also press Del.
  • To clear deletion mark, run this command again. 
  • You can not use this option with predefined report variants.

To administrator/user responsible for reports

  • Usually personal report variants of other authors are hidden. But you may occasionally need to make a personal report from this list available to all users or assign it to another user. To view all the report variants, select the Show personal report variants of other authors check box in the More menu.
